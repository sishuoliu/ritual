# 《功德轮回：众生百态》游戏快速入门

> **人数**：2-4人 | **时长**：30-45分钟 | **类型**：合作+竞争

---

## 📖 故事背景

在这个世界里，众生皆苦，劫难频仍。你将扮演一位心怀慈悲的修行者——农夫、商人、学者或僧侣。

你们需要**共同化解劫难、渡化苦难众生**。但在团队胜利的前提下，功德与慧最高者将获得最终的荣耀。

**核心问题**：什么时候为集体牺牲？什么时候为自己积累？

---

## 🏆 胜利条件

### 团队胜利（必须同时满足）
- 游戏结束时 **劫难 ≤ 12**
- 游戏结束时 **渡化众生 ≥ 5**

### 个人胜利
- 团队胜利后，**功德 + 慧** 总和最高者获胜

### 立即失败
- 任意时刻 **劫难 ≥ 20**，游戏立即结束，全员失败

---

## 🔄 回合流程（共6回合）

| 阶段 | 内容 |
|------|------|
| **1. 集体事件** | 翻开1张集体事件卡，全体按卡牌指示执行抉择 |
| **2. 众生阶段** | 场上众生滞留+1，超时（≥2）未渡化：劫难+5并移除 |
| **3. 个人事件** | 奇数回合：每人抽1张通用事件卡<br>偶数回合：每人抽1张职业专属卡 |
| **4. 行动阶段** | 按顺序，每人执行 **2次行动**（见下方行动表） |
| **5. 回合结束** | 偶数回合：全体资粮-1（消耗）<br>补充众生卡至2张 |

---

## ⚡ 基本行动

每回合可执行 **2次行动**，同一行动可重复执行。

| 行动 | 条件 | 效果 | 职业加成 |
|------|------|------|----------|
| **劳作** | 无 | 资粮 +3 | 农夫 +4 |
| **修行** | 无 | 慧 +2 | 学者 +3 |
| **布施** | 资粮≥2 | 资-2，功德+1，劫难-1 | 商人 功德+2（共3） |
| **渡化** | 慧≥众生需求 | 支付慧，获功德，渡化+1 | 完成团队目标的关键 |
| **护法** | 资粮≥2 | 资-2，功德+3，劫难-3 | 僧侣 资-1，功德+4 |
| **互助** | 每局限2次 | 请求他人帮助，成功双方+1功德 | 促进合作 |

## 💡 新手建议

1. **前期积累**：前2回合多劳作/修行积累资源，后面再布施/渡化
2. **别忘渡化**：渡化众生是团队目标！超时会+5劫难
3. **警惕劫难**：劫难≥15时要警惕，多布施/护法降劫难
4. **同时亮牌**：集体事件时用A/B选择卡同时亮牌，增加博弈乐趣
5. **发愿加分**：发愿不强求，但达成可获大量分数

## 📝 示例：第1回合演练（4人游戏）

**设置完成**：
- 农夫(资4 功德2 慧2)
- 商人(资8 功德2 慧1)
- 学者(资4 功德2 慧5)
- 僧侣(资1 功德6 慧5)
- 众生区有2张众生卡

### 阶段1 - 集体事件
翻开「旱魃肆虐」，劫难+6。大家同时亮A/B卡：
- 农夫选A（资-2，功德+1）→ 资2，功德3
- 商人选A（资-2，功德+1）→ 资6，功德3
- 学者选B（资-1）→ 资3
- 僧侣资粮不够，只能选B
- 3人选A，劫难-3。最终劫难 = 0+6-3 = **3**

### 阶段2 - 众生阶段
2张众生卡滞留+1，现在都是1回合。

### 阶段3 - 个人事件
第1回合是奇数，每人抽1张通用事件卡执行。

### 阶段4 - 行动阶段（每人2次行动）
- 农夫：劳作(+4资)→资6，劳作(+4资)→资10
- 商人：劳作(+3资)→资9，修行(+2慧)→慧3
- 学者：修行(+3慧)→慧8，修行(+3慧)→慧11
- 僧侣：修行(+2慧)→慧7，布施→功德7，劫难2

### 阶段5 - 回合结束
第1回合是奇数，无消耗。补充众生至2张。回合标记移到第2回合。

## ❓ 常见问题 FAQ

| 问题 | 答案 |
|------|------|
| 资源不够执行行动怎么办？ | 不能执行该行动，选择其他行动 |

| 皈依/大乘可以中途选择吗？ | 可以！任何时候都可以选择 |
| 发愿没达成会扣分吗？ | 不会扣分，只是没有奖励 |
| 同一行动可以执行两次吗？ | 可以！每回合2次行动可重复 |
| 商人用资粮渡化是什么意思？ | 商人可用双倍资粮代替慧来渡化 |
| 护法祝福效果是什么？ | 护法后本回合其他玩家渡化+1功德 |

## 🌍 共业倍率（核心机制）

> *"一人得道，鸡犬升天；众生共业，同登彼岸。"*

**终局得分 = 个人得分 × 共业倍率**

| 终局劫难 | 共业倍率 | 境界 |
|----------|----------|------|
| **0-4** | **×1.5** | 净土境界 |
| **5-8** | **×1.2** | 功德圆满 |
| 9-12 | ×1.0 | 勉强成功 |
| 13-19 | 失败 | 全员无分 |
| ≥20 | 立即失败 | 全员无分 |

### 为什么这解决了"搭便车"问题？

- 搭便车第一名（30分）× 勉强成功（1.0）= **30分**
- 帮忙但第二名（25分）× 净土境界（1.5）= **37.5分**

**帮助团队 = 帮助自己**，不需要惩罚，机制自然引导利他。
