🏆 胜利条件

• 团队胜利：劫难 ≤12 且 渡化 ≥5

• 个人胜利：团队胜利后，终局得分最高者获胜

• 立即失败：劫难 ≥20

🔄 回合流程（共6回合）

阶段内容
1. 集体事件翻卡，全体用A/B卡同时抉择
2. 众生阶段滞留+1，超时（≥2）劫难+5
3. 个人事件每人抽1张个人事件卡
4. 行动阶段每人执行2次行动
5. 回合结束偶数回合全体资粮-1
