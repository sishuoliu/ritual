📖 完整游戏规则（1/2）

一、游戏背景

在这个世界里，众生皆苦，劫难频仍。你们扮演修行者，需要共同化解劫难、渡化众生。团队胜利后，功德与慧最高者获得荣耀。

核心问题：什么时候牺牲自己帮助团队？什么时候积累个人资源？

二、资源说明

资源符号颜色用途
资粮资 / 💰金色支付行动成本，布施/护法消耗
功德功德 / ❤️红色善业积累，计入得分
慧慧 / 💙蓝色智慧修行，渡化条件，计入得分

三、游戏设置

  1. 将游戏主板放在桌子中央
  2. 在劫难轨道的 0 格放置标记（可用硬币）
  3. 在渡化轨道的 0 格放置标记
  4. 在回合轨道的 第1回合 格放置标记
  5. 洗匀集体事件卡（6张），背面朝上放在主板旁
  6. 将5张众生卡正面朝上放在众生区
  7. 每人选择1个角色，拿取角色板
  8. 按角色板发放初始资源（使用资源标记）
  9. 每人从本职业2张发愿卡中选1张，正面朝上放在面前
  10. 每人拿1套A/B选择卡（A卡+B卡各1张）

四、回合流程（共6回合）

阶段流程详细说明
1. 集体事件翻1张事件卡 → 全体同时出A/B卡见第五节
2. 众生阶段所有众生滞留+1 → 超时检查滞留≥3的众生移除，劫难+5
3. 行动阶段按顺序，每人执行2次行动见第六节
4. 回合结束偶数回合(2,4,6)全体资粮-1表示生活消耗

※ 核心版移除了"个人事件"阶段，简化流程

五、集体事件详解（A/B抉择）

流程：

  1. 翻开1张集体事件卡，朗读内容
  2. 所有玩家同时选择A卡或B卡，扣在面前
  3. 所有人同时翻开，按卡牌指示执行
选择通常效果意义
A（牺牲）资粮-2，劫难-1牺牲自己，帮助团队
B（保全）无代价，但总劫难+3保存资源，但集体承受

七、渡化详解

渡化是救助众生的核心行动。每张众生卡显示：

项目说明示例
慧需求你的慧必须 ≥ 此数值才能渡化慧需求4 → 你的慧必须≥4
资粮成本需支付的资粮资粮成本2 → 支付2资粮
功德奖励渡化成功获得的功德功德奖励3 → 获得3功德

渡化流程：

  1. 选择1张众生卡
  2. 确认：你的慧 ≥ 慧需求
  3. 支付：资粮成本（僧侣-1）
  4. 获得：功德奖励
  5. 渡化轨道 +1，移除该众生卡

※ 众生滞留≥3时会"超时"：移除该卡，劫难+5，不算渡化！

八、共业倍率（核心机制）

终局得分 = (功德 + 慧 + 发愿奖励) × 共业倍率

终局劫难0-45-89-1213-19≥20
共业倍率 ×1.5 ×1.2 ×1.0 失败 立即失败
境界 净土境界 功德圆满 勉强成功 团队失败 全员失败

关键洞察：帮助团队降低劫难 = 提高所有人的得分倍率！

计分示例：

• 玩家A：功德20 + 慧10 + 发愿12 = 42分，终局劫难5 → 42 × 1.2 = 50分

• 玩家B：功德25 + 慧8 + 发愿0 = 33分，终局劫难11 → 33 × 1.0 = 33分

A虽然基础分低，但因为团队劫难低，最终得分反而更高！

九、游戏结束与胜负判定

触发条件结果
任意时刻劫难 ≥20立即失败，游戏结束，全员无分
第6回合结束检查团队胜利条件

团队胜利条件（必须同时满足）：

  • 劫难 ≤ 12
  • 渡化 ≥ 5

个人胜利：团队胜利后，终局得分最高者获胜。平局比功德，仍平局则共享胜利。

十一、常见问题

Q1: 资源不够执行行动怎么办？

A: 不能执行该行动，选择其他行动。但集体事件选A时，资源不足仍可选A（资粮扣至0，获功德+3）。

Q2: 众生卡用完了怎么办？

A: 核心版只有5张众生卡，全部渡化后众生阶段跳过。

Q3: 同一行动可以连续做两次吗？

A: 可以！每回合2次行动可以是相同行动。

Q4: 发愿没达成会扣分吗？

A: 不扣分，只是没有奖励分数。
