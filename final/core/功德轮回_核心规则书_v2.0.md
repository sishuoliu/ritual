## 一、游戏概述

### 1.1 背景故事

在这个世界里，众生皆苦，劫难频仍。你将扮演一位心怀慈悲的修行者——农夫、商人、学者或僧侣。

你们需要**共同化解劫难、渡化苦难众生**。但在团队胜利的前提下，功德与慧最高者将获得最终的荣耀。

**核心问题**：什么时候为集体牺牲？什么时候为自己积累？

### 1.2 资源说明

| 资源 | 含义 | 获取方式 |
|------|------|----------|
| **资粮** | 物质资源 | 劳作、某些事件 |
| **功德** | 善业积累 | 布施、渡化、护法、发愿 |
| **慧** | 智慧修行 | 修行、渡化、某些事件 |

### 1.3 游戏目标

**团队胜利条件**（必须同时满足）：
- 游戏结束时，**劫难 ≤ 12**
- 游戏结束时，**渡化众生 ≥ 5**

**个人胜利**：团队胜利后，计算**终局得分**，最高者获胜

**团队失败**：
- 任意时刻劫难 ≥ 20，立即失败
- 游戏结束时未满足团队胜利条件

## 五、回合流程

游戏共 **6回合**，每回合包含5个阶段：

### 5.1 集体事件阶段

1. 翻开1张**集体事件卡**
2. 所有玩家**同时**用A/B选择卡决定自己的选择
3. 同时亮牌，按卡牌指示执行

### 5.2 众生阶段

1. 场上所有众生卡**滞留+1**
2. 滞留 ≥2 的众生：**劫难+5**，移除该众生
3. 补充众生卡至 **2张**

### 5.3 个人事件阶段

每人抽1张**个人事件卡**，按指示执行。

### 5.4 行动阶段

按顺序，每人执行 **2次行动**。同一行动可重复执行。

| 行动 | 条件 | 效果 | 职业加成 |
|------|------|------|----------|
| **劳作** | 无 | 资粮 +3 | 农夫 +5 |
| **修行** | 无 | 慧 +2 | 学者 +3 |
| **布施** | 资粮≥2 | 资-2，功德+1，劫难-1 | 商人 功德+3 |
| **渡化** | 慧≥众生需求 | 支付慧和资粮，获功德，渡化+1 | 僧侣 成本-1 |
| **护法** | 资粮≥2 | 资-2，功德+3，劫难-3 | 僧侣 功德+5 |

### 5.5 回合结束阶段

1. **偶数回合**（2、4、6）：全体资粮 -1
2. 回合标记移动到下一回合

## 六、渡化众生

### 6.1 众生卡

每张众生卡显示：
- **渡化条件**：需要的慧（3-6不等）
- **资粮成本**：需要支付的资粮
- **功德奖励**：渡化成功获得的功德

### 6.2 渡化流程

1. 选择1张场上的众生卡
2. 检查：慧 ≥ 渡化条件
3. 支付：资粮和慧（按卡牌要求）
4. 获得：功德奖励
5. 渡化计数 +1，移除该众生卡

---

## 七、集体事件

集体事件分为三类：

### 7.1 天灾共业（A/B抉择）

所有玩家同时选择A或B：
- **选A（牺牲）**：支付个人代价，帮助减少劫难
- **选B（保全）**：不支付代价，但劫难增加更多

**牺牲加成**：
- 劫难≥10时选A：额外功德+2
- 资源不足仍选A：额外功德+3
- 全员选A：每人功德+1

### 7.2 人祸事件

根据卡牌指示执行，通常需要资源或劫难增加。

### 7.3 功德事件

团队获得功德、慧或减少劫难的正面事件。

## 八、共业倍率（核心机制）

> *"一人得道，鸡犬升天；众生共业，同登彼岸。"*

### 8.1 终局得分公式

**终局得分 = 个人得分 × 共业倍率**

### 8.2 共业倍率表

| 终局劫难 | 共业倍率 | 境界 |
|----------|----------|------|
| **0-4** | **×1.5** | 净土境界 |
| **5-8** | **×1.2** | 功德圆满 |
| 9-12 | ×1.0 | 勉强成功 |
| 13-19 | 团队失败 | 全员无分 |
| ≥20 | 立即失败 | 全员无分 |

### 8.3 个人得分计算

| 项目 | 分数 |
|------|------|
| 功德 | 每点 1分 |
| 慧 | 每点 1分 |
| 发愿达成 | 简单+10~12 / 困难+16~20 |

**个人得分 = 功德 + 慧 + 发愿奖励**

### 8.4 示例

| 玩家 | 策略 | 个人分 | 终局劫难 | 倍率 | 最终 |
|------|------|--------|----------|------|------|
| A | 搭便车 | 30 | 11 | ×1.0 | **30** |
| B | 帮助者 | 25 | 5 | ×1.2 | **30** |
| C | 菩萨 | 22 | 3 | ×1.5 | **33** ✓ |

**结论**：帮助团队降低劫难 = 提高自己的得分倍率！

## 九、游戏结束

### 9.1 结束触发

- 第6回合结束
- 或 劫难 ≥20（立即失败）

### 9.2 胜负判定

1. 检查团队是否胜利（劫难≤12 且 渡化≥5）
2. 若团队失败，全员无分
3. 若团队胜利，计算每人终局得分
4. 最高分者获胜（平局则功德高者胜）

## 十、速查表

### 行动速查

| 行动 | 条件 | 效果 |
|------|------|------|
| 劳作 | - | 资+3（农夫+5） |
| 修行 | - | 慧+2（学者+3） |
| 布施 | 资≥2 | 资-2，功德+1，劫难-1（商人功德+3） |
| 渡化 | 慧≥需求 | 支付资源，获功德，渡化+1 |
| 护法 | 资≥2 | 资-2，功德+3，劫难-3（僧侣功德+5） |

### 回合流程速查

```
1. 集体事件 → 2. 众生阶段 → 3. 个人事件 → 4. 行动阶段 → 5. 回合结束
```

### 胜利条件速查

- **团队胜利**：劫难≤12 且 渡化≥5
- **个人胜利**：终局得分 = (功德+慧+发愿) × 共业倍率

## 十一、新手建议

1. **前期积累**：前2回合多劳作/修行，后面再布施/渡化
2. **别忘渡化**：超时众生+5劫难！
3. **警惕劫难**：劫难≥15时多布施/护法
4. **团队优先**：共业倍率让利他成为高分策略
5. **发愿规划**：根据发愿条件调整策略

## 十二、FAQ

**Q：资源不够执行行动怎么办？**
A：不能执行该行动，选择其他行动。

**Q：众生超时后会怎样？**
A：劫难+5，移除该众生卡，不算渡化。

**Q：发愿没达成会扣分吗？**
A：不扣分，只是没有奖励。

**Q：同一行动可以执行两次吗？**
A：可以！每回合2次行动可重复。

**Q：商人用资粮渡化是什么意思？**
A：商人可用双倍资粮代替慧来渡化。
