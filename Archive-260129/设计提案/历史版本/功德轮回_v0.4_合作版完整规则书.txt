【故事背景】
末法时代，天下动荡。刀兵劫、瘟疫劫、饥馑劫三灾将至，生灵涂炭。

你们是生活在这乱世中的不同身份：僧侣、贵族、商人、农夫...
每个人都有自己的生存之道，但面对即将到来的大劫，
唯有放下成见，共同拯救众生，方能化解灾难。

【游戏目标】
• 主要目标：全员合作，化解三灾，拯救众生
• 次要目标：完成个人发愿，证得果位

▌4.1 设置步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 【选择身份】
   • 4人游戏：僧侣、贵族、商人、农夫
   • 每人拿取对应身份卡和个人面板

2. 【发放起始资源】（根据身份卡）
   • 僧侣：功德6、财富2、影响5、戒律10
   • 贵族：功德2、财富12、影响6、土地1、声望2
   • 商人：功德1、财富10、影响3、商路1
   • 农夫：功德2、财富2、影响1、土地1、虔诚5

3. 【发放发愿卡】
   • 每人抽取自己身份的3张发愿卡
   • 秘密选择1张放在面前面朝下
   • 剩余2张返回牌堆

4. 【准备事件卡堆】
   • 每人拿取自己身份的8张专属事件卡，洗牌后放在身边
   • 将20张共享事件卡洗牌，放在中央
   • 将20张众生卡洗牌，放在中央

5. 【设置轨道】
   • 劫难标记放在劫难轨道的"0"位置
   • 渡化标记放在渡化轨道的"0"位置

6. 【决定起始玩家】
   • 掷骰子，点数最高者为起始玩家

═══════════════════════════════════════════════════════════════════════════════
                          第五章：游戏流程
═══════════════════════════════════════════════════════════════════════════════

游戏进行8轮，每轮分为4个阶段。

▌5.1 阶段1：协商阶段（开放讨论，约3分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【目的】玩家公开讨论本轮策略

【可讨论内容】
• "劫难已经30了，这轮谁能护法？"
• "有个'战乱孤儿'众生需要8财富，谁能捐？"
• "我差2财富就能建寺，谁能借我？"
• "僧侣，你能讲经吗？大家都需要功德"

【规则】
• 可以公开自己的资源和计划
• 可以许诺（但可以不履行）
• 不能公开发愿卡内容（除非选择公开发愿）
• 不能直接交易资源（必须通过行动）


▌5.2 阶段2：行动阶段（同时选择，约5分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【行动选择】
1. 每人选择2个行动（写在纸上或用行动卡）
2. 所有人同时翻开
3. 按顺序依次执行

【基础行动】（所有人都可执行）
┌──────────────────────────────────────────────────────────────────────┐
│  A. 修行                                                             │
│     效果：+2功德                                                     │
│     特殊：若连续2轮未修行，+1痴                                      │
│                                                                      │
│  B. 布施（指定受益者）                                               │
│     消耗：X财富（1-5）                                               │
│     效果：受益者+X财富，你+X功德                                     │
│     特殊：若X≥5，你+1贪                                              │
│                                                                      │
│  C. 善行（指定受益者）                                               │
│     消耗：1行动                                                      │
│     效果：受益者移除1个负面状态，你+2功德                            │
│                                                                      │
│  D. 护法（降低劫难）                                                 │
│     消耗：根据身份不同                                               │
│     效果：劫难-X                                                     │
│     详见身份专属行动                                                 │
│                                                                      │
│  E. 建寺                                                             │
│     消耗：10财富+5影响                                               │
│     效果：+1寺庙                                                     │
│     特殊：建寺者+2功德，所有人+1功德                                 │
└──────────────────────────────────────────────────────────────────────┘

【身份专属行动】见第六章


▌5.3 阶段3：事件阶段（约5分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【执行顺序】

1. 【翻开众生卡】
   • 从众生卡堆顶翻开2张
   • 放在中央展示区
   • 玩家可以在本阶段尝试渡化这些众生

2. 【专属事件】
   • 每人从自己的专属事件卡堆顶抽1张
   • 按回合顺序依次执行
   • 许多事件需要掷骰子判定

3. 【共享事件】
   • 从共享事件卡堆顶翻开1张
   • 按卡牌指示影响所有人或部分人


▌5.4 阶段4：劫难阶段（约2分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【劫难递增】
• 劫难指数+5（标准难度）
• 检查是否触发灾难效果

【灾难效果】
┌──────────────────────────────────────────────────────────────────────┐
│  劫难≥20：启动"刀兵劫"                                               │
│    每轮随机1名玩家（掷骰决定）：                                     │
│    -3财富-2功德（战乱伤害）                                          │
│                                                                      │
│  劫难≥35：启动"瘟疫劫"                                               │
│    每轮所有玩家：-1功德                                              │
│    （叠加刀兵劫）                                                    │
│                                                                      │
│  劫难≥50：世界毁灭                                                   │
│    游戏立即结束，全员失败                                            │
└──────────────────────────────────────────────────────────────────────┘

【检查众生】
• 若场上有众生卡停留≥3轮无人救助：
  该众生卡移除，劫难+5
  渡化失败数+1

【回合结束】
• 检查是否有人功德<5（失败条件）
• 检查是否有人三毒≥20（失败条件）
• 若未失败，进入下一轮

▌6.1 僧侣（The Monk）

• 特殊：戒律10

【专属行动】
┌──────────────────────────────────────────────────────────────────────┐
│  ◆ 托钵                                                              │
│     消耗：0                                                          │
│     效果：向任意玩家请求布施（对方自愿）                             │
│                                                                      │
│  ◆ 讲经                                                              │
│     消耗：1影响                                                      │
│     效果：所有玩家+1功德（包括自己）                                 │
│     护法贡献：劫难-2                                                 │
│                                                                      │
│  ◆ 闭关修行                                                          │
│     消耗：本轮2个行动                                                │
│     效果：下轮所有修行效果翻倍，你+3功德                             │
│                                                                      │
│  ◆ 渡化（用于众生卡）                                                │

【戒律系统】
• 杀生（劫掠等）：-3戒律
• 偷盗：-5戒律
• 妄语（违背承诺）：-2戒律
• 戒律≤3：还俗（转变为农夫身份）
• 戒律=10：每轮自动+1功德

【团队角色】
• 核心功能：讲经为全队提供功德
• 护法能力：中等
• 渡化能力：强（许多众生需要僧侣）


▌6.2 贵族（The Noble）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【起始资源】
• 功德：2
• 财富：12
• 影响：6
• 土地：1
• 特殊：声望2

【专属行动】
┌──────────────────────────────────────────────────────────────────────┐

│     消耗：10财富                                                     │
│     效果：指定受益者+10财富，你+4功德+2影响+2声望                    │
│     护法贡献：劫难-3                                                 │
│                                                                      │
│  ◆ 赈灾                                                              │
│     消耗：8财富                                                      │
│     效果：劫难-5，你+3功德+2声望                                     │

│                                                                      │

• 声望≥10：所有交易成本-1

【团队角色】
• 核心功能：财力支持，资助建寺
• 护法能力：强（赈灾-5劫难）

• 避免任何失败条件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【利他型玩家】
• 选择"菩萨道"或"护法道"发愿
• 全力协助团队，牺牲个人分数
• 适合僧侣、农夫

【平衡型玩家】
• 选择"中愿"发愿
• 在团队贡献和个人成就间平衡
• 适合贵族、商人

【自利型玩家】（高风险）
• 选择"阿罗汉道"或高分发愿
• 专注个人修行
• 但必须确保护法贡献≥10，否则分数减半
• 适合有经验的玩家
