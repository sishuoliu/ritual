                          设计历程回顾
═══════════════════════════════════════════════════════════════════════════════

【v0.1】初始设计
• 8种非对称身份
• 纯竞争模式
• 秘密目标系统
• 问题：与佛教"普度众生"精神矛盾

【v0.2】平衡性修正
• 校准起始资源
• 调整32个秘密目标难度
• 校准64个专属事件期望值
• 问题：仍是竞争游戏

【v0.3】综合修正方案
• 秘密目标→发愿系统（佛教化）
• 增加三毒系统
• 简化为4种基础身份
• 拒绝Feature Creep（删除7项建议）
• 问题：核心矛盾未解决

【v0.4】合作模式重构 ★关键转折★
• 游戏目标从"竞争"改为"合作拯救众生"
• 引入NPC众生卡系统（20张）
• 引入劫难轨道（共同威胁）
• 胜利条件：全员共同达成
• 个人果位：游戏胜利后才评定
• 问题：遗漏修行阶段系统

【v0.5】最终版（当前）★完整设计★
• 补充修行阶段系统（4阶段）
• 每轮新增"修行升级阶段"
• 舍弃资源+骰子判定
• 胜利条件新增：至少1人达到阶段3
• 整合所有系统

                          最终设计核心特色

▌特色1：真正的佛教游戏
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【机制即佛法】
• 三毒系统：贪嗔痴阻碍修行
• 修行阶段：凡夫→初果→菩萨→佛果
• 发愿系统：大乘发愿，普度众生
• 众生渡化：慈悲喜舍，救度苦难
• 因果业报：舍弃资源，放下执着

【不是标签式佛教】
• 不是简单把"修行"改名为"修行行动"
• 而是每个机制都有深刻的佛教意义


▌特色2：合作中的道德困境
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心张力：自利 vs 利他】

┌──────────────────────────────────────────────────────────────────────┐
│  自利路径：                                                          │
│  • 选择"阿罗汉道"发愿（+12分）                                       │
│  • 全力个人修行                                                      │
│  • 冲刺高修行阶段                                                    │
│  • 风险：护法贡献<10，分数减半                                       │
│  • 佛典：自了汉，只求自证涅槃                                        │
├──────────────────────────────────────────────────────────────────────┤
│  利他路径：                                                          │
│  • 选择"地藏宏愿"发愿（+30分，失败-10）                              │
│  • 优先护法、渡化众生                                                │
│  • 协助他人升级                                                      │
│  • 风险：个人修行不足，果位较低                                      │
│  • 佛典：众生度尽，方证菩提                                          │
└──────────────────────────────────────────────────────────────────────┘

【不是非黑即白】
• 可以选择"平衡型"策略
• 团队胜利需要至少1人达到阶段3
• 创造"谁应该牺牲？谁应该冲刺？"的团队协商


▌特色3：丰富的职业事件系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【事件卡设计】
• 32张专属事件卡（每职业8张）
• 20张共享事件卡
• 20张众生NPC卡

【专属事件特色】
• 僧侣：修行考验、法难、传法
• 贵族：权力斗争、赈灾、家族事务
• 商人：商机风险、诚信危机、贸易波动
• 农夫：天灾、疾病、虔诚感应

【骰子判定】
• 50%事件需要掷骰子
• 2d6分布：7最常见（16.7%），2/12最稀有（2.8%）
• 提供应对选项，减少纯运气


▌特色4：修行阶段的高风险高回报
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【机制设计】
1. 舍弃资源（立即失去）
2. 掷骰子判定
3. 失败：舍弃的资源不返还
4. 成功：升级+额外消耗+强大加成

【示例：僧侣升阶段3】
• 总投入：5+8=13财富，3+5=8影响，2+3=5功德
• 失败率：16.7%
• 失败损失：5财富+3影响+2功德
• 成功收益：每轮+2功德（剩余3-4轮=+6~+8功德）+ 讲经×2

【策略深度】
• 何时升级？第3轮还是第5轮？
• 是否冒险？失败会影响团队吗？
• 请求协助：谁来帮我祈福？
• 团队配合：1-2人冲高阶段，其他人支持

▌v0.4测试结果（无修行阶段）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 测试次数：600局
• 胜率：0%（所有配置）
• 主要失败原因：玩家功德<5（82%）

【诊断】
• 游戏过难，不可能获胜
• 功德获取过慢
• 劫难递增过快
• 失败条件过严

▌v0.5修正方案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【关键修正】
1. 增加功德获取：修行+3（原+2），讲经+2（原+1）
2. 减缓劫难：每轮+4（原+5）
3. 放宽失败条件：≥2人功德<5才失败（原1人<5）
4. 增强护法效果：各+1
5. 降低众生难度：财富需求-20%
6. 增加初始资源：所有人功德+1
7. ★核心补充★：加入修行阶段系统

【预期胜率】（v0.5）
• 全利他型：60-75%
• 全平衡型：45-60%
• 混合型：50-65%
• 全自利型：<20%

【成功整合的佛教概念】

1. 四圣谛
   • 苦：劫难轨道，众生受苦
   • 集：三毒（贪嗔痴）
   • 灭：修行阶段，证得果位
   • 道：八正道→游戏中的行动选择

2. 三学
   • 戒：戒律系统（僧侣）
   • 定：修行行动
   • 慧：渡化众生，开悟

3. 六度
   • 布施：布施行动
   • 持戒：戒律系统
   • 忍辱：承受负面事件
   • 精进：修行升级
   • 禅定：闭关修行
   • 般若：高阶段加成

4. 四弘誓愿
   • 众生无边誓愿度：渡化众生系统
   • 烦恼无尽誓愿断：三毒系统
   • 法门无量誓愿学：修行阶段
   • 佛道无上誓愿成：证得佛果

5. 业力因果
   • 舍弃资源不返还：因果不虚
   • 三毒阻碍升级：恶业障道
   • 助人获奖励：善因善果

═══════════════════════════════════════════════════════════════════════════════
                          结语
═══════════════════════════════════════════════════════════════════════════════

从v0.1的纯竞争游戏，到v0.5的合作+个人修行游戏，
《功德轮回：众生百态》经历了一次**根本性的设计转变**。

这个转变源于一个简单而深刻的问题：
"我们这是佛教游戏，真的要这样竞争么？"

最终设计成功整合了：
• 合作拯救众生（慈悲）
• 个人修行证果（智慧）
• 自利与利他的张力（中道）
• 舍弃与升级的抉择（放下）
• 三毒与觉悟的对抗（戒定慧）

这不仅是一款桌游，更是一次完整的佛教经济学体验。
