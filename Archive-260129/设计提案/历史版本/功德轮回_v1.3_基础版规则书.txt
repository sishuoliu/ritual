═══════════════════════════════════════════════════════════════════════════════
                   游戏概述
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】30-45分钟
【游戏轮数】6轮

【核心理念】
"管理好自己，帮助他人，共同渡过劫难。"

【半合作机制】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：福慧得分最高者获得最高果位


═══════════════════════════════════════════════════════════════════════════════
                   资源系统
═══════════════════════════════════════════════════════════════════════════════

【三种资源】

┌──────────────────────────────────────────────────────────────────────┐
│  资源    │  含义          │  获取方式                               │
│  ─────────────────────────────────────────────────────────────────  │
│  财富    │  世俗资源      │  劳作                                   │
│  福      │  利他的成果    │  布施、渡化、护法、讲法                 │
│  慧      │  自度的成果    │  修行、布施、受教                       │
└──────────────────────────────────────────────────────────────────────┘

【福慧双修原理】
• 只有福（只帮助他人）→ 得分低
• 只有慧（只管自己）→ 得分低
• 福慧平衡 → 得分最高

【4个角色】

【僧侣讲法】
每轮结束前，僧侣可选择一位其他玩家进行讲法：
• 目标玩家：+2福+1慧
• 僧侣自己：+2福

                   回合结构

【每轮流程】

┌──────────────────────────────────────────────────────────────────────┐
│  阶段1：事件阶段                                                     │
│  • 翻开1张事件卡，全体执行效果                                       │
│                                                                      │
│  阶段2：众生阶段                                                     │
│  • 翻开1张众生卡，放入众生区                                         │
│  • 检查超时：众生区满2轮未渡化 → 劫难+3，移除                        │
│                                                                      │
│  阶段3：行动阶段                                                     │
│  • 每人按顺序执行2个行动                                             │
│                                                                      │
│  阶段4：讲法阶段（僧侣专属）                                         │
│  • 僧侣可选择一位玩家讲法                                            │
│                                                                      │
│  阶段5：持续帮助奖励                                                 │
│  • 本轮帮助过他人的玩家，连续帮助轮数+1                              │
│  • 连续帮助≥3轮的玩家：+1福+1慧                                      │
│                                                                      │
│  阶段6：结算阶段                                                     │
│  • 检查团队胜利/失败条件                                             │
└──────────────────────────────────────────────────────────────────────┘

【渡化奖励】
基础奖励 = 众生卡福慧奖励

【超时惩罚】
众生停留满2轮未被渡化：劫难+3，移除

【连续帮助轨道】

每轮如果你帮助了他人，在轨道上前进一格：

┌──────────────────────────────────────────────────────────────────────┐
│  连续帮助轮数  │  每轮额外奖励                                       │

【个人得分计算】

┌──────────────────────────────────────────────────────────────────────┐
│  步骤1：计算基础得分                                                 │
│  基础得分 = √(福 × 慧) × 3                                          │
│                                                                      │
│  示例：                                                              │
│  • 福=20，慧=5 → √(100)×3 = 30                                      │
│  • 福=5，慧=20 → √(100)×3 = 30                                      │
│  • 福=12，慧=12 → √(144)×3 = 36 ← 平衡更优                         │
│  • 福=15，慧=15 → √(225)×3 = 45 ← 高度平衡最优                     │
│                                                                      │
│  步骤2：评定道的加分                                                 │
│  根据行为模式自动评定最佳道                                          │
│                                                                      │
│  步骤3：计算最终得分                                                 │
│  最终得分 = 基础得分 + 道的加分                                      │
└──────────────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════════════
                   策略指南
═══════════════════════════════════════════════════════════════════════════════

【核心策略】

1. 【先积累再帮助】
   早期多劳作积累财富
   中后期开始布施和渡化

2. 【福慧平衡】
   关注两者的比例
   差距太大会降低乘积得分

3. 【持续帮助】
   保持每轮都有帮助行为
   连续3轮以上有额外奖励

4. 【角色发挥】
   农夫：多劳作，稳定积累
   商人：多布施，发挥加成
   官员：修行帮助兼顾
   僧侣：讲法助人，自己也获益

【禁忌策略】

• ✗ 全程只修行不帮助 → 团队会失败
• ✗ 全程只帮助不积累 → 资源耗尽
• ✗ 临终大量布施 → 不如持续帮助

【测试结果摘要】

经过5000局模拟测试验证：

✓ 自私策略0%胜率 → 合作是必须的
✓ 平衡策略~99%胜率 → 先顾好自己再帮助他人
✓ 智能策略~60%胜率 → 难度适中
✓ 福慧乘积计分有效 → 强制两者平衡
