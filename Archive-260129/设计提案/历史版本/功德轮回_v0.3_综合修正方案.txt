▌1.1 核心设计原则（更新）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【原v0.2问题】
• 8种身份同时存在，新手无从入手
• 缺乏清晰的核心循环
• 秘密目标与佛教主题脱节
• 缺乏4-6人的最优玩家数版本

【v0.3核心原则】

1. 【简洁核心循环】
   玩家每轮：修行积德 → 发愿立志 → 因果循环 → 证果成就
   
2. 【佛教概念融合】
   不是"贴上佛教标签"，而是"机制即佛法"
   • 秘密目标 → 发愿（Vow）
   • 资源管理 → 三毒克服（贪嗔痴）
   • 身份转变 → 轮回转世

3. 【渐进复杂度】
   • 基础版：4种身份（僧侣、贵族、商人、农夫）
   • 完整版：+4种身份（皇帝、士人、医者、盗匪）
   
4. 【最优玩家数】
   • 核心设计：4-6人
   • 可扩展：3人、7-8人变体

【发愿机制】

1. 【发愿公开时机】
   • 游戏第4轮开始，玩家可选择"公开发愿"
   • 公开后：若达成+额外5分，若失败-10分
   • 不公开：正常分数，失败无惩罚
   • 佛教意义：公开发愿如"当众立誓"，增加压力和动力

2. 【改愿机制】
   • 消耗5功德，可重新抽3张发愿卡，选1张
   • 但旧愿记录为"违愿"，游戏结束-5分
   • 佛教意义：违背誓言有业力代价

3. 【协助他人发愿】
   • 若你帮助他人达成公开发愿，你+3分
   • 创造合作动力


▌2.2 资源系统 → 三毒克服
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【佛教背景】
三毒（贪、嗔、痴）是轮回根源。

【游戏化设计】

┌──────────────────────────────────────────────────────────────────────┐
│  【三毒系统】                                                            │
│                                                                          │
│  每人有3个隐藏的"三毒标记"：                                            │
│                                                                          │
│  ◆ 贪（Greed）                                                          │
│     • 触发：每次获得财富≥5时，+1贪                                       │
│     • 效果：贪≥5时，所有布施成本+2                                       │
│     • 克服：连续2轮不获得财富，-1贪                                      │
│                                                                          │
│  ◆ 嗔（Anger）                                                          │
│     • 触发：每次使用"攻击性"行动（劫掠、灭佛），+1嗔                    │
│     • 效果：嗔≥5时，无法获得善行加成                                     │
│     • 克服：对曾伤害者执行善行，-2嗔                                     │
│                                                                          │
│  ◆ 痴（Delusion）                                                       │
│     • 触发：连续2轮未修行，+1痴                                          │
│     • 效果：痴≥5时，所有功德获得-1                                       │
│     • 克服：闭关/冥想，-2痴                                              │
│                                                                          │
│  【三毒清净】                                                            │
│  • 若游戏结束时三毒皆≤2：+10分                                           │
│  • 若任一毒≥8：-5分                                                      │
└──────────────────────────────────────────────────────────────────────┘

【设计意义】
• 不是惩罚玩家玩得好，而是鼓励"平衡"
• 纯粹追求财富会增加贪，导致布施困难
• 创造"修行"行动的价值

▌4.1 最优玩家数：4-6人
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【为什么4-6人】
• 市场调研：成功桌游多为4-6人最优
• 游戏时长：6人×2行动×8轮=可控时间
• 互动密度：足够的玩家互动，不过于拥挤

【4人游戏】
• 身份：僧侣、贵族、商人、农夫
• 事件卡：调整为"影响至少2人"
• 游戏时长：6轮

【5人游戏】
• 加入：医者
• 标准规则

【6人游戏】
• 加入：医者、士人
• 标准规则


▌4.2 变体规则：3人 / 7-8人
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【3人变体】
• 每人控制1.5个身份
• 或 引入"中立AI玩家"（简化规则）

【7-8人变体】
• 分为2个"僧团"进行团队竞争
• 每团3-4人，共享部分目标
• 增加"团队发愿"卡
