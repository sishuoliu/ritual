【设计目标】
• 简化：资源从8种减少到4种
• 平衡：所有角色胜率目标10-20%
• 互动：增加竞标和交易机制
• 快捷：核心规则3分钟可讲解
• 支持：3-6人（取消7人配置以简化平衡）

【简化逻辑】
• 财富 = 世俗资本，用于捐献和交易
• 功德 = 精神资本，决定最终胜负
• 影响 = 社会资本，角色专长的体现
• 土地 = 固定资产，可捐可产出

▌所有玩家可用的行动（6种）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基础行动】每种消耗1行动点

1. 【布施】消耗3财富 → 获得1功德
   • 商贾加成：消耗5财富 → 获得2功德

2. 【修行】消耗2影响 → 获得1功德
   • 僧伽加成：消耗2影响 → 获得2功德

3. 【善行】消耗1财富 → 获得1功德，选择一位玩家也+1功德
   • 农夫加成：不消耗财富
   • 信女加成：双方各+2功德

4. 【回向】消耗2功德 → 选择一位玩家+3功德，自己+1影响
   • 信女加成：目标+4功德

5. 【耕作】消耗1行动点 → 获得2财富
   • 地主加成：若有土地，每块地额外+1财富

6. 【交易】不消耗行动点（每轮1次）
   • 与任意玩家交换资源（双方同意）

【特殊行动】消耗2行动点

7. 【建寺】消耗8财富+2影响 → 获得5功德，标记"已建寺"
   • 地主可用2土地替代8财富

8. 【捐地】消耗1土地 → 获得3功德+1影响

▌道路设计原则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 所有道路使用统一格式：消耗X资源 + 满足Y状态
• 目标完成率：30-50%
• 道路加分：统一为25分（满足）或12分（小成）

▌5.1 法会竞标（每3轮触发）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3、6、9轮结束时触发"法会"：
1. 僧伽（或影响最高者）主持
2. 所有玩家暗中出价（财富）
3. 出价最高的2位参与法会
4. 参与者获得：功德+3，影响+1
5. 所有出价的财富进入"寺院基金"

【寺院基金】
• 累积的财富用于公共事件
• 灾难时可从基金赈济

▌5.2 自由交易阶段
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

每轮行动阶段前，有30秒自由交易时间：
• 玩家可以交换任意资源
• 可以承诺未来行动（如"下轮对你回向"）
• 交易完成后双方各+1功德

▌5.3 互助机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

当一位玩家功德最低时：
• 其他玩家对其执行"善行"或"回向"时，效果+1
• 该玩家每轮额外+1行动点（追赶机制）

【3分钟教学版】

1. 这是什么游戏？
   你扮演唐宋时期的人物，通过布施积累功德，追求解脱。
   功德最多者获胜，完成"道路"可获得额外加分。

2. 四种资源
   💰财富 — 用来捐献
   ☸功德 — 决定胜负
   ⭐影响 — 社会地位
   🏠土地 — 产出财富

3. 每轮做什么？
   ① 翻事件卡 ② 领资源 ③ 交易 ④ 用行动点做事

4. 六种行动
   • 布施：3财富→1功德
   • 修行：2影响→1功德
   • 善行：1财富→你和另一人各+1功德
   • 回向：2功德→另一人+3功德，你+1影响
   • 耕作：获得2财富
   • 交易：与他人换资源

5. 怎样获胜？
   8轮后，功德×2 + 影响 + 财富÷3 + 道路加分 = 总分
   分最高者获胜！
