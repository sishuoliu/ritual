═══════════════════════════════════════════════════════════════════════════════

《救赎之路》是一款模拟宋代佛教功德经济的策略桌游。

• 时长：30-45分钟
• 人数：4-6人
• 适合年龄：14+
• 核心机制：资源管理 + 谈判交易 + 非对称角色

▌角色详细说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【富商】💰
初始：粮食2，铜钱6，土地0
能力：
  • 交易专精：与其他玩家交易时，获得10%额外收益（向上取整）
  • 市场洞察：可在村庄以更优汇率买卖（粮食2:铜钱2）
策略：成为资源枢纽，通过差价获利

【地主】🏠
初始：粮食3，铜钱2，土地2
能力：
  • 土地产出：每块土地每轮产出1粮食
  • 土地购买：购买土地费用-1铜钱
策略：投资土地，后期稳定产出，大额捐献

【官员】🏛️
初始：粮食2，铜钱4，土地1
能力：
  • 皇宫特权：可使用"访问皇宫"行动（其他角色不可）
  • 政治影响：皇帝相关事件时+1声望
策略：利用政治资本获取额外资源和声望

【农民】🌾
初始：粮食4，铜钱2，土地0
能力：
  • 勤劳：每轮生产阶段+1粮食
  • 多劳多得：每轮+1行动点（共3点）
限制：不能购买超过1块土地
策略：稳健积累，避免风险，用额外行动点换取优势

【寡妇】🕯️（修订）
初始：粮食2，铜钱2，土地0，功德2
能力：
  • 虔诚之心：捐献时额外+1功德
  • 回向增幅：回向时，双方各+2功德（非转移）
  • 哀悼祈福：每轮可免费获得1功德（向亡夫祈福）
限制：不能访问皇宫，不能购买土地
策略：专注功德路线，利用回向建立互惠网络

【僧侣】🧘（完全重设计）
初始：粮食0，铜钱0，土地0，功德0，法力5
资源系统：
  • 法力（新资源）：僧侣专属，代表精神修为
  • 授德点（新得分）：授予他人功德时累积
能力：
  • 接受供养：其他玩家捐献时，僧侣获得等值法力
  • 授予功德：消耗X法力，为一位玩家授予X功德，
              僧侣获得X授德点
  • 主持法会：发起法会，每位参与者支付2铜钱，
              各获得1功德，僧侣获得参与人数×1授德点
  • 赈灾济民：消耗法力，为粮食不足的玩家提供救济
              每救济1人，僧侣+1授德点
限制：不能持有粮食/铜钱/土地超过2单位
      超出部分必须在当轮分配给其他玩家或寺院
胜利条件：
  • 标准胜利：授德点 × 2 = 最终得分
  • 涅槃胜利：法力=0 + 授德点≥12 → 直接获胜

▌回合流程（共10轮）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1：生产】
• 地主：每块土地产出1粮食
• 农民：获得1粮食
• 寡妇：获得1功德（哀悼祈福）
• 僧侣：获得1法力（冥想修行）

【阶段2：事件】
• 翻开1张事件卡，按指示执行
• 事件卡详见事件卡系统

【阶段3：行动】
每位玩家有2行动点（AP），农民有3AP

行动消耗：

├─────────────────────┼──────┼────────────────────────────┤
│  交易               │  0   │  与其他玩家自由谈判交换    │

• 本轮捐献者获得功德和声望
• 僧侣执行功德授予，获得授德点
• 处理竞标（如有多人竞争同一服务）

【阶段5：衰老】
• 所有玩家寿命-1
• 寿命归零者进入来世结算

▌平衡后的回向规则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

当玩家A将功德回向给玩家B：

【普通角色】
• 玩家A选择回向X点功德给玩家B
• 玩家B获得X点功德
• 玩家A失去X点功德，但获得+1功德作为回向奖励
• 净效果：A减少(X-1)功德，B获得X功德，系统总增1

【寡妇角色】
• 寡妇回向X点功德给玩家B
• 玩家B获得X点功德
• 寡妇失去X点功德，但获得+2功德作为回向奖励
• 净效果：A减少(X-2)功德，B获得X功德，系统总增2

【限制】
• 每轮每位玩家最多回向1次
• 不能回向给自己
• 不能回向给僧侣（僧侣不需要功德）

若需求超过供应，进入竞标：
• 所有竞争者秘密出价
• 出价最高者获得服务

▌标准胜利
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

游戏在10轮后结束。

【最终得分计算】
• 普通角色：救赎点 = 功德×3 + 声望×1 + 剩余资源÷3（向下取整）
• 僧侣：救赎点 = 授德点×2

【胜利判定】
1. 救赎点最高者获胜
2. 平局时，功德/授德点高者获胜
3. 再平局，声望高者获胜

▌特殊胜利：解脱（修订版）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

条件（放宽后）：
• 功德 ≥ 12
• 声望 ≤ 3
• 剩余资源（粮食+铜钱+土地价值）≤ 5
