▌设计理念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【主题风格】
• 唐宋时期中国佛教文化
• 古雅、简约、禅意
• 学术性与艺术性结合

【色彩基调】
• 主色调：暖金色、米白色、深棕色
• 辅助色：紫色（影响）、绿色（土地）
• 强调色：红色（重要提示）

【字体风格】
• 中文：使用书法风格字体（如：方正清刻本悦宋）
• 英文：使用古典衬线字体（如：Times New Roman）
• 数字：使用清晰的无衬线字体（如：Arial）

【插画风格】
• 参考：唐宋时期佛教壁画、经书插图
• 技法：工笔画风格，线条细腻
• 元素：莲花、祥云、寺院、经文等

▌福报骰子设计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方案A：标准点数骰子】
• 主体颜色：金色或暖黄色
• 点数颜色：深棕色或黑色
• 点数设计：标准1-6点

【方案B：定制图案骰子】
• 主体颜色：金色或暖黄色
• 图案设计：
  - 1-2面：空白或"平"字（平常）
  - 3-4面：莲花图案或"福"字（小福）
  - 5-6面：祥云图案或"吉"字（大福）

【设计建议】
推荐方案A（标准点数），因为：
• 成本更低
• 玩家更容易识别
• 规则书中已有对应说明

如果预算允许，方案B更符合主题

▌正面设计（资源区）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【布局】
┌─────────────────────────────────────┐
│  顶部：道路名称（大号字体）          │
│                                     │
│  中央：资源区（4个圆形槽）          │
│  [💰财富] [☸功德] [⭐影响] [🏠土地]│
│                                     │
│  底部：行动点显示区                 │
│  当前行动点：○ ○                    │
│                                     │
│  右下角：道路完成标记区（✓）        │
└─────────────────────────────────────┘

【设计元素】
• 背景：淡雅的米白色或浅金色
• 边框：深棕色细线，古典纹样
• 资源槽：圆形，带图标和文字
• 道路名称：顶部居中，大号字体，道路专属配色

【配色方案】
• 福田道：深紫色背景，金色文字
• 布施道：暖金色背景，深棕色文字
• 土地道：绿色背景，金色文字
• 文化道：淡蓝色背景，深蓝色文字
• 勤劳道：橙色背景，深棕色文字
• 渡化道：白色背景，金色文字
