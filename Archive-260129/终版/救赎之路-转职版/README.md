# 救赎之路：功德经济 - 转职版

**Path to Salvation: The Merit Economy - Ascension Edition**

## 版本信息

- **版本号**: v8.3 转职版
- **基于版本**: v6.3 简易版
- **游戏人数**: 3-6人
- **游戏时长**: 60-90分钟
- **适合年龄**: 12+

## 版本特点

本版本包含**本体 + 转职系统扩展**：

### 转职系统
- 完成道路"小成"后可选择转职
- 转职需要牺牲资源，但获得专属能力
- 解锁提前胜利条件
- 弱者转职成本更低，帮助翻盘

### 六个转职职业
| 道路 | 转职职业 | 特色 |
|------|---------|------|
| 福田道 | 罗汉 | 稳健型，主要靠得分取胜 |
| 布施道 | 大檀越 | 转职型，主要靠提前胜利 |
| 土地道 | 舍宅主 | 稳健型，主要靠得分取胜 |
| 文化道 | 文殊化身 | 转职型，主要靠提前胜利 |
| 勤劳道 | 行脚僧 | 稳健型，主要靠得分取胜 |
| 渡化道 | 菩萨 | 转职型，主要靠提前胜利 |

## 快速开始

1. 如果你是**新玩家**：先阅读 `01_玩家规则书_本体.txt`
2. 熟悉基础规则后，再阅读 `02_玩家规则书_转职扩展.txt`
