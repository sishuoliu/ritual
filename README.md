# 功德轮回 (Merit Cycle)

**一款融合佛教主题的半合作桌游**

## 版本

**v5.8 FINAL** - 2026-01-30

## 快速开始

所有发布文件在 `final/` 文件夹：

1. 打印 `print_pack_v58_FINAL.html`
2. 阅读规则 `功德轮回_完整规则书_v5.8_FINAL.md`
3. 准备1-2个骰子，2-4名玩家即可游玩

## 游戏特色

- **半合作机制**：团队共同对抗劫难，个人争夺排名
- **四大职业**：农夫、商人、学者、僧侣，各具特色
- **信仰系统**：不皈依→皈依→大乘发心三层选择
- **道德抉择**：事件卡包含历史佛教故事与道德选择

## 平衡性验证

基于5000局Monte Carlo模拟：

| 指标 | 基础版 | 人间炼狱版 |
|------|--------|-----------|
| 团队胜率 | 74.4% | ~46% |
| 职业第1名率 | 12-23% | 7-16% |
| 护法使用 | 0.72/人/局 | 1.23/人/局 |
