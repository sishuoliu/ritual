## 游戏概述

《功德轮回：众生百态》是一款2-4人半合作桌游，游戏时长30-45分钟。

### 核心玩法
- **团队目标**：6回合内渡化≥5个众生，劫难≤12
- **个人目标**：团队胜利后，福+慧最高者获胜
- **核心抉择**：何时为集体牺牲？何时为自己积累？

### 主要特色
- **三层信仰体系**：不皈依 → 小乘 → 大乘
- **四大职业**：农夫、商人、学者、僧侣，各有独特技能
- **佛教故事事件**：贫女一灯、割肉喂鹰等经典典故
- **动态皈依系统**：可在游戏任意时刻选择皈依或发大乘心
- **天灾两难抉择**：每人独立选择，同时亮牌，体现囚徒困境
