# 佛教经济学桌游项目

**Buddhist Economics Board Game Project**

基于学术论文《救赎的经济学：中国佛教交换理论》(The Economics of Salvation: Toward a Theory of Exchange in Chinese Buddhism) 设计的策略桌游。

---

## 项目包含两个游戏系列

### 1. 救赎之路 (Path to Salvation)
竞争型策略桌游，玩家扮演唐宋时期不同身份角色，通过资源管理和行动选择积累功德。

### 2. 功德轮回：众生百态 (Merit Cycle: Lives of All Beings) ★新★
**半合作型**策略桌游，玩家合作拯救众生、化解劫难，同时竞争个人修行果位。

## 功德轮回：众生百态 v3.6 ★当前定稿★

### 核心特色（v3.6）

| 特色 | 说明 |
|------|------|
| **半合作** | 团队需劫难≤12 且 渡化≥6 才胜利，个人按福慧+发愿计分竞争 |
| **渡化差异化** | 农夫/商人/学者/僧侣渡化成本不同，慧≥5 方可渡化 |
| **发愿系统** | 每角色 2 张发愿，每回合奖励，结束时达成则加分、未达成扣分 |
| **事件卡 v3.6** | 8 张共享事件（旱灾/洪水/瘟疫/丰收/法会/高僧开示/国泰民安/浴佛节） |
| **众生卡** | 8 张众生，基础成本+福慧奖励；滞留 2 回合未渡化则劫难+3 |
| **投资与持续帮助** | 投资 5 财每 2 回合+2 财；连续 3 回合帮助+1 福 |
| **回测与平衡** | 模拟器+回测系统，可复现调优（见 模拟测试/README_回测与v3.6.md） |

### 胜利条件（v3.6）

- **团队胜利**：游戏结束时 劫难≤12 且 渡化计数≥6。
- **个人得分**：仅当团队胜利时，按 √(福×慧)×3 + 皈依加分 + 发愿 计分排名。

### 四种身份

| 身份 | 初始财富/福/慧 | 渡化成本特点 |
|------|----------------|--------------|
| 农夫 | 5/2/2 | 标准财富成本 |
| 商人 | 8/1/1 | 多付 1 财 |
| 学者 | 3/1/4 | 少付 1 财、付 1 慧 |
| 僧侣 | 0/3/3 | 少付 1 财、付 1 福 |

## 快速开始

### 救赎之路
1. **新手玩家**：从 `终版/救赎之路-简易版/` 开始
2. **策略玩家**：查看 `终版/救赎之路-转职版/`

### 功德轮回（v3.6）
1. **玩家规则**：阅读 `设计提案/功德轮回_规则说明书_v3.6.txt`
2. **设计与平衡**：阅读 `设计提案/功德轮回_v3.6_最终版_回测与平衡报告.txt`
3. **回测与调优**：阅读 `模拟测试/README_回测与v3.6.md`；运行调优：在**仓库根目录**执行 `python run_merit_cycle_tuning.py`

### 开发者
- **功德轮回**：模拟器与回测在 `模拟测试/`（game_simulator_v3.5_final.py、backtest_system.py、balance_tuning_v36.py）
- **救赎之路**：模拟器在 `01_代码_模拟器/`（若存在），测试数据在 `04_测试数据/`

- 简易版：经过 v1–v6 迭代，v6.3 达成平衡

## 佛教概念整合

| 佛教概念 | 游戏机制 |
|----------|----------|
| 四圣谛 | 劫难(苦)、三毒(集)、修行阶段(灭)、行动选择(道) |
| 三学 | 戒律系统(戒)、修行行动(定)、渡化众生(慧) |
| 六度 | 布施、持戒、忍辱、精进、禅定、般若 |
| 布施波罗蜜 | 财布施、法布施、无畏施 |
| 正命 | 正当交易 vs 高利盘剥 |
