                          第一部分：骰子系统概览
═══════════════════════════════════════════════════════════════════════════════

【设计理念】
佛教强调"因缘和合"与"无常变化"，骰子完美体现了这一哲学：
• 业力：过去的因，影响现在的果
• 修行：精进与否，成效不同
• 因缘：人与人之间的缘分深浅
• 无常：一切皆变，随时可能顿悟

【四骰体系】

┌───────────┬─────────────┬─────────────────────────────────┐
│   骰子    │   时机      │   作用                          │
├───────────┼─────────────┼─────────────────────────────────┤
│  业力骰   │  每轮开始   │   决定本轮运势，影响行动点      │
│  修行骰   │  功德行动   │   决定修行成效，影响功德获取    │
│  因缘骰   │  玩家互动   │   决定缘分深浅，影响互动效果    │
│  无常骰   │  生命阶段   │   可能触发顿悟，获得觉醒令牌    │
└───────────┴─────────────┴─────────────────────────────────┘

                          第二部分：各骰子详细规则
═══════════════════════════════════════════════════════════════════════════════

▌业力骰（每轮开始掷）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【时机】每轮开始时，所有玩家同时掷业力骰

【结果表】
┌─────────┬──────────┬────────────────────────────────────┐
│  点数   │   结果   │   效果                             │
├─────────┼──────────┼────────────────────────────────────┤
│   1-2   │   逆境   │   本轮行动点-1（最少保留1点）      │
│   3-4   │   平常   │   无特殊效果                       │
│   5-6   │   顺缘   │   本轮行动点+1                     │
└─────────┴──────────┴────────────────────────────────────┘

【主题解读】
业力代表过去行为的累积影响。
有时善业成熟，诸事顺遂；有时恶业现前，处处受阻。

【战略影响】
• 逆境时应集中资源做最重要的事
• 顺缘时可以多尝试高风险高回报的行动

───────────────────────────────────────────────────────────────────────

▌修行骰（功德行动时掷）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【时机】执行以下行动时掷修行骰：
• 捐献（普通/匿名）
• 授法（僧侣）
• 大布施（富商）
• 捐献土地/建寺（地主）
• 日行一善（农民）

【结果表】
┌─────────┬──────────────┬────────────────────────────────┐
│  点数   │    结果      │   效果                         │
├─────────┼──────────────┼────────────────────────────────┤
│    1    │  修行受阻    │   功德减半或无效               │
│   2-4   │  修行平常    │   正常效果                     │
│    5    │  修行精进    │   功德+50%                     │
│    6    │  大彻大悟    │   功德翻倍或更多，额外+3分     │
└─────────┴──────────────┴────────────────────────────────┘

【大彻大悟特效】
掷出6时，玩家体验"顿悟"时刻：
• 捐献3金→获得3功德（而非1）
• 匿名捐献4金→获得5功德（而非2）
• 僧侣授法→可授予6点法力
• 建寺→获得18功德（而非12）

【主题解读】
修行不是机械的行为，而是心与法的相应。
同样的布施，用心不同，功德迥异。
偶尔的"大彻大悟"代表修行者突破境界的关键时刻。

───────────────────────────────────────────────────────────────────────

▌因缘骰（玩家互动时掷）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【时机】执行以下行动时掷因缘骰：
• 渡众生（寡妇菩萨状态）
• 为民请命（官员）
• 分享收成（农民）

【结果表】
┌─────────┬──────────────┬────────────────────────────────┐
│  点数   │    结果      │   效果                         │
├─────────┼──────────────┼────────────────────────────────┤
│   1-2   │    缘浅      │   互动效果减半                 │
│   3-4   │  普通因缘    │   正常效果                     │
│   5-6   │  深厚因缘    │   互动效果+50%                 │
└─────────┴──────────────┴────────────────────────────────┘

【深厚因缘特效】
掷出5-6时，代表与对方有深厚的前世因缘：
• 渡众生：赋予6功德（而非4）
• 为民请命：获得5功德（而非3）
• 分享收成：双方都获得额外资源

【主题解读】
佛教讲"百年修得同船渡，千年修得共枕眠"。
人与人之间的因缘有深有浅，这决定了互动的效果。

───────────────────────────────────────────────────────────────────────

▌无常骰（生命阶段掷）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【时机】每轮结束生命减少时，所有玩家掷无常骰

【结果表】
┌─────────┬──────────────┬────────────────────────────────┐
│  点数   │    结果      │   效果                         │
├─────────┼──────────────┼────────────────────────────────┤
│   1-3   │    无事      │   无特殊效果                   │
│   4-5   │    顿悟      │   获得1个觉醒令牌              │
│    6    │   大顿悟     │   获得2个觉醒令牌+2功德        │
└─────────┴──────────────┴────────────────────────────────┘

【觉醒令牌用途】
消耗1个觉醒令牌可以：
• 立即获得4功德
• 可在任意行动阶段使用

【主题解读】
"诸行无常，是生灭法。"
每一次生命的流逝都是对无常的体悟。
有时这种体悟能带来顿悟，加速修行进程。

▌10000局测试数据
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【每局平均骰子事件】
• 大彻大悟（修行骰掷6）：5.58次
• 大顿悟（无常骰掷6）：9.99次
• 深厚因缘（因缘骰掷5-6）：4.33次

【概率分析】
• 修行骰掷出6的概率：16.67%
• 每局约34次功德行动 × 16.67% ≈ 5.6次大彻大悟
• 无常骰每轮6人 × 10轮 × 16.67% ≈ 10次大顿悟

└─────────┴─────────┴─────────┴──────────────────────────────┘

【分析】
骰子系统对游戏平衡产生了显著影响：

1. 【农民崛起】
   - 农民执行最多的功德行动（日行一善、分享收成）
   - 更多行动 = 更多骰子机会 = 更多"大彻大悟"
   - 勤劳道的高完成率使农民获益于骰子加成

2. 【富商/地主翻身】
   - 他们的大额捐献行动在"大彻大悟"时获得巨额回报
   - 骰子增加了他们的上限

3. 【僧侣衰落】
   - 僧侣的授法行动虽然也有骰子，但次数较少
   - 原本稳定的涅槃道优势被骰子的随机性抵消

4. 【寡妇/官员下降】
   - 这两个角色的行动较为特化
   - 骰子带来的波动对他们不利

▌骰子使用仪式感
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【业力骰】
• 每轮开始，所有玩家同时掷骰
• 大声宣布结果："顺缘！"/"逆境..."
• 增加游戏开始时的期待感

【修行骰】
• 执行功德行动前，先宣布行动
• 然后掷骰，根据结果调整效果
• 掷出6时庆祝"大彻大悟！"

【因缘骰】
• 与特定玩家互动前掷骰
• 结果代表两人的因缘
• 可以增加角色扮演元素

【无常骰】
• 每轮结束时，所有人同时掷
• 掷出高点的玩家可以分享"顿悟体验"
• 增加轮末的仪式感

───────────────────────────────────────────────────────────────────────

▌可选变体规则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【简化版】
只使用修行骰和无常骰，减少骰子频率

【硬核版】
• 业力骰1：本轮无法执行功德行动（业障深重）
• 修行骰1：功德行动完全失败，浪费资源
• 因缘骰1：互动产生负面效果（结怨）

【合作版】
• 深厚因缘时，双方都获得功德
• 鼓励玩家之间的良性互动

【设计评价】

主题契合度：★★★★★（5/5）
• 业力、因缘、无常都是核心佛学概念
• 骰子完美体现了"不可控"与"因果相续"

趣味性增加：★★★★★（5/5）
• 每次行动都有悬念
• "大彻大悟"时刻创造高潮
• 增加了玩家之间的互动话题

平衡性影响：★★★☆☆（3/5）
• 增加了弱势角色的翻盘机会
• 但也增加了游戏的不确定性
• 可能需要进一步调整基础数值

复杂度增加：★★★☆☆（3/5）
• 每轮需要掷多次骰子
• 可能延长游戏时间
• 新手可能感到复杂

【最终建议】
骰子系统成功增加了游戏的随机性和趣味性。
建议在正式版本中采用，但可以：
1. 提供"简化模式"（只用修行骰和无常骰）
2. 调整基础数值使六角色更均衡
3. 在规则书中强调骰子的仪式感
