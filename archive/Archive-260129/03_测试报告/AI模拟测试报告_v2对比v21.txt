▌问题1：寡妇仍然过强的原因
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【数据】寡妇平均功德约35（v2.1），远超解脱门槛20

【原因分析】
寡妇特权：
1. 不能访问皇宫 → 声望自然=0
2. 不能购买土地 → 资源自然低
3. 仍有回向+1功德
4. 仍有初始功德2

这使寡妇成为"天生解脱候选人"——她的限制反而成为优势！

【根本问题】
解脱条件与寡妇的限制高度契合，几乎是为寡妇量身定制。

───────────────────────────────────────────────────────────────────────

▌问题2：解脱条件仍然太容易
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【v2.1解脱条件】
• 功德 ≥ 20
• 声望 = 0
• 资源 ≤ 3

【问题】
• 功德20仍然太低，寡妇平均35远超
• 声望=0对寡妇来说是默认状态（限制变优势）
• 资源≤3对没有土地的角色很容易

───────────────────────────────────────────────────────────────────────

▌问题3：标准胜利从未发生
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【问题】
两个版本中，标准胜利（比点数）都是0%

【原因】
只要有人达成解脱，标准胜利就不可能发生。
而每局都至少有1人达成解脱。

【设计反思】
解脱作为"直接获胜"条件太有吸引力，
使得追求高分变得毫无意义。

▌核心修改1：重新设计解脱机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方案A：解脱改为加分而非直接获胜】（推荐）

取消"达成解脱直接获胜"规则。
改为：达成解脱者获得+30分加成。

解脱条件（保持v2.1）：
• 功德 ≥ 20
• 声望 = 0
• 资源 ≤ 3

效果：
• 解脱者仍有优势，但需要与高分策略竞争
• 地主的高资源策略重新有意义
• 标准胜利成为可能

【方案B：大幅提高解脱门槛】

解脱条件收紧：
• 功德 ≥ 30（从20提高）
• 声望 = 0
• 资源 = 0（从≤3收紧到=0）
• 必须放弃所有物质

───────────────────────────────────────────────────────────────────────

▌核心修改2：寡妇差异化解脱条件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

由于寡妇的限制与解脱条件重合，可以：

【方案】寡妇解脱门槛提高

• 寡妇解脱需要：功德 ≥ 30（其他角色20）
• 理由：寡妇获得功德更容易，门槛应更高

───────────────────────────────────────────────────────────────────────

▌核心修改3：增加标准胜利吸引力
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【得分公式再调整】

新公式：救赎点 = 功德×2 + 声望×3 + 资源÷2

• 声望权重从×2提高到×3
• 鼓励"声望路线"与"解脱路线"竞争
• 官员和地主的策略更有吸引力

【v2.2 核心修改】

1. 解脱机制改革（方案A）
   • 达成解脱：+30分加成（而非直接获胜）
   • 解脱条件不变：功德≥20，声望=0，资源≤3
   
2. 寡妇差异化门槛
   • 寡妇解脱需要：功德 ≥ 30
   
3. 涅槃机制改革
   • 达成涅槃：+30分加成（而非直接获胜）
   • 涅槃条件不变：法力≤2，授德点≥15
   
4. 得分公式调整
   • 新公式：功德×2 + 声望×3 + 资源÷2

【保持v2.1的修改】
• 寡妇哀悼祈福：每2轮1功德
• 寡妇回向：+1功德
• 僧侣闭关机制
• 地主施田济寺
• 匿名捐献选项

                          结论与建议
═══════════════════════════════════════════════════════════════════════════════

【核心结论】

v2.0问题：解脱太容易，农民意外获胜多
v2.1问题：寡妇成为"天选解脱者"

根本问题：解脱作为"直接获胜"条件设计不当

【最终建议】

采用v2.2方案：
• 解脱/涅槃改为+30分加成
• 寡妇解脱门槛提高到功德≥30
• 声望权重提高

这将创造多元获胜路径：
1. 解脱路线：高功德+低声望+低资源+30加分
2. 声望路线：高声望+中等功德
3. 囤积路线：高资源+中等功德
4. 涅槃路线（僧侣）：高授德点+30加分

各路线互有胜负，增加策略深度和重玩价值。
