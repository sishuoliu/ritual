═══════════════════════════════════════════════════════════════════════════════
                          第一章：修行阶段概述
═══════════════════════════════════════════════════════════════════════════════

【佛教背景】
修行有多个果位阶段：
• 凡夫 → 须陀洹（初果）→ 斯陀含（二果）→ 阿那含（三果）→ 阿罗汉（四果）
• 或 凡夫 → 菩萨（初地）→ 菩萨（二地）...→ 佛果

【游戏化设计】
每个玩家有独立的"修行阶段"轨道，共4个阶段：

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│   阶段1     →    阶段2     →    阶段3     →    阶段4                │
│   凡夫          初果          菩萨          圆满                    │
│                                                                      │
│  起始状态   条件：功德≥15  条件：功德≥25  条件：功德≥35            │
│  无加成     升级成本        升级成本        升级成本                │
│            舍弃+骰子       舍弃+骰子       舍弃+骰子              │
└──────────────────────────────────────────────────────────────────────┘

【核心机制】
1. 所有人起始都是"阶段1：凡夫"
2. 达到条件后，可选择"尝试升级"
3. 升级流程：舍弃资源 → 掷骰子 → 成功/失败
4. 成功：升级+消耗额外资源+获得新加成
5. 失败：停留原阶段，舍弃的资源不返还

▌阶段1：凡夫（Starting Stage）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【状态】
• 所有人起始阶段
• 无特殊加成
• 佛典：在轮回中挣扎的凡夫俗子

【条件】
• 无


▌阶段2：初果（Stream-Enterer）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【升级条件】
• 功德≥15
• 三毒总和≤10

【升级流程】
1. 【舍弃资源】
   • 舍弃3财富+2影响（表示放下执着）
   
2. 【掷骰判定】
   • 掷2d6
   • 2-6：失败，停留凡夫阶段，舍弃的资源不返还
   • 7-9：成功，进入初果阶段
   • 10-12：大成功，进入初果阶段+额外奖励（+2功德）
   
3. 【成功后额外消耗】
   • 再消耗5财富+3影响（表示修行投入）
   
4. 【骰子修正】
   • 僧侣：+2骰子（修行专家）
   • 戒律=10：+1骰子
   • 虔诚≥15：+1骰子
   • 其他玩家协助（消耗1行动为你祈福）：+1骰子

【阶段2加成】
• 每轮自动+1功德（见道）
• 修行行动效果+1（+3变+4）
• 三毒增长速度-50%
• 佛典：须陀洹果，入圣道流

【成功率】（无修正）
• 2-6失败：13.9%
• 7-9成功：41.7%
• 10-12大成功：16.7%
• 总成功率：58.4%


▌阶段3：菩萨（Bodhisattva）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【升级条件】
• 功德≥25
• 渡化众生≥2个
• 护法贡献≥8

【升级流程】
1. 【舍弃资源】
   • 舍弃5财富+3影响+2功德
   
2. 【掷骰判定】
   • 掷2d6
   • 2-5：失败，停留初果阶段（16.7%）
   • 6-8：成功，进入菩萨阶段（44.4%）
   • 9-12：大成功，进入菩萨阶段+额外奖励（+3功德+2影响）（38.9%）
   
3. 【成功后额外消耗】
   • 再消耗8财富+5影响+3功德
   
4. 【骰子修正】
   • 僧侣：+2骰子
   • 渡化众生≥3：+1骰子
   • 其他玩家协助：+1骰子/人（最多+2）

【阶段3加成】
• 每轮自动+2功德（菩提心）
• 所有"渡化众生"效果翻倍
• 善行行动：受益者+双倍效果
• 护法行动：劫难额外-1
• 佛典：发菩提心，利益众生

【成功率】（无修正）
• 总成功率：83.3%

• 佛典：无上正等正觉，圆满佛果

▌4.1 升级的风险/收益分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【升级到初果】
• 舍弃：3财富+2影响
• 失败率：41.6%
• 成功额外消耗：5财富+3影响
• 总投入：8财富+5影响
• 回报：每轮+1功德（8轮=+8功德）+ 修行+1

【期望分析】
• 若成功（58.4%）：8轮获得至少+8功德，值得投资
• 若失败（41.6%）：损失3财富+2影响，但可以再次尝试
• 结论：初期应该尝试升级

【升级到菩萨】
• 舍弃：5财富+3影响+2功德
• 失败率：16.7%
• 成功额外消耗：8财富+5影响+3功德
• 总投入：13财富+8影响+5功德
• 回报：每轮+2功德 + 渡化效果×2 + 护法-1

【期望分析】
• 成功率高（83.3%），但投入巨大
• 如果中后期（第4-5轮）升级，剩余4轮获得+8功德
• 渡化效果翻倍能加速完成胜利条件
• 结论：中期值得投资，但需要团队配合提供资源


▌4.2 策略选择
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【激进策略】
• 第2轮尝试升初果
• 第4-5轮尝试升菩萨
• 风险：失败会浪费资源
• 适合：利他型玩家，愿意冒险

【保守策略】
• 第4轮才升初果（确保资源充足）
• 不升菩萨（投入太大）
• 风险：错过后期加成
• 适合：自利型玩家

【团队配合策略】
• 1-2名核心玩家冲刺高阶段
• 其他玩家提供资源支持
• 高阶段玩家承担主要护法任务
• 适合：混合型团队

【其他职业发愿】
类似调整，整合修行阶段要求


▌5.2 胜利条件整合
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【原胜利条件】
1. 劫难≤12
2. 渡化众生≥5
3. 建寺≥2
4. 所有玩家功德≥12

【新增条件】
5. 至少1名玩家达到阶段3或以上（必须有圣人领路）

【佛教意义】
• 不是所有人都要成圣
• 但团队中必须有"觉悟者"
• 体现"以觉悟者带领众生"的佛教理念

                          第六章：游戏流程更新

▌每轮增加"修行升级阶段"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【修改后的轮流程】

1. 协商阶段（3分钟）
   • 讨论本轮策略
   • 讨论是否有人要尝试升级
   • 其他人是否协助

2. 行动阶段（同时选择，5分钟）
   • 每人选择2个行动

3. **修行升级阶段**（新增，2分钟）
   • 满足条件的玩家可选择"尝试升级"
   • 舍弃资源 → 掷骰子 → 结算
   • 其他玩家可选择"协助"（消耗1行动为其祈福，+1骰子）

4. 事件阶段（5分钟）
   • 翻开众生卡
   • 专属事件
   • 共享事件

5. 劫难阶段（2分钟）
   • 劫难递增
   • 检查灾难效果
   • 检查众生超时

▌修行阶段系统对游戏平衡的影响
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【正面影响】
1. ✓ 增加功德获取途径（高阶段每轮+2~+3功德）
2. ✓ 增加护法能力（高阶段劫难-1~-2/轮）
3. ✓ 增加决策深度（何时升级？是否冒险？）
4. ✓ 增加合作互动（协助他人升级）
5. ✓ 增加佛教体验（修行证果的核心感）

【负面影响】
1. ⚠ 增加规则复杂度
2. ⚠ 升级失败会浪费资源，可能导致团队失败
3. ⚠ 需要重新测试平衡性

【预期影响】
• 胜率应该从0%提升到30-50%（因为高阶段加成）
• 利他型/平衡型胜率提升更明显
• 自利型玩家可能因升级失败浪费资源
