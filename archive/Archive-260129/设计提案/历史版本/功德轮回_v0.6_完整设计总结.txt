▌2.1 游戏模式定稿：半合作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【用户最终选择】
方案B：默认半合作（无破坏性发愿）

理由：
1. ✓ 保留合作的佛教精神（团队必须拯救众生）
2. ✓ 增加个人竞争动力（争取最高果位）
3. ✓ 创造道德困境（牺牲个人vs利他）
4. ✓ 避免破坏性行为（删除灭佛等冲突发愿）
5. ✓ 符合佛教"弟子有高低"的现实

【半合作特征】
• 合作比例：70%（必须合作拯救世界）
• 竞争比例：30%（个人果位有高低）

▌2.2 双层胜利系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一层：团队胜利（门槛）】

必须全部达成：
1. 劫难≤12
2. 渡化众生≥5
3. 建寺≥2
4. 所有玩家功德≥12
5. 至少1人达到修行阶段3

→ 若未达成：全员失败，无人获胜
→ 若达成：进入第二层，比较个人成就

【第二层：个人成就（竞争）】

个人分数 = 功德×2 + 发愿分 + 修行阶段分 + 成就分 - 惩罚

【果位评定】
• 佛果（55+分）：★唯一佛果★ 游戏胜者
• 菩萨果（45-54分）：1-2名玩家
• 阿罗汉果（35-44分）：2-3名玩家
• 初果（25-34分）：其余玩家
• 凡夫（<25分）：虽团队胜利，但个人修行不足

【关键规则】
• 只有最高分者获得"佛果"称号
• 其他人虽然"得救"了，但果位较低
• 团队失败，所有人都无果位

▌系统1：修行阶段系统（4阶段）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1】凡夫 → 所有人起始
【阶段2】初果/居士 → 舍弃资源+骰子判定
【阶段3】菩萨/阿罗汉 → 舍弃更多资源+骰子
【阶段4】圆满/佛果 → 舍尽求法

【升级流程】
1. 满足条件（功德/戒律/虔诚等）
2. 舍弃资源（立即失去，无论成败）
3. 掷骰子判定（2d6+职业修正）
4. 成功→额外消耗+获得加成
5. 失败→停留原阶段，舍弃不返还

【设计意义】
• 高风险高回报
• 体现"舍弃执着"
• 每职业路径不同


▌系统2：劫难轨道系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【机制】
• 劫难轨道（0-50）
• 每轮+4劫难
• 超过阈值触发三灾效果
• 劫难≥50：全员失败

【三灾】
• 刀兵劫（劫难20）：随机玩家-3财富
• 瘟疫劫（劫难30）：所有人-2功德
• 饥馑劫（劫难40）：所有人-4财富

【降低劫难方法】
• 护法行动：-3~-5
• 讲经：-3
• 渡化众生：-3
• 高阶段加成：自动-1~-2


▌系统3：众生渡化系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【众生卡】20张
• 按需求分类：饥饿、疾病、愚痴
• 需要玩家合作完成渡化
• 超时未渡化：劫难+5

【渡化条件示例】
• 战乱孤儿：8财富
• 瘟疫患者：5财富+3影响
• 迷途商人：2功德+讲经

【奖励】
• 劫难-3
• 参与者+功德
• 团队胜利条件之一


▌系统4：发愿系统（v0.6更新）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【发愿类型】（共12张）

利他型（4张）：对团队帮助大
• 地藏宏愿、护法檀越、布施长者、慈济苍生
• 奖励：18-25分

平衡型（6张）：福慧双修
• 菩萨道、功德商人、明君护法等
• 奖励：15-20分

自利型（2张）：对团队帮助小
• 阿罗汉道、富甲天下
• 奖励：10-15分

【设计决策】
• ✗ 删除冲突型发愿（灭佛等）
• ✓ 保留合作精神
• ✓ 竞争仅体现在"谁修行更高"


▌系统5：三毒系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【三毒】贪、嗔、痴

【触发条件】
• 贪：贪婪交易、高利贷、囤积财富
• 嗔：违背承诺、恶意抬价、争吵
• 痴：接受不利交易、被盘剥

【惩罚】
• 贪≥5：商人阶段2骰子-1
• 嗔≥5：无法升阶段3
• 痴≥5：所有升级骰子-1
• 任意≥10：最多只能得凡夫

【降低三毒】
• 布施：-1贪
• 履行承诺：-1嗔
• 升级成功：-2贪/-1嗔
• 阶段4：免疫三毒


▌系统6：交易系统（v0.6新增）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【交易时机】
• 仅在"协商阶段"（3分钟）
• 行动阶段不可交易

【可交易资源】
✓ 财富、影响（每轮≤5）、土地、商路收益共享
✗ 功德、修行阶段、三毒、发愿卡

【交易类型】

1. 即时交易：双方立即交换资源
2. 无偿布施：给予者+功德（根据数量）
3. 承诺交易：约定未来行动，违背+2嗔
4. 借贷：约定本金+利息，不还+3贪

【布施功德表】
• 1-2财富：+1功德
• 3-5财富：+2功德
• 6-10财富：+4功德
• 10+财富：+6功德

【三毒风险】
• 高利贷（利息>20%）：放贷者+2贪
• 违背承诺：+2嗔
• 接受不利交易：+1痴


▌系统7：骰子判定系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基础】2d6（总和2-12）

【概率分布】
• 7最常见（16.7%）
• 2/12最稀有（2.8%）

【修正因素】
• 职业专长：僧侣+2，农夫+1
• 他人协助：每人+1（最多+2~+3）
• 三毒影响：贪/痴≥5时-1
• 特殊条件：寺庙+1，虔诚+1

【应用场景】
• 修行阶段升级
• 部分事件卡判定
• 众生渡化判定

                         第五章：游戏流程
═══════════════════════════════════════════════════════════════════════════════

【游戏时长】60-90分钟
【玩家人数】4人（基础版）
【总轮数】8轮

【每轮5个阶段】

阶段1：协商阶段（3分钟）
├── 讨论本轮策略
├── 协商交易（v0.6新增）
├── 讨论升级计划
└── 商议资源分配

阶段2：行动阶段（5分钟）
├── 每人选择2个行动
├── 基础行动：修行、布施、善行、护法、建寺
└── 专属行动：根据身份

阶段3：修行升级阶段（3分钟）
├── 检查升级条件
├── 舍弃资源
├── 他人协助
├── 骰子判定
└── 成功/失败结算

阶段4：事件阶段（5分钟）
├── 翻开2张众生卡
├── 每人抽1张专属事件卡
└── 翻开1张共享事件卡

阶段5：劫难阶段（2分钟）
├── 劫难+4
├── 检查灾难效果
├── 检查众生超时
└── 检查失败条件

═══════════════════════════════════════════════════════════════════════════════

【四圣谛】
• 苦：劫难轨道，众生受苦
• 集：三毒（贪嗔痴）
• 灭：修行阶段，证得果位
• 道：八正道→游戏中的行动选择

【三学】
• 戒：戒律系统（僧侣）
• 定：修行行动
• 慧：渡化众生，开悟

【六度】
• 布施：布施行动
• 持戒：戒律系统
• 忍辱：承受负面事件
• 精进：修行升级
• 禅定：闭关修行
• 般若：高阶段加成

【四弘誓愿】
• 众生无边誓愿度：渡化众生系统
• 烦恼无尽誓愿断：三毒系统
• 法门无量誓愿学：修行阶段
• 佛道无上誓愿成：证得佛果

【布施波罗蜜】（交易系统）
• 财布施：无偿给予财富
• 法布施：僧侣讲经
• 无畏施：救治众生

【正命】（交易系统）
• 正当贸易是八正道之一
• 惩罚高利贷、欺诈
• 鼓励诚信交易

                         第八章：测试结果与平衡性
═══════════════════════════════════════════════════════════════════════════════

【v0.4测试结果】（无修行阶段）
• 测试次数：600局
• 胜率：0%（所有配置）
• 主要失败原因：玩家功德<5（82%）
• 诊断：游戏过难，不可能获胜

【v0.5修正方案】
1. 增加功德获取：修行+3（原+2），讲经+2（原+1）
2. 减缓劫难：每轮+4（原+5）
3. 放宽失败条件：≥2人功德<5才失败
4. 增强护法效果：各+1
5. 降低众生难度：财富需求-20%
6. 增加初始资源：所有人功德+1
7. ★核心补充★：加入修行阶段系统

【预期胜率】（v0.6）
• 全利他型：60-75%
• 全平衡型：45-60%
• 混合型：50-65%
• 全自利型：<20%

                         结语
═══════════════════════════════════════════════════════════════════════════════

从v0.1的纯竞争游戏，到v0.6的半合作游戏，
《功德轮回：众生百态》经历了一次**根本性的设计转变**。

这个转变源于一个简单而深刻的问题：
"我们这是佛教游戏，真的要这样竞争么？"

最终设计成功整合了：
• 合作拯救众生（慈悲）
• 个人修行证果（智慧）
• 自利与利他的张力（中道）
• 舍弃与升级的抉择（放下）
• 三毒与觉悟的对抗（戒定慧）
• 交易与布施的道德困境（正命）

**v0.6的核心定位：**
• 团队必须合作（否则全员失败）
• 个人可以竞争（争取最高果位）
• 竞争方式健康（无破坏性行为）
• 佛教精神完整（慈悲+智慧+中道）

这不仅是一款桌游，更是一次完整的佛教经济学体验。
