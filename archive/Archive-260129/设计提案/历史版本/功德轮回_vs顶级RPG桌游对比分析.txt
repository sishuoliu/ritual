▌2.1 战斗/行动系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌────────────────┬────────────────────────────────────────────────────────────┐
│  Gloomhaven    │  【卡牌手牌管理系统】                                      │
│                │  • 每回合选2张卡，用1张上半+1张下半                        │
│                │  • 卡牌分"弃牌"、"失去"、"激活"三种状态                   │
│                │  • 手牌耗尽=力竭退场                                       │
│                │  • 攻击修正牌堆替代骰子                                    │
│                │  ★ 核心张力：手牌是生命，每次行动都在消耗生存资源         │
├────────────────┼────────────────────────────────────────────────────────────┤
│  Root          │  【派系专属行动系统】                                      │
│                │  • 每派系有完全不同的行动阶段流程                          │
│                │  • 猫族：建造-运营-行军                                    │
│                │  • 鸟族：法令系统，必须执行所有法令否则政府崩溃            │
│                │  • 联盟：同情标记蔓延，游击战术                            │
│                │  ★ 核心张力：不同派系的胜利路径完全不同                   │
├────────────────┼────────────────────────────────────────────────────────────┤
│  Spirit Island │  【快慢行动系统】                                          │
│                │  • 行动分"快速"和"慢速"两种                               │
│                │  • 快速行动在殖民者行动前执行                              │
│                │  • 慢速行动在殖民者行动后执行                              │
│                │  ★ 核心张力：时间顺序决定策略                             │
├────────────────┼────────────────────────────────────────────────────────────┤
│  功德轮回      │  【简单行动点系统】                                        │
│  (当前)        │  • 每轮2个行动点                                           │
│                │  • 选择执行布施/修行/善行等                                │
│                │  • 无手牌管理                                              │
│                │  ✗ 缺失：行动没有"重量感"，选择之间缺乏紧张感             │
└────────────────┴────────────────────────────────────────────────────────────┘

【差距分析】
我们的行动系统过于简单：
1. 缺乏Gloomhaven式的"行动=消耗生命"的紧张感
2. 缺乏Root式的"派系机制完全不同"的深度非对称
3. 缺乏Spirit Island式的"时间顺序"策略层

【改进建议】
★★★ 增加"精力系统"：每轮消耗精力执行行动，精力≤0时强制休息
★★☆ 为每个身份设计独特的行动阶段流程
★☆☆ 增加"急行动"和"缓行动"概念


▌2.2 角色发展系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌────────────────┬────────────────────────────────────────────────────────────┐
│  Gloomhaven    │  【完整成长系统】                                          │
│                │  • 经验值→升级→解锁新能力卡                               │
│                │  • 个人任务→完成后角色退休→解锁新职业                     │
│                │  • 装备系统：购买/出售/增强                                │
│                │  • 攻击修正牌堆个人化：添加祝福/诅咒                       │
│                │  ★ 成长是渐进且有实感的                                   │
├────────────────┼────────────────────────────────────────────────────────────┤
│  Sword&Sorcery │  【灵魂等级系统】                                          │
│                │  • 英雄有多个灵魂等级                                      │
│                │  • 死亡后变为灵魂形态，可被复活                            │
│                │  • 不同等级解锁不同能力                                    │
│                │  ★ 死亡不是终点，而是状态转变                             │
├────────────────┼────────────────────────────────────────────────────────────┤
│  功德轮回      │  【专属轨道系统】                                          │
│  (当前)        │  • 皇恩/戒律/声望/业力等轨道                               │
│                │  • 三乘升级系统                                            │
│                │  ✓ 有成长感                                               │
│                │  ✗ 缺乏装备/能力卡解锁                                    │
│                │  ✗ 缺乏"死亡/失败"后的状态转变                            │
└────────────────┴────────────────────────────────────────────────────────────┘

【差距分析】
我们有轨道成长，但：
1. 缺乏"装备/道具"层
2. 缺乏"新能力解锁"层
3. 缺乏"失败后转变"机制（如僧侣还俗→变农夫是好的开始）

【改进建议】
★★★ 增加"法器/圣物"装备系统（每身份可装备1-2件）
★★★ 增加"觉醒能力"：达成特定条件后解锁新能力
★★☆ 完善"身份转变"机制：不仅是惩罚，也是新机遇

我们的叙事系统：
1. 事件是"发生在你身上"而非"你创造的故事"
2. 缺乏Crossroads式的"条件触发叙事"
3. 缺乏Legacy式的"永久改变"

【改进建议】
★★★ 增加"因果业报"系统：早期选择影响后期事件
★★★ 增加"历史碑刻"：游戏结束后记录本局重大事件
★★☆ 设计"剧情链"：某些事件触发后续事件

▌5.1 精力系统（解决行动重量感）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
• 每人起始精力=8
• 每个行动消耗1-3精力
• 精力≤2时进入"疲惫"状态，行动效率-50%
• 精力=0时强制休息（跳过1轮，恢复至6）

【精力消耗表】
• 布施：1精力
• 修行：2精力
• 善行：1精力
• 回向：2精力
• 专属行动：2-3精力
• 升乘尝试：3精力

【恢复方式】
• 每轮自动+1精力
• 医者治疗：+2精力
• 休息轮：恢复至6


▌5.2 法难倒计时（解决时间压力）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
• 游戏开始时放置"法难轨道"
• 起始值=20
• 每轮自动-2
• 某些负面事件额外-1~-3
• 玩家可通过"护法"行动+1~+2

【法难触发】
• 法难轨道=0时：
  - 所有玩家-5功德
  - 所有寺庙摧毁
  - 僧侣还俗
  - 游戏提前结束

【意义】
• 创造共同威胁，促进合作
• 皇帝的"灭佛"会加速法难
• 全员有动力维持平衡


▌5.3 稀缺位置竞争（解决玩家互动）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
设置3个稀缺位置，每个位置只能有1人占据：

1. 【皇帝宠臣】
   • 条件：影响最高的非皇帝玩家
   • 效果：每轮+2影响，可建议皇帝事件选择
   • 变更：若被超越则失去

2. 【寺院方丈】
   • 条件：僧侣身份+建寺+功德最高
   • 效果：每轮+2功德，讲经效果翻倍
   • 变更：若功德被超越则失去

3. 【商会会长】
   • 条件：商人身份+商路最多
   • 效果：每轮+3财富，贸易效率+50%
   • 变更：若商路被超越则失去


▌5.4 因果业报系统（解决叙事创造感）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
• 设置"因果牌堆"（20张）
• 某些选择会让你抽取因果牌
• 因果牌在未来特定轮次触发

【因果牌示例】
• "布施业报"：若本局布施≥5次，第6轮+5功德
• "杀生业报"：若本局有杀生行为，第7轮-3功德
• "护法业报"：若阻止灭佛，最后1轮+8功德
• "欺诈业报"：若有欺骗行为，随机1轮失去3影响

【触发机制】
• 抽到因果牌后面朝下放置
• 到达指定轮次时翻开执行
• 创造"因果循环"的沉浸感


▌5.5 法器/圣物装备系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
• 每人可装备1件法器+1件圣物
• 法器：主动使用效果
• 圣物：被动加成效果

【法器示例】
• 念珠：消耗后+3功德
• 木鱼：修行效率+1
• 禅杖：可阻止1次负面事件
• 玉玺（皇帝专属）：颁诏成本-2

【圣物示例】
• 舍利子：每轮+1功德
• 菩提叶：戒律下降减半
• 金刚杵：免疫法难
• 如意：交易时+2财富

【获取方式】
• 建寺后可选1件圣物
• 完成特定事件可获法器
• 商人可购买（8财富/件）

                          结论
═══════════════════════════════════════════════════════════════════════════════

【总体评估】
《功德轮回》v0.2是一个有潜力的设计，但要达到顶级RPG桌游水平，
需要在以下4个核心领域进行强化：

1. ★★★ 行动系统：增加精力消耗机制
2. ★★★ 时间压力：增加法难倒计时
3. ★★★ 玩家互动：增加稀缺位置竞争
4. ★★★ 叙事深度：增加因果业报系统

完成这4项改进后，可进入v0.3版本测试。
