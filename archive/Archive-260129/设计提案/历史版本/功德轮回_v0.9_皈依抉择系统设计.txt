▌1.1 当前设计的问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【问题】
v0.8设计中，所有玩家默认就是"修行者"，追求功德。
这导致"皈依"没有成为一个选择，而是理所当然的事。

【缺失的抉择】
• 是否皈依三宝？
• 走出世修行路线还是在世行善路线？
• 追求功德还是追求世俗成功？

【佛教现实】
在真实历史中：
• 有人正式皈依三宝，成为居士或出家
• 有人做"善信"，布施护法但不正式皈依
• 有人完全追求世俗成功，但也可能晚年皈依
• 每个选择都有其意义和后果

▌1.2 设计目标
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 皈依成为一个【真正的选择】，而非默认状态
2. 皈依与不皈依都有【可行的胜利路线】
3. 选择有【佛学意义】和【游戏意义】
4. 增加游戏的【策略深度】和【道德张力】

▌2.1 起始状态：众生
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【新设定】
游戏开始时，所有玩家都是【众生】状态——
你可能是虔诚的信徒，也可能是世俗之人，
这取决于你游戏中的选择。

【起始资源调整】
起始不再有"功德"，改为"善缘"（中性资源）

┌──────────┬────────┬────────┬────────┬────────────────────────────────┐
│  身份    │  善缘  │  财富  │  影响  │  专属能力                      │
├──────────┼────────┼────────┼────────┼────────────────────────────────┤
│  僧侣    │   5    │   2    │   4    │  默认已皈依，讲经传法          │
│  贵族    │   2    │   10   │   6    │  未皈依，可选择皈依或护法      │
│  商人    │   2    │   8    │   3    │  未皈依，可选择皈依或行商      │
│  农夫    │   3    │   3    │   1    │  未皈依，可选择皈依或虔诚      │
└──────────┴────────┴────────┴────────┴────────────────────────────────┘

【僧侣特殊】
僧侣作为出家人，默认已皈依。
其他身份需要在游戏中做出选择。

▌2.2 皈依行动
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【皈依三宝】是一个可选行动

┌──────────────────────────────────────────────────────────────────────┐
│  【皈依三宝】                                                        │
│                                                                      │
│  条件：未皈依状态，善缘≥3                                           │
│  成本：消耗1个行动点 + 消耗3善缘                                    │
│  效果：                                                              │
│  • 状态变为"已皈依"                                                 │
│  • 所有善缘转化为功德（1:1）                                        │
│  • 获得"皈依加成"                                                   │
│  • 承诺"五戒约束"                                                   │
│                                                                      │
│  【时机】                                                            │
│  可在任意轮次的行动阶段执行                                          │
│  早皈依：长期收益高                                                  │
│  晚皈依：灵活但收益少                                                │
│  不皈依：走另一条路线                                                │
└──────────────────────────────────────────────────────────────────────┘

▌2.3 皈依后的加成与约束
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【皈依加成】
┌──────────────────────────────────────────────────────────────────────┐
│  1. 修行效率×1.5（修行行动从+2功德变为+3功德）                       │
│  2. 可以积累"功德"（未皈依者只能积累"善缘"）                        │
│  3. 可以获得"果位"（佛果、菩萨果等）                                │
│  4. 布施获得功德（未皈依者布施只获得善缘）                           │
│  5. 寺庙产出对你有效（每座寺庙+1功德）                               │
│  6. 可使用"回向"能力（将功德转给他人）                              │
└──────────────────────────────────────────────────────────────────────┘

【五戒约束】
┌──────────────────────────────────────────────────────────────────────┐
│  皈依后必须遵守五戒，违反则受惩罚：                                  │
│                                                                      │
│  1. 不杀生：不可使用"劫掠"等暴力行动（若有）                        │
│  2. 不偷盗：不可索要高利贷（>20%利息）                               │
│  3. 不妄语：违背承诺惩罚加倍（+4三毒而非+2）                         │
│  4. 不邪淫：（主题不涉及）                                           │
│  5. 不饮酒：不可拒绝合理的救助请求                                   │
│                                                                      │
│  【破戒惩罚】                                                        │
│  每次违反五戒：功德-3，三毒+2                                        │
│  若皈依后功德降到0以下：失去皈依状态，降为"破戒者"                  │
└──────────────────────────────────────────────────────────────────────┘

▌3.1 善缘与功德的区别
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【善缘】未皈依者积累的资源
• 代表善行带来的因果福报
• 不等同于佛教意义上的"功德"
• 但对团队胜利仍有贡献

【功德】已皈依者积累的资源
• 代表正式修行所得
• 可以"回向"给他人
• 与个人果位直接挂钩

┌──────────────────────────────────────────────────────────────────────┐
│                    善缘 vs 功德                                      │
│                                                                      │
│  维度          │  善缘（未皈依）      │  功德（已皈依）              │
│  ─────────────────────────────────────────────────────────────────  │
│  获取来源      │  善行、护法           │  修行、布施、渡化            │
│  修行效率      │  +2善缘/次            │  +3功德/次                   │
│  布施获得      │  3财富=+1善缘         │  3财富=+1功德                │
│  寺庙产出      │  无效果               │  +1功德/座/轮                │
│  回向能力      │  不可回向             │  可回向给他人                │
│  团队计算      │  计入团队目标         │  计入团队目标                │
│  个人果位      │  无果位               │  有果位评定                  │
│  约束          │  无五戒约束           │  五戒约束                    │
└──────────────────────────────────────────────────────────────────────┘

▌3.2 未皈依者的胜利路线
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【世俗成就路线】

未皈依者不追求"果位"，而追求"世俗成就"

┌──────────────────────────────────────────────────────────────────────┐
│  【世俗成就评定】（未皈依者专用）                                    │
│                                                                      │
│  个人得分 = 善缘×1 + 财富×0.5 + 影响×1 + 成就分                    │
│                                                                      │
│  世俗成就等级：                                                      │
│  ★ 护法大檀越（最高分）：虽未皈依，但护持三宝功德无量              │
│  ○ 善信居士（40+分）：善行满满                                      │
│  ○ 仁德商人（30-39分）：诚信经营                                    │
│  ○ 良善百姓（20-29分）：安分守己                                    │
│  ○ 平庸众生（<20分）：碌碌无为                                      │
└──────────────────────────────────────────────────────────────────────┘

【策略优势】
• 无五戒约束，行动更自由
• 可以高利贷、囤积、拒绝帮助
• 财富和影响直接计分
• 适合"护法不皈依"的策略

▌3.3 护法不皈依的历史依据
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【历史案例】
• 给孤独长者（须达）最初未正式皈依，但大力护持佛法
• 许多帝王如梁武帝，既是皇帝又护持三宝
• 现代很多企业家捐建寺庙但未正式皈依
• "外护"与"内修"是两条有效的佛教参与路径

【佛教意义】
护法不皈依是一种有效的善行
虽不能证得果位，但积累善缘，来世可能得度
这正是"未来佛"的种子

▌4.1 新团队胜利条件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【调整后的团队条件】
┌──────────────────────────────────────────────────────────────────────┐
│  必须全部达成：                                                      │
│                                                                      │
│  □ 劫难≤15                                                          │
│  □ 渡化≥5个众生                                                     │
│  □ 建寺≥2座                                                         │
│  □ 至少1人已皈依                          ← 新条件                  │
│  □ 所有人（功德+善缘）≥10                 ← 调整                    │
│                                                                      │
│  任一未达成 = 全员失败                                               │
└──────────────────────────────────────────────────────────────────────┘

【设计意义】
• "至少1人皈依"：团队中需要有正式的修行者
• "功德+善缘≥10"：皈依者和未皈依者都能贡献

▌4.2 个人胜利条件分流
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【已皈依者】
• 评定果位（佛果、菩萨果等）
• 最高功德者获得"佛果"
• 计分：功德×2 + 发愿分 + 成就分

【未皈依者】
• 评定世俗成就（护法大檀越等）
• 最高总分者获得"护法大檀越"
• 计分：善缘×1 + 财富×0.5 + 影响×1 + 成就分

【最终赢家】
┌──────────────────────────────────────────────────────────────────────┐
│  游戏结束时，比较：                                                  │
│  • 已皈依者中的最高分                                                │
│  • 未皈依者中的最高分                                                │
│                                                                      │
│  【佛果】始终是最高荣誉                                              │
│  但【护法大檀越】若分数超过所有皈依者，获得特殊称号：                │
│  "大护法"——虽未证果，但功德无量，来世必得度                        │
└──────────────────────────────────────────────────────────────────────┘

▌5.1 早期皈依（第1-2轮）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优势】
• 5轮的修行加成（+3功德/次 vs +2善缘/次）
• 5轮的寺庙产出
• 更高的最终功德

【劣势】
• 5轮的五戒约束
• 不能使用某些高收益世俗策略
• 消耗1个早期行动点

【适合角色】
• 僧侣（默认已皈依）
• 想走修行路线的农夫

▌5.2 中期皈依（第3-4轮）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优势】
• 前期可自由行动积累资源
• 后期仍能获得修行加成
• 灵活应对局势

【劣势】
• 修行加成轮数减少
• 可能错过寺庙产出
• 善缘转化时机需把握

【适合角色】
• 观望型商人
• 灵活型贵族

▌5.3 晚期皈依（第5-6轮）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优势】
• 几乎不受五戒约束
• 世俗资源最大化后转化
• 临终皈依仍有意义

【劣势】
• 修行加成很少
• 可能来不及积累足够功德
• 发愿可能无法达成

【佛学意义】
这对应"临终皈依"——佛教认为临终一念皈依仍有功德
但不如一生修行

▌5.4 始终不皈依
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优势】
• 完全不受五戒约束
• 可以最大化世俗资源
• 财富和影响直接计分
• 独特的"护法大檀越"路线

【劣势】
• 无法获得果位
• 修行效率低
• 寺庙产出对你无效

【适合角色】
• 专注护法的贵族
• 商业路线的商人

【佛学意义】
这对应"外护"角色——护持三宝，但不追求个人解脱
在佛教中同样有重要价值

▌6.1 皈依者发愿（原发愿卡）

▌6.2 未皈依者发愿（新增）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【未皈依者专用发愿】4张

┌──────────────────────────────────────────────────────────────────────┐
│  【护法大檀越】                                                      │
│  条件：参与建寺≥2座 + 从未皈依                                       │
│  分数：+15分                                                         │
│  佛学意义：护持三宝，功德无量                                        │

▌6.3 发愿选择规则调整
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【抽牌规则】
游戏开始时，每人抽3张发愿卡：
• 2张皈依者发愿
• 1张未皈依者发愿

选择保留1张，弃掉2张。

【策略意义】
• 你的发愿决定了你的路线倾向
• 选择皈依者发愿→倾向皈依
• 选择未皈依者发愿→倾向不皈依
• 但最终是否皈依仍可在游戏中改变

▌7.1 修行行动调整
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【修行】
• 已皈依者：+3功德
• 未皈依者：+2善缘

【护法】
• 已皈依者：消耗3财富，劫难-2，+1功德
• 未皈依者：消耗3财富，劫难-2，+1善缘

【布施】
• 已皈依者：3财富=+1功德
• 未皈依者：3财富=+1善缘

【渡化众生】
• 已皈依者：参与者+2功德
• 未皈依者：参与者+2善缘

【建寺】
• 效果不变（消耗8财富+4影响）
• 建成后：已皈依者每轮+1功德
• 建成后：未皈依者无产出（但建寺行为本身+2善缘）

▌7.2 新增行动：皈依
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【皈依三宝】
┌──────────────────────────────────────────────────────────────────────┐
│  条件：未皈依状态，善缘≥3                                           │
│  成本：1行动点，消耗3善缘                                            │
│  效果：                                                              │
│  • 状态变为"已皈依"                                                 │
│  • 所有善缘转化为功德（1:1）                                        │
│  • 之后可享受皈依加成                                                │
│  • 之后需遵守五戒                                                    │
│                                                                      │
│  佛学意义：                                                          │
│  皈依三宝（佛、法、僧）是成为佛教徒的正式仪式                        │
│  需要一定的善缘基础才能皈依                                          │
└──────────────────────────────────────────────────────────────────────┘

【僧侣】（默认已皈依）
• 讲经：消耗1功德 → 他人+2功德/善缘，劫难-1，自己+1功德
• 可以帮助未皈依者积累善缘

【贵族】
• 皈依前：赈灾消耗5财富 → 劫难-3，+1善缘
• 皈依后：赈灾消耗5财富 → 劫难-3，+2功德

【商人】
• 贸易：每轮+2财富（不受皈依状态影响）

【农夫】
• 皈依前：虔诚，每2财富布施=+1善缘
• 皈依后：虔诚，每2财富布施=+2功德（效率翻倍）

▌8.1 皈依路线预期收益
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第1轮皈依的贵族】
• 皈依成本：1行动点 + 3善缘
• 修行加成（5轮×1差值）：约+5功德
• 寺庙产出（假设第3轮建成，3轮有效）：+3功德
• 总额外收益：约+8功德
• 约束：5轮五戒

【终局预测】
• 功德约：23 + 8 = 31
• 得分：31×2 + 13（发愿）+ 3（成就）= 78分

▌8.2 未皈依路线预期收益
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【始终不皈依的贵族】
• 无皈依成本
• 修行效率低：约18善缘
• 财富积累更多：约20财富
• 影响保持：约8影响

【终局预测】
• 得分：18×1 + 20×0.5 + 8×1 + 15（发愿）+ 3（成就）= 54分

▌8.3 两条路线对比
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│  路线          │  预期得分  │  特点                                  │
│  ─────────────────────────────────────────────────────────────────  │
│  早期皈依      │  78分      │  高功德，有果位，受五戒约束            │
│  晚期皈依      │  60分      │  中等功德，灵活                        │
│  始终不皈依    │  54分      │  低功德但多财富，无约束                │
└──────────────────────────────────────────────────────────────────────┘

【分析】
• 皈依路线分数更高——这符合佛教价值观
• 但未皈依路线仍然可行——体现了"护法"的意义
• 差距约20-25分——有区分但不悬殊

▌8.4 平衡建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【如差距过大】
• 增加未皈依者发愿分数
• 或增加世俗成就的计分权重

【如差距过小】
• 增加皈依加成
• 或增加五戒违反的惩罚

【目标】
• 皈依路线应有优势（符合佛教主题）
• 但不皈依路线应可行（体现真正的抉择）
• 差距控制在15-25分为宜

▌9.1 佛学意义总结
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【皈依的意义】
• 正式成为佛教徒
• 获得修行的"资格"
• 承诺遵守戒律
• 有机会证得果位

【不皈依的意义】
• 做"外护"，护持三宝
• 积累善缘，种下未来得度的种子
• 世间善行同样有价值
• 不强迫信仰，尊重自由选择

【核心抉择】
皈依与否，体现了佛教的核心问题：
你愿意放下世俗执着，追求解脱吗？
还是留在世间，做一个善良的人？

两条路都有意义。

▌9.2 游戏意义总结
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【策略多样性】
• 不再是所有人都走相同的"修行"路线
• 有人可以专注护法而不皈依
• 有人可以晚期皈依，灵活应对

【道德张力】
• 皈依后有五戒约束，不能为所欲为
• 不皈依更自由，但失去修行效率
• 每个选择都有代价

【重玩价值】
• 同一身份可以走不同路线
• 商人可以皈依成"布施长者"
• 商人也可以不皈依做"仁德商人"

                          结论
═══════════════════════════════════════════════════════════════════════════════

【设计总结】

皈依抉择系统解决了原设计的"鸡肋"问题：

✓ 皈依成为真正的选择（而非默认）
✓ 两条路线都可行（修行 vs 护法）
✓ 每个选择有代价（五戒约束 vs 效率降低）
✓ 佛学意义深刻（内修 vs 外护）
✓ 游戏意义丰富（策略多样化）

【与原设计的区别】

原设计：所有人追求功德 → 皈依无意义
新设计：善缘/功德分离 → 皈依成为关键抉择

【推荐实施】

将此系统纳入v0.9基础版
作为核心机制而非扩展模块
