═══════════════════════════════════════════════════════════════════════════════
                   游戏概述
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】45-60分钟
【游戏轮数】6轮

【半合作机制】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：福慧得分+发愿加成最高者获胜

【核心理念】
• 福慧双修，缺一不可
• 渡化需要智慧门槛（慧≥5）
• 金钱只是资源，不计入得分（但可投资增值）
• 资源多者收益递减，防止滚雪球

═══════════════════════════════════════════════════════════════════════════════
                   投资系统（整数收益）
═══════════════════════════════════════════════════════════════════════════════

【投资行动】
• 消耗：5财富
• 效果：每2回合获得+2财富（固定整数）
• 游戏结束时：返还投资本金

【设计理念】
• 鼓励前期积累，后期行善的策略
• 提供金钱增值途径，但金钱不计入得分
• 投资收益为固定整数，简化计算


═══════════════════════════════════════════════════════════════════════════════
                   资源递减机制
═══════════════════════════════════════════════════════════════════════════════

【防止滚雪球】
当某项资源≥15时，该资源相关行动收益-1
当某项资源≥20时，该资源相关行动收益-2

示例：
• 财富20时，劳作收益从3降为1
• 福25时，渡化福收益-2
• 慧18时，修行慧收益-1

│  递减：慧≥15时收益-1                                                 │

│  消耗：3财富                                                         │
│  效果：+2福+1慧（商人+2福+2慧，但福递减）                            │
│  劫难≥10时：额外+1福                                                 │
│                                                                      │
│  【渡化】★需要慧≥5                                                  │
│  消耗：众生卡成本+角色修正                                           │
│  效果：+众生卡福慧奖励+递减奖励                                      │
│  递减：每渡化1次，后续奖励-1                                         │
│                                                                      │
│  【护法】                                                            │
│  消耗：2财富                                                         │
│  效果：+2福+1慧，劫难-2                                              │
│  劫难≥10时：额外+1福                                                 │
│                                                                      │
│  【投资】                                                            │
│  消耗：5财富                                                         │
│  效果：每2回合+2财富，结束返还本金                                   │
└──────────────────────────────────────────────────────────────────────┘

【学者专属】
• 讲学：自己+2福+1慧，目标+1福+1慧

【僧侣专属】
• 托钵（骰子）：成功+3财+1福+1慧，失败+1财+1慧
• 法事：+4财-2福-1慧（应急用）


═══════════════════════════════════════════════════════════════════════════════
                   回合流程
═══════════════════════════════════════════════════════════════════════════════

【每回合流程】
1. 事件阶段：抽取共享事件卡
2. 众生阶段：更新众生区域，检查超时
3. 行动阶段：每玩家执行2次行动
4. 结算阶段：生存消耗(-1财/每2回合)，投资收益(+2财/每2回合)
