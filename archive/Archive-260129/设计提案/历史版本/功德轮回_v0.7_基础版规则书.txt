▌1.1 游戏主题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

你们是一群修行者，身份各异：僧侣、贵族、商人、农夫。
你们的共同使命是：在末法时代拯救苍生，抵御劫难。

但这是一场【合作下的竞争】：
• 团队必须合作——如果劫难降临，所有人都将失败
• 个人争证果位——团队成功后，谁的修行最高，谁证得佛果

这完美体现了佛教精神：
• 菩萨道：普度众生（合作）
• 阿罗汉道：自证涅槃（个人修行）


▌1.2 胜利条件（双层系统）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一层：团队胜利（必须全部达成）】

┌──────────────────────────────────────────────────────────────────────┐
│  ✓ 劫难轨道 ≤ 12                                                    │
│  ✓ 渡化众生 ≥ 5 个                                                  │
│  ✓ 建成寺庙 ≥ 2 座                                                  │
│  ✓ 所有玩家功德 ≥ 10                                                │
│                                                                      │
│  → 若未全部达成：【全员失败】，无论个人分数多高                     │
└──────────────────────────────────────────────────────────────────────┘

【第二层：个人果位（团队成功后）】

┌──────────────────────────────────────────────────────────────────────┐
│  个人得分 = 功德×2 + 发愿分 + 成就分                                │
│                                                                      │
│  果位评定：                                                          │
│  • 佛果（45+分）：★最高分者★ 唯一佛果，游戏胜者                    │
│  • 菩萨果（35-44分）：证得菩萨，但非最高                            │
│  • 阿罗汉果（25-34分）：自证涅槃                                    │
│  • 初果（15-24分）：初入圣道                                        │
│  • 凡夫（<15分）：虽得救，但修行不足                                │
│                                                                      │
│  关键：只有【最高分者】获得"佛果"称号                               │
│  其他人虽然"得救"，但果位较低                                       │
└──────────────────────────────────────────────────────────────────────┘

▌2.1 准备步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 【选择身份】
   每位玩家选择或随机抽取1张身份卡，放在面前。

2. 【领取起始资源】
   根据身份卡领取起始资源（见下表）。

3. 【选择发愿】
   洗混发愿卡，每人抽2张，【秘密】选择1张保留，弃掉另1张。
   发愿是你的个人目标，游戏结束时公开结算。

4. 【准备牌堆】
   洗混众生卡，正面朝下放在众生区。
   洗混事件卡，正面朝下放在事件区。

5. 【设置劫难轨道】
   将劫难标记放在"0"位置。

游戏进行 6 轮，每轮包含 4 个阶段。

▌阶段1：协商阶段（2分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【做什么】
• 讨论本轮策略
• 协商谁来做什么
• 可以进行【布施】（无偿给予资源）

【布施规则】
┌──────────────────────────────────────────────────────────────────────┐
│  布施（无偿给予）：                                                  │
│  • 你可以无偿将财富或影响给予任何玩家                                │
│  • 布施者获得功德：每3财富=+1功德                                   │
│  • 接受者无需回报                                                    │
│                                                                      │
│  布施体现佛教"三轮清净"：                                           │
│  • 施者清净：不求回报                                                │
│  • 受者清净：感恩接受                                                │
│  • 施物清净：真实给予                                                │
└──────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

│  1. 修行：+2功德                                                     │
│                                                                      │
│  2. 劳作：+3财富                                                     │
│                                                                      │
│  3. 护法：消耗3财富，-2劫难                                          │
│                                                                      │
│  4. 渡化：尝试渡化1个众生（见众生卡要求）                            │
│                                                                      │
│  5. 建寺：消耗8财富+4影响，建造1座寺庙                               │
│            团队共享，完成后每轮所有人+1功德                          │

┌──────────────────────────────────────────────────────────────────────┐
│  【专属行动】根据身份                                                │

【步骤2：翻开事件卡】
• 翻开1张事件卡，立即执行效果

【众生卡示例】

▌阶段4：劫难阶段（1分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1：劫难增长】
• 劫难+3（每轮固定增长）

【步骤2：检查众生超时】
• 停留2轮未渡化的众生，劫难+3，移除该众生卡

【步骤3：检查失败条件】
• 若劫难≥20：【立即失败】，全员输

【步骤4：结束本轮】
• 开始下一轮

▌4.2 发愿的策略意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【半合作的张力】

• 利他型发愿：帮助团队多，分数高，但需要牺牲个人资源
• 自利型发愿：帮助团队少，分数低，但容易达成

【道德困境】

情景：第5轮，众生卡"战乱孤儿"需要6财富
• 商人有8财富，但自己的发愿需要保留10财富
• 如果不救孤儿，劫难+3，可能导致团队失败
• 如果救孤儿，自己的发愿可能失败

→ 这就是"我应该牺牲多少？"的佛教道德困境

▌5.1 游戏结束触发
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【立即结束】
• 劫难≥20：全员失败
• 完成6轮：进入结算

【团队胜利检查】
• 劫难≤12：✓
• 渡化众生≥5：✓
• 建寺≥2：✓
• 所有玩家功德≥10：✓

→ 全部达成：团队胜利，进入个人计分
→ 任一未达成：全员失败


▌5.2 个人计分
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【计分公式】

个人得分 = 功德×2 + 发愿分 + 成就分

【成就分】
• 渡化众生最多：+3分
• 建寺贡献最多：+3分
• 布施最多：+3分
• 护法最多：+2分

【示例计分】

僧侣：功德20，发愿"地藏宏愿"达成（+18），渡化最多（+3）
→ 20×2 + 18 + 3 = 61分 → 佛果★

贵族：功德15，发愿"护法檀越"达成（+15），建寺最多（+3）
→ 15×2 + 15 + 3 = 48分 → 佛果（并列）

商人：功德12，发愿"功德商人"未达成（+0）
→ 12×2 + 0 = 24分 → 阿罗汉果

农夫：功德18，发愿"布施长者"达成（+16），布施最多（+3）
→ 18×2 + 16 + 3 = 55分 → 佛果

▌6.1 团队合作要点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 【优先级】
   • 劫难控制 > 渡化众生 > 建寺 > 个人修行
   • 如果劫难失控，所有人都输

2. 【分工建议】
   • 贵族负责赈灾（-3劫难效率最高）
   • 僧侣负责渡化迷途众生
   • 商人提供财富支持
   • 农夫虔诚布施积累功德

3. 【资源共享】
   • 积极布施，帮助资源不足的玩家
   • 布施也能获得功德，并非纯牺牲


▌6.2 个人竞争要点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 【选择合适的发愿】
   • 根据身份选择发愿
   • 僧侣适合"地藏宏愿"
   • 贵族适合"护法檀越"
   • 商人适合"功德商人"
   • 农夫适合"布施长者"

2. 【平衡合作与竞争】
   • 不能只顾自己——团队失败你也输
   • 不能完全无私——最终要比较个人分数

3. 【功德积累】
   • 功德×2是最大计分来源
   • 修行、布施、渡化都能积累功德

▌每轮流程速查
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 协商阶段（2分钟）→ 讨论、布施
2. 行动阶段（5分钟）→ 每人2个行动
3. 事件阶段（3分钟）→ 翻众生卡、事件卡
4. 劫难阶段（1分钟）→ 劫难+3，检查失败


▌胜利条件速查

团队胜利（必须全达成）：
□ 劫难≤12
□ 渡化≥5个
□ 建寺≥2座
□ 所有人功德≥10

个人果位（团队成功后）：
• 最高分 = 佛果（唯一胜者）

▌基础版简化了哪些内容？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【删除的复杂机制】
• 三毒系统（贪嗔痴）→ 扩展版
• 修行阶段系统（4阶段升级）→ 扩展版
• 承诺交易与借贷 → 扩展版
• 冲突型发愿 → 已永久删除
• 朝廷弹劾机制 → 扩展版

• 团队条件：4个（原5个）

【保留的核心体验】
✓ 半合作模式（合作渡众生，竞争证果位）
✓ 布施系统（无偿给予获得功德）
✓ 发愿系统（秘密个人目标）
