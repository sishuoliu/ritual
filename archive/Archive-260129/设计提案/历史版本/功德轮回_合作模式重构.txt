▌1.1 佛教的核心教义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【大乘佛教核心】
┌──────────────────────────────────────────────────────────────────────┐
│  "众生无边誓愿度"  ─ 四弘誓愿之首                                   │
│  "自利利他"        ─ 菩萨道精神                                     │
│  "众生平等"        ─ 一切众生皆有佛性                               │
│  "慈悲喜舍"        ─ 四无量心                                       │
└──────────────────────────────────────────────────────────────────────┘

【现有设计的矛盾】
• v0.2/v0.3都是竞争游戏：每人争取最高分
• 这与"普度众生"的精神相悖
• 盗匪劫掠、皇帝灭佛都是"伤害他人"的行动
• 胜利条件是"我比你强"，而非"我们一起解脱"

▌2.1 核心设计理念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【游戏主题】
"末法时代，三灾将至，唯有众生同心，方能度过劫难"

【胜利条件】
┌──────────────────────────────────────────────────────────────────────┐
│  【共同胜利】                                                            │
│  所有玩家必须共同达成以下条件：                                          │
│                                                                          │
│  1. 劫难指数≤10（共同威胁被化解）                                       │
│  2. 所有玩家功德≥15（众生皆得救度）                                     │
│  3. 至少建寺3座（法脉传承）                                             │
│                                                                          │
│  → 满足以上3条件：全员胜利                                              │
│  → 任意1条件未达成：全员失败                                            │
│                                                                          │
│  【个人成就】（胜利后才计分）                                            │
│  游戏胜利后，根据个人贡献评定"果位"：                                    │
│  • 佛果（45+分）：圆满觉悟                                               │
│  • 菩萨果（35-44分）：普度众生                                           │
│  • 阿罗汉果（25-34分）：自证涅槃                                         │
│  • 初果（15-24分）：初入圣道                                             │
│  • 凡夫（<15分）：仍在轮回                                               │
│                                                                          │
│  ★ 关键：即使游戏胜利，个人也可能只得初果                               │
│           这不是失败，而是"修行路径不同"                                 │
└──────────────────────────────────────────────────────────────────────┘


▌2.2 共同威胁：三灾系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【佛教背景】
三灾：刀兵劫、瘟疫劫、饥馑劫

【游戏机制】

┌──────────────────────────────────────────────────────────────────────┐
│  【劫难轨道】                                                            │
│                                                                          │
│   [死亡危机] ← 50 ← 40 ← 30 ← 20 ← 10 ← 0 [安宁]                       │
│                                                                          │
│  • 起始：劫难=0                                                         │
│  • 每轮自动+5（世道衰败）                                               │
│  • 负面事件额外+3~+8                                                    │
│  • 玩家可通过"护法"行动-2~-5                                            │
│                                                                          │
│  【劫难阈值效果】                                                        │
│  • 劫难≥20：启动"刀兵劫"                                                │
│    效果：每轮随机1名玩家-3财富-2功德（战乱伤害）                         │
│                                                                          │
│  • 劫难≥35：启动"瘟疫劫"                                                │
│    效果：每轮所有玩家-1功德，医者必须义诊                                │
│                                                                          │
│  • 劫难≥50：世界毁灭                                                    │
│    效果：游戏立即结束，全员失败                                          │
└──────────────────────────────────────────────────────────────────────┘

【对抗劫难的行动】

┌──────────────────────────────────────────────────────────────────────┐
│  所有玩家都可以执行的"护法"行动：                                       │
│                                                                          │
│  • 僧侣讲经：消耗2功德，劫难-4                                          │
│  • 皇帝护法：消耗5影响，劫难-6                                          │
│  • 贵族赈灾：消耗8财富，劫难-5                                          │
│  • 商人捐资：消耗10财富，劫难-4                                         │
│  • 农夫祈福：消耗1行动，劫难-2                                          │
│  • 医者救灾：消耗3药物，劫难-5                                          │
│  • 士人著书：消耗4影响，劫难-3（安抚民心）                              │
│                                                                          │
│  ★ 关键设计：                                                           │
│  每个身份都能贡献，但方式不同                                            │
│  玩家必须协调："谁负责压制劫难？谁负责积累资源？"                        │
└──────────────────────────────────────────────────────────────────────┘


▌2.3 NPC系统：众生相
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计思路】
引入NPC"众生"，玩家的目标是"渡化众生"

┌──────────────────────────────────────────────────────────────────────┐
│  【众生卡牌】（20张）                                                   │
│                                                                          │
│  每轮翻开2张众生卡，代表需要救度的苦难众生：                             │
│                                                                          │
│  ◆ 战乱孤儿                                                             │
│     苦难值：8                                                           │
│     渡化条件：累计给予8财富 或 农夫+贵族各给4财富                        │
│     奖励：劫难-3，参与者各+2功德                                         │
│     佛典：收养孤儿，功德无量                                             │
│                                                                          │
│  ◆ 病困村民                                                             │
│     苦难值：6                                                           │
│     渡化条件：医者治疗2次 或 僧侣+医者各治疗1次                          │
│     奖励：劫难-2，参与者各+3功德                                         │
│     佛典：救人一命，胜造七级浮屠                                         │
│                                                                          │
│  ◆ 饥饿灾民                                                             │
│     苦难值：10                                                          │
│     渡化条件：累计布施10财富                                             │
│     奖励：劫难-4，参与者各+2功德                                         │
│     特殊：若本轮无人救助，劫难+5                                         │
│                                                                          │
│  ◆ 迷途盗匪                                                             │
│     苦难值：12                                                          │
│     渡化条件：僧侣讲经+贵族提供工作(各5财富)                             │
│     奖励：劫难-6，获得1名"改邪归正"的帮手（每轮助力）                    │
│     佛典：放下屠刀，立地成佛                                             │
│                                                                          │
│  ◆ 无明凡夫                                                             │
│     苦难值：5                                                           │
│     渡化条件：士人著书+僧侣讲经                                          │
│     奖励：劫难-2，所有人+1功德                                           │
│     佛典：破除无明，见性成佛                                             │
│                                                                          │
│  【众生渡化计数】                                                        │
│  • 每渡化1个众生，众生轨道+1                                            │
│  • 游戏目标之一：渡化≥6个众生                                           │
│  • 若有众生卡无人救助3轮，劫难+10（失去救度机会）                        │
└──────────────────────────────────────────────────────────────────────┘

【意义】
• 创造合作动力：必须配合才能渡化众生
• 佛教意义强：每张众生卡都有佛典出处
• 道德压力：看到众生受苦而不救，会有惩罚

▌每轮三阶段
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1：协商阶段】（开放讨论，3分钟）
• 玩家可公开讨论策略
• "这轮谁负责压制劫难？"
• "这个众生谁能救？"
• "我差2财富，谁能借我？"

【阶段2：行动阶段】（同时选择行动）
• 每人选2个行动（写在纸上或用行动卡）
• 同时翻开执行
• 可能有人"背叛"协商（选择自利行动）

【阶段3：劫难阶段】（检查共同威胁）
• 翻开本轮众生卡
• 劫难自动+5
• 检查是否触发灾难
• 结算众生渡化

▌4.1 三毒系统的新意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【在合作游戏中，三毒变成"诱惑"】

┌──────────────────────────────────────────────────────────────────────┐
│  贪（Greed）：诱惑你积累个人财富，而非布施                              │
│    • 每次获得≥5财富不用于团队：+1贪                                     │
│    • 贪≥8：无法执行护法行动                                             │
│    → 你被排除在救世之外                                                │
│                                                                          │
│  嗔（Anger）：诱惑你伤害他人                                            │
│    • 盗匪劫掠、皇帝灭佛：+2嗔                                           │
│    • 嗔≥8：其他玩家拒绝与你合作                                         │
│    → 你失去团队信任                                                    │
│                                                                          │
│  痴（Delusion）：诱惑你不关心团队                                       │
│    • 连续2轮不参与护法/渡化众生：+1痴                                   │
│    • 痴≥8：所有合作行动效果减半                                         │
│    → 你的帮助变得无力                                                  │
│                                                                          │
│  【业力崩坏】                                                            │
│  • 若任意玩家三毒总和≥20：游戏立即失败                                  │

▌4.2 "自利"玩家的平衡

                          结论与建议
═══════════════════════════════════════════════════════════════════════════════

【推荐：模式3（合作主导+个人修行）】

理由：
1. ✓ 契合佛教"自利利他"精神
2. ✓ 保留8种非对称身份的独特性
3. ✓ 创造道德困境："我该利己还是利他？"
4. ✓ 引入NPC"众生"系统，目标是"拯救众生"
5. ✓ 团队必须合作才能胜利
6. ✓ 但个人可以有不同的修行路径（阿罗汉vs菩萨）

【核心机制】
• 共同威胁：三灾劫难轨道
• NPC系统：20张众生卡，需要渡化
• 胜利条件：劫难≤10 + 渡化≥6众生 + 建寺≥3 + 全员功德≥15
• 个人果位：游戏胜利后才评定个人成就
• 三毒系统：诱惑玩家自利，但会伤害团队
