▌1.1 在家居士（Lay Practitioner）的人生
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1：初遇佛法】
• 因缘际会接触佛教（听闻讲经、遇见僧侣、家族传统）
• 产生信心，决定皈依

【阶段2：皈依三宝】
• 正式仪式：皈依佛、法、僧
• 接受五戒（不杀生、不偷盗、不邪淫、不妄语、不饮酒）
• 获得法名

【阶段3：日常修行】
• 定期布施（供养僧团、资助寺院）
• 参加法会、听经闻法
• 持守戒律
• 朝圣（四大圣地、名山古刹）
• 抄经、念佛、禅修

【阶段4：深入修行】
• 受持八关斋戒（更严格的戒律）
• 可能成为护法居士（大檀越）
• 可能参与寺院管理
• 追求证果（须陀洹、斯陀含等）

【阶段5：临终与往生】
• 临终念佛、助念
• 功德回向给亡者
• 超度法会


▌1.2 出家僧侣（Monastic）的人生
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1：出家发心】
• 因缘成熟，决意出家
• 剃度仪式
• 受沙弥戒（10戒）

【阶段2：正式受戒】
• 受具足戒（比丘250戒，比丘尼348戒）
• 成为正式僧侣
• 开始僧团生活

【阶段3：修行精进】
• 托钵乞食
• 禅修、诵经、参禅
• 受法传承（师徒关系）
• 可能云游参学

【阶段4：弘法利生】
• 讲经说法
• 建寺安僧
• 收徒传法
• 著书立说

【阶段5：证果圆寂】
• 追求阿罗汉果或菩萨果
• 圆寂后舍利供养
• 塔葬或入祖师堂


▌1.3 历史上著名的佛教居士案例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【给孤独长者（Anathapindika）】— 商人
┌──────────────────────────────────────────────────────────────────────┐
│  身份：舍卫城首富商人                                                │
│  皈依：佛陀时代最重要的在家护法                                      │
│  功德：购买土地建造祇园精舍，每日供养大量僧侣                        │
│  特点：以"布施第一"著称，是在家居士的典范                           │
│  结局：证得须陀洹果（入流）                                          │
│                                                                      │
│  → 游戏对应：商人身份 + 布施长者发愿                                │
└──────────────────────────────────────────────────────────────────────┘

【毗舍佉（Visakha）】— 贵族女性
┌──────────────────────────────────────────────────────────────────────┐
│  身份：摩揭陀国贵族女子                                              │
│  皈依：7岁时听佛说法即证须陀洹                                       │
│  功德：建造鹿子母讲堂，终生护持僧团                                  │
│  特点：在家女居士中布施第一                                          │
│  结局：被称为"最胜女施主"                                           │
│                                                                      │
│  → 游戏对应：贵族身份 + 护法檀越发愿                                │
└──────────────────────────────────────────────────────────────────────┘

【裴休】— 唐代士大夫
┌──────────────────────────────────────────────────────────────────────┐
│  身份：唐代宰相                                                      │
│  皈依：师从黄檗希运禅师                                              │
│  功德：护持佛法，送子出家（即后来的法眼宗创始人）                    │
│  特点：朝廷大员与禅宗的深厚关系                                      │
│  结局：成为重要的居士护法典范                                        │
│                                                                      │
│  → 游戏对应：贵族/皇帝身份 + 明君护法发愿                           │
└──────────────────────────────────────────────────────────────────────┘

│  在家居士可证                                                        │
├──────────────────────────────────────────────────────────────────────┤

│  断除：减弱贪欲和嗔恚                                                │

│  在家居士可证                                                        │
├──────────────────────────────────────────────────────────────────────┤
│  【阿那含】（不还/三果）                                             │
│  断除：完全断除贪欲和嗔恚                                            │
│  特点：不再返回欲界，在天界证涅槃                                    │
│  在家居士难以证得                                                    │
├──────────────────────────────────────────────────────────────────────┤
│  【阿罗汉】（无生/四果）                                             │
│  断除：完全断除所有烦恼                                              │
│  特点：不再轮回，证得涅槃                                            │
│  传统上认为需出家修行                                                │
└──────────────────────────────────────────────────────────────────────┘

根据以上分析，以下机制值得考虑加入游戏。
按重要性和可行性排序：

▌3.1 【高优先级】皈依系统 ★强烈推荐★
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【真实佛教意义】
皈依三宝是成为佛教徒的第一步，是一切修行的基础。

【游戏设计建议】

┌──────────────────────────────────────────────────────────────────────┐
│  【皈依机制】                                                        │
│                                                                      │
│  时机：游戏开始时或第1-2轮                                           │
│  效果：                                                              │
│  • 皈依前：所有功德获取效率-50%                                      │
│  • 皈依后：正常获取功德                                              │
│  • 皈依需要：找僧侣执行"传授三皈"行动                               │
│                                                                      │
│  皈依仪式：                                                          │
│  • 僧侣消耗1功德，为他人传授皈依                                     │
│  • 被皈依者获得"三宝弟子"身份标记                                   │
│  • 僧侣获得+1功德（法布施）                                          │
│                                                                      │
│  游戏意义：                                                          │
│  • 僧侣角色更重要（传法者）                                          │
│  • 创造玩家间的师徒联系                                              │
│  • 体现佛教入门的真实流程                                            │
└──────────────────────────────────────────────────────────────────────┘


▌3.2 【高优先级】功德回向系统 ★强烈推荐★
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【真实佛教意义】
回向是将自己积累的功德分享给他人（尤其是亡者），是大乘佛教的核心实践。

【游戏设计建议】

┌──────────────────────────────────────────────────────────────────────┐
│  【功德回向】                                                        │
│                                                                      │
│  基础规则：                                                          │
│  • 每轮结束时，可以将自己的功德回向给他人                            │
│  • 回向X功德 → 自己-X功德，他人+X功德                               │
│  • 回向者额外获得+1功德（回向的功德）                               │
│                                                                      │
│  特殊情况：                                                          │
│  • 若某玩家功德<10（团队失败条件），可回向帮助                       │
│  • 创造"牺牲自己帮助团队"的机会                                     │
│                                                                      │
│  与布施的区别：                                                      │
│  • 布施：给财富/影响                                                 │
│  • 回向：给功德（精神资源）                                          │
│                                                                      │
│  佛教意义：                                                          │
│  • 功德回向是菩萨道的核心                                            │
│  • "愿以此功德，普及于一切"                                         │
└──────────────────────────────────────────────────────────────────────┘

│  • 鹿野苑（初转法轮）                                                │

│  • 增加版图互动                                                      │
│  • 提供功德积累的新途径                                              │
│  • 体现朝圣的重要性                                                  │
└──────────────────────────────────────────────────────────────────────┘


▌3.4 【中优先级】师徒传承系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【游戏设计建议】

┌──────────────────────────────────────────────────────────────────────┐

│  师徒效果：                                                          │

│  策略意义：                                                          │
│  • 创造玩家间的长期联盟                                              │

▌设计哲学
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

真实的佛教修行是一生的旅程，包含：
• 入门（皈依、受戒）
• 日常修行（布施、持戒、禅修）
• 深入修行（朝圣、法会、师徒传承）
• 证果（四果阶位）
• 利他（讲经、渡众）
• 圆满（临终、往生）

游戏虽然只有6轮（代表人生的关键阶段），
但可以通过机制设计体现这完整旅程的精髓。
