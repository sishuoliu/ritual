═══════════════════════════════════════════════════════════════════════════════
                    一、游戏概述
═══════════════════════════════════════════════════════════════════════════════

【游戏目标】
玩家扮演不同身份的修行者，在有限回合内共同积累「福」与「慧」，渡化众生、化解劫难。
团队胜利条件：游戏结束时「劫难」≤12，且「渡化」的众生数量 ≥6。
满足团队胜利时，按个人福慧与发愿结算个人得分，分高者排名靠前；团队失败则所有人得 0 分。

【游戏人数】2–4 人（标准 4 人：农夫、商人、学者、僧侣）

【游戏时长】约 45–60 分钟（6 回合，每回合含事件、众生、行动、结算）

【核心设计】
• 渡化差异化：不同职业渡化众生时付出的资源不同
• 发愿系统：每回合获得奖励，结束时按达成与否加分或扣分
• 资源递减：福/慧/财富越高，同类型行动的收益会略减，防止滚雪球
• 皈依选择：开局可选择「皈依」或「不皈依」，两条路线均可获胜
• 投资系统：支付 5 财富可投资，每 2 回合获得 2 财富，游戏结束返还本金
• 持续帮助：连续 3 回合执行「帮助类」行动，额外获得 1 福
• 团队合作：需兼顾渡化进度与劫难控制，共同达成胜利条件


═══════════════════════════════════════════════════════════════════════════════
                    二、游戏准备
═══════════════════════════════════════════════════════════════════════════════

【2.1 配件】
• 角色 token / 角色板：农夫、商人、学者、僧侣（各 1）
• 资源标记：财富、福、慧（可用筹码或笔记录）
• 事件牌：8 张（见「事件卡」表），洗匀背面朝上
• 众生牌：8 张（见「众生卡」表），洗匀背面朝上
• 骰子：2 颗六面骰（用于「修行」「化缘」等判定）
• 劫难指示物：1 个，置于劫难轨（0–20）
• 渡化计数：记录当前已渡化众生数量（目标 ≥6）

【2.2 初始设置】
1. 每位玩家选择或随机获得一个角色（农夫 / 商人 / 学者 / 僧侣）。
2. 每位玩家（除僧侣外）秘密选择「皈依」或「不皈依」；僧侣固定为皈依。
3. 按角色发放初始资源：

   ┌──────────┬────────┬────┬────┐
   │ 角色     │ 财富   │ 福 │ 慧 │
   ├──────────┼────────┼────┼────┤
   │ 农夫     │ 5      │ 2  │ 2  │
   │ 商人     │ 8      │ 1  │ 1  │
   │ 学者     │ 3      │ 1  │ 4  │
   │ 僧侣     │ 0      │ 3  │ 3  │
   └──────────┴────────┴────┴────┘

4. 皈依者：福 +1，慧 +1。不皈依者：财富 +3。
5. 从众生牌堆顶翻出 1 张，置于「众生区域」；事件牌堆与众生牌堆放在桌面中央。
6. 劫难轨置于 0；渡化计数置于 0。
7. 每位玩家随机抽取 1 张发愿卡（该角色 2 张中随机 1 张），面朝上放置，游戏开始后每回合执行发愿的「每回合奖励」。

【2.3 游戏参数】
• 总回合数：6 回合
• 渡化目标：6 人（即「渡化计数」≥6 才能满足团队胜利条件之一）
• 劫难上限：20（若劫难 ≥20，游戏立即结束，团队失败）
• 团队胜利的劫难条件：结束时劫难 ≤12


═══════════════════════════════════════════════════════════════════════════════
                    三、资源与术语
═══════════════════════════════════════════════════════════════════════════════

【财富】用于布施、护法、渡化（部分众生需支付财富）、投资等。可为 0，不能为负（不足时扣到 0 并可能触发「挨饿」）。

【福】修行与利他获得的功德，与慧一起决定个人得分，并用于渡化（僧侣需消耗福）、讲学收益等。

【慧】修行与渡化获得的智慧，≥5 才能执行「渡化」行动；与福一起参与最终计分。

【劫难】全局数值，由事件卡、未渡化众生滞留、布施/护法减少等改变。劫难 ≥20 游戏提前结束；结束时劫难 ≤12 且渡化 ≥6 团队才胜利。

【渡化】将「众生区域」中的一张众生牌完成救度，消耗财富/福/慧（依角色与众生类型），团队渡化计数 +1，该众生移出游戏；执行渡化的玩家获得该众生牌上标注的福、慧奖励（并受资源递减影响）。

【发愿】每人 1 张发愿卡，每回合开始可获得卡上「每回合」奖励；游戏结束时若达成卡上「达成条件」则加对应分数，未达成则扣分。


═══════════════════════════════════════════════════════════════════════════════
                    四、回合流程
═══════════════════════════════════════════════════════════════════════════════

每一回合按以下顺序进行：

（1）事件阶段
   • 翻开事件牌堆顶 1 张，按牌面效果修改：劫难、全体财富、全体福、全体慧。
   • 皈依者从「福」「慧」类事件中获得的数值额外 +50%（向下取整）。

（2）众生阶段
   • 众生区域内，每张众生牌的「滞留回合数」+1。
   • 滞留已达 2 回合的众生：若本回合仍未被渡化，则劫难 +3，并将该众生牌移出游戏（不再补牌）。
   • 若众生牌堆仍有牌，将牌堆顶 1 张加入众生区域（正面朝上）。

（3）行动阶段
   • 从起始玩家开始，顺时针进行。每位玩家在本回合内执行共「2 次」行动。
   • 每次行动：从该角色允许的行动中选择 1 项并执行（见「五、行动说明」）。
   • 若某行动有骰子判定（如修行、化缘），执行时掷 2d6 并按结果修正收益。

（4）持续帮助结算
   • 若某玩家本回合执行过「帮助类」行动（布施、渡化、护法、讲学），其「连续帮助回合数」+1；否则清零。
   • 若「连续帮助回合数」≥3，该玩家立即获得 1 福；之后每回合只要继续帮助，再获得 1 福；若某回合未帮助，连续帮助回合数清零。

（5）生存与投资结算
   • 生存消耗：若当前回合为第 2、4、6 回合，每位玩家财富 -1；若财富不足则扣至 0，且福 -1、慧 -1（各不低于 0），并记录「挨饿」1 次。
   • 投资收益：若当前回合为第 2、4、6 回合，凡曾执行过「投资」且尚未返还本金的玩家，获得 2 财富（不扣本金，游戏结束时才返还本金）。

（6）劫难检查
   • 若劫难 ≥20，游戏立即结束，团队失败。
   • 否则进入下一回合；若已完成第 6 回合，进入「终局计分」。


═══════════════════════════════════════════════════════════════════════════════
                    五、行动说明
═══════════════════════════════════════════════════════════════════════════════

以下为各角色可选行动及其规则。收益若受「资源递减」影响，则：该资源 ≥20 时收益 -2（最少 1），≥15 时收益 -1，否则为基础值。

▌5.1 劳作（LABOR）
• 适用：农夫、商人、学者（僧侣不可）
• 效果：获得财富。皈依者基础 +3，不皈依者基础 +4，再按「财富」当前值做资源递减。
• 说明：主要经济来源之一。

▌5.2 修行（PRACTICE）
• 适用：所有角色
• 效果：掷 2d6。根据骰子结果与角色基础慧收益计算本次慧的获得量（资源递减按「慧」）。
   • 学者基础 3，其余角色基础 2；骰子大成功 +1，大失败 -1（最少 1）。
   • 皈依者最终再 +1 慧。
• 说明：提升慧以达成渡化条件（慧≥5）并提高得分。

▌5.3 布施（DONATE）
• 适用：农夫、商人、学者
• 条件：财富 ≥3
• 效果：财富 -3；获得福与慧（商人福收益更高，且按「财富」递减）；劫难 -1；本回合计为「帮助」。
   • 基础：福 +2，慧 +1。商人：福按财富递减基础 3，慧 +2。劫难 ≥10 时，福再 +1。

▌5.4 渡化（SAVE）
• 适用：农夫、商人、学者、僧侣
• 条件：慧 ≥5；众生区域存在至少 1 张你可支付成本的众生（见「渡化成本」）。
• 成本与收益：见「六、渡化规则」与「众生卡」表。支付财富/福/慧后，获得该众生牌上的福、慧奖励（首 2 次渡化有额外福奖励，再受资源递减）。
• 效果：该众生移出游戏，团队渡化计数 +1；本回合计为「帮助」。

▌5.5 护法（PROTECT）
• 适用：农夫、商人、学者、僧侣
• 条件：财富 ≥2
• 效果：财富 -2；福 +2，慧 +1；若劫难 ≥10，福再 +1；劫难 -2；本回合计为「帮助」。

▌5.6 讲学（TEACH）
• 适用：仅学者
• 效果：自己获得福与慧（按福、慧分别资源递减，基础福 +2、慧 +1）；另选一名其他玩家，其福 +1、慧 +1。本回合计为「帮助」。

▌5.7 化缘（ALMS）— 僧侣专属
• 适用：仅僧侣
• 效果：掷 2d6。大成功或成功：财富 +3，福 +1；否则财富 +1。慧 +1。

▌5.8 法会（CEREMONY）— 僧侣专属
• 适用：仅僧侣
• 效果：财富 +4；福 -2，慧 -1（不低于 0）。
• 说明：用福慧换财富，慎用。

▌5.9 投资（INVEST）
• 适用：农夫、商人、学者
• 条件：财富 ≥5
• 效果：财富 -5，将 5 记为「投资本金」。之后每在第 2、4、6 回合的「生存与投资结算」时获得 2 财富；游戏结束时返还 5 财富本金。每局通常只能投资一次（若规则允许多次，则每次 5 财，收益独立计算）。


═══════════════════════════════════════════════════════════════════════════════
                    六、渡化规则
═══════════════════════════════════════════════════════════════════════════════

【6.1 渡化条件】
• 你的慧 ≥5。
• 众生区域中至少有一张众生，你当前财富、福、慧满足其「对你角色的渡化成本」。

【6.2 渡化成本（按角色）】
众生牌有「基础财富成本」。你实际支付为：

• 农夫：财富 = 众生基础成本，福 0，慧 0
• 商人：财富 = 众生基础成本 + 1，福 0，慧 0
• 学者：财富 = 众生基础成本 - 1（至少 0），福 0，慧 1
• 僧侣：财富 = 众生基础成本 - 1（至少 0），福 1，慧 0

【6.3 渡化收益】
• 获得该众生牌上写的「福奖励」「慧奖励」；再受「资源递减」（按你当前福/慧）。
• 若你本局渡化次数为 0 或 1，额外 +1 福（首两次渡化奖励）。

═══════════════════════════════════════════════════════════════════════════════
                    十、胜负与计分
═══════════════════════════════════════════════════════════════════════════════

【10.1 游戏结束】
• 正常：第 6 回合的「生存与投资结算」及劫难检查后结束。
• 提前：某回合结束时劫难 ≥20，立即结束，团队失败。

【10.2 团队胜负】
• 团队胜利：游戏结束时 劫难 ≤12 且 渡化计数 ≥6。
• 不满足任一条则团队失败。

【10.3 个人得分】（仅当团队胜利时计算；团队失败则所有人 0 分）
• 基础分 = 3 × √(福 × 慧)（福或慧为 0 时按 1 计算，再开根号）。
• 皈依加成：
  - 皈依者：福≥15 且 慧≥15 → +16；福≥12 且 慧≥12 → +12。
  - 不皈依者：福≥20 且 慧≥20 → +18；福≥16 且 慧≥16 → +10。
• 发愿：按发愿卡「奖励」或「失败」加上对应分数。
• 最终得分 = 基础分 + 皈依加成 + 发愿分。财富不计入得分。
• 按最终得分排名，高者名次靠前（可再约定同分规则）。


═══════════════════════════════════════════════════════════════════════════════
                    十一、骰子判定（2d6）
═══════════════════════════════════════════════════════════════════════════════

掷 2 颗六面骰，和为：
• 2–4：大失败（修行/化缘等时收益 -1，最少 1）
• 5–7：失败（无修正）
• 8–9：成功（无修正）
• 10–12：大成功（修行/化缘等时收益 +1）

具体行动中的用法以「五、行动说明」为准。


═══════════════════════════════════════════════════════════════════════════════
                    十二、规则摘要与可选变体
═══════════════════════════════════════════════════════════════════════════════

【摘要】
• 6 回合，每回合：事件 → 众生 → 每人 2 次行动 → 持续帮助 → 生存与投资 → 劫难检查。
• 胜利：劫难 ≤12 且 渡化 ≥6；失败则 0 分。
• 渡化需慧 ≥5，成本依角色与众生表；首两次渡化额外 +1 福。
• 资源递减：福/慧/财富高时同类型收益略减；持续 3 回合帮助 +1 福；投资 5 财每 2 回合 +2 财。

【可选变体】
• 渡化目标 5 人：将「渡化计数 ≥6」改为「≥5」，难度略降。
• 事件卡加重：若希望劫难更容易导致失败，可采用设计文档中的「方案 C」或「方案 D」事件数值（见《功德轮回_v3.6_最终版_回测与平衡报告》）。
