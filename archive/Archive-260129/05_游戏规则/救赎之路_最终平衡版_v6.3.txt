▌对称基础 + 道路分化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本设计采用"对称初始+道路分化"方案，确保：
1. 所有玩家起点相同 → 起点公平
2. 差异通过"选择道路"实现 → 策略多样
3. 道路加成对等设计 → 数学平衡

▌初始资源（所有玩家相同）

• 财富：6
• 功德：2
• 影响：3
• 土地：1
• 行动点：2

▌道路一：福田道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】福田（Field of Merit）
"种子播种在功德之田中将会生长。"

【启动奖励】+5影响 +2功德
【专属加成】修行效率+100%（修行获得双倍功德）
【完成条件】回向3位不同玩家 + 消耗影响≥4
【小成条件】回向1位玩家

【策略提示】
专注于修行积累功德，然后通过回向分享给他人。
你的影响力很高，可以大量修行。

▌道路二：布施道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】檀越（Dānapati，布施主）
"必须给予才能获得；必须播种才能收获。"

【启动奖励】+6财富
【专属加成】捐献效率+75%（捐献获得更多功德）
【完成条件】累计捐献≥12 + 当前影响≥3
【小成条件】累计捐献≥7

【策略提示】
大量积累财富然后捐献。
不要把影响花光，完成道路需要保持影响≥3。

▌道路三：土地道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】舍宅为寺
寺院的土地经济是宗教资本的重要形式。

【启动奖励】+1土地 +2财富
【专属加成】土地产出+1（每块土地每轮多产1财富）
【完成条件】已建寺 + 捐地≥3
【小成条件】已建寺 或 捐地≥2

【策略提示】
前期积累土地和财富，中后期建寺和捐地。
注意：完成道路需要同时建寺和捐3块地，规划好资源。

▌道路四：文化道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】文化资本与精神资本
"我的文章能让寺院名扬四海。"

【启动奖励】+2影响 +2财富
【专属加成】善行效率+50%（善行获得更多功德）
【完成条件】影响≥7 + 善行3位不同玩家
【小成条件】影响≥5 或 善行2位玩家

【策略提示】
通过善行与多位玩家建立联系。
同时保持高影响力。平衡积累和互动。

▌道路五：勤劳道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】实践宗教性
普通人通过日常行为积累功德。

【启动奖励】无（但有额外行动点能力）
【专属加成】耕作收益+1 + 每轮额外行动点
【完成条件】善行≥5次 + 耕作≥5次
【小成条件】善行≥3次 或 耕作≥3次

【策略提示】
你有更多行动点，可以同时做更多事。
平衡善行和耕作，两者都要达标。

▌道路六：渡化道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学术概念】功德回向与菩萨道
"我的功德，回向给所有苦难众生。"

【启动奖励】+5功德 +2影响
【专属加成】回向效率+100%（回向给予双倍功德）
【完成条件】回向≥3次 + 渡化2位不同玩家
【小成条件】回向≥1次 或 渡化1位玩家

【策略提示】
你起始功德最高，善用回向帮助他人。
回向会消耗自己的功德但增加影响力。

▌所有玩家可用的行动（7种）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基础行动】每种消耗1行动点

1. 【布施】消耗3财富 → 获得1功德
   • 布施道加成：获得1.75功德（向下取整为1，但有效率加成）

2. 【修行】消耗2影响 → 获得1功德
   • 福田道加成：获得2功德

3. 【善行】消耗1财富 → 你+1功德，选择另一人也+1功德
   • 文化道加成：双方各+1.5功德

4. 【回向】消耗2功德 → 选择另一人+3功德，你+1影响
   • 渡化道加成：目标+6功德

5. 【耕作】无消耗 → 获得2财富
   • 土地道加成：额外+（土地数）财富
   • 勤劳道加成：额外+1财富

【特殊行动】

6. 【建寺】消耗8财富+2影响 → 获得5功德，标记"已建寺"
   • 可用2土地替代8财富

7. 【捐地】消耗1土地 → 获得3功德+1影响

【这是什么游戏？】
你扮演唐宋时期的人物，通过布施积累功德，追求解脱。
功德最多者获胜。完成"道路"可获得额外25分！

【4种资源】
• 💰财富 — 用来捐献（布施/建寺）
• ☸功德 — 决定胜负（得分×2）
• ⭐影响 — 社会地位（用于修行/回向）
• 🏠土地 — 产出财富（可捐可产）

【每轮做什么？】
1. 翻事件卡，选择一个选项
2. 领取资源：+2财富 +1影响 +(土地数)财富
3. 用2个行动点做事

【6种行动】
• 布施：3财富→1功德
• 修行：2影响→1功德
• 善行：1财富→你和另一人各+1功德
• 回向：2功德→另一人+3功德，你+1影响
• 耕作：获得2财富
• 建寺/捐地：特殊高回报行动

【怎样获胜？】
8轮后，功德×2 + 影响 + 财富÷3 + 道路加分 = 总分
分最高者获胜！

▌无偏见测试方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

根据桌游AI测试最佳实践，本测试采用：

1. 【多策略交叉验证】
   • 激进型AI：优先高风险高回报行动
   • 保守型AI：优先稳定收益
   • 平衡型AI：根据道路目标行动
   • 随机型AI：完全随机选择
   • 机会型AI：根据局势调整

2. 【大规模Monte Carlo模拟】
   • 每配置测试10000局
   • 总测试60000+局

3. 【控制变量测试】
   • 混合策略测试（随机分配）
   • 单一策略测试（所有玩家同策略）

▌策略偏见检查结果
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

策略胜率分析表明：
• 有策略的AI（机会型/平衡型）表现更好
• 随机型AI表现最差（0.7-1.8%）

这证明游戏：
✓ 奖励策略性思考
✓ 不是纯运气游戏
✓ 不同策略有不同的最优道路

▌从v4.0到v6.3的平衡之路
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

v4.0：非对称设计，僧伽72%胜率，农人0%胜率
  ↓ 问题：复杂度高，平衡极差

v5.0：简化资源（8→4种），统一行动系统
  ↓ 问题：农夫88%胜率，仍不平衡

v6.0：对称初始+道路分化，多策略AI测试
  ↓ 问题：土地道68%胜率

v6.1：土地道削弱，福田道加强
  ↓ 改善：土地道32%，福田道17%

v6.2：勤劳道削弱，福田道继续加强
  ↓ 改善：勤劳道18%，5人配置全OK

v6.3：最终微调，渡化道/布施道加强
  ↓ 最终：所有配置平衡！
