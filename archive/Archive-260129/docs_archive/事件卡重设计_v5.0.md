# 《功德轮回》事件卡重设计 v5.0

> **已归档**：2026-01-29。事件卡 v5.0 已完整整合进《功德轮回_完整规则书_v5.0》与打印包 print_pack_v50.html；本文件保留设计溯源与详细卡面参考。
>
> 参考：Dead of Winter（危机卡/十字路口卡）、Spirit Island（条件触发事件）、Betrayal（预兆累积）
>
> 核心目标：增加**个人vs集体利益的真实抉择**、**职业差异化响应**、**骰子与玩家互动**

**术语说明**：规则书与打印包统一使用佛教用语——**资粮**（物质，公式中简写「资」）、**功德**（善业所积）、**慧**（智慧）。本设计文档中为简洁仍部分使用「财富/福」，与规则书对应关系：财富=资粮，福=功德。

## 设计理念总结

### 核心改进
1. **英雄时刻系统**：特定职业可以在天灾中"挺身而出"，大幅牺牲换取大幅减灾
2. **条件触发**：效果根据游戏状态变化（劫难高低、资源多少、渡化众生数量）
3. **链式决策**：某些事件按顺位依次选择，第一人的选择影响后续
4. **隐性代价**：袖手旁观也有小代价，但远小于主动牺牲

### 个人事件卡重构
- **4张抉择卡/职业**：大幅牺牲vs大幅自利 + 小幅牺牲vs小幅自利
- **4张骰子/互动卡/职业**：涉及骰子和玩家互动

### 已实装
- 集体事件卡 12张、个人事件卡 32张已全部整合至规则书 v5.0 与 print_pack_v50.html

## 一、集体事件卡重设计（12张）

### 设计原则
1. **差异化机制**：每张天灾有独特的响应规则，不只是换个名字
2. **英雄时刻**：特定职业或满足特定条件的玩家可以"挺身而出"，大幅牺牲换取大幅减灾
3. **旁观者代价**：选择不行动也有小代价，但远小于主动牺牲
4. **条件触发**：根据游戏状态（劫难高低、资源多少）触发不同效果

---

### 天灾共业（4张）- 红色

#### 1. 旱魃肆虐
**基础效果**：劫难 +4  
**抉择机制**（A/B卡同时揭示）：🅰 共济 资粮-2、功德+1，每多1人选A劫难-1；🅱 自保 资粮-1，全选B则劫难额外+2。  
**英雄时刻**：农夫可宣布「开仓放粮」：资粮-4，功德+3，劫难-3，他人本次无需选。

#### 2. 洪水滔天
**基础效果**：劫难 +4  
**抉择**：🅰 筑堤 资粮-2、功德+1；🅱 逃难 功德-1。  
**英雄时刻**：商人「捐资筑堤」：资粮-5，功德+2，劫难-4；若商人资粮≥10 可免费发动。

#### 3. 瘟疫流行
**基础效果**：劫难 +5  
**抉择**：🅰 救治 资粮-1、慧-1、功德+2；🅱 隔离 慧-1。  
**英雄时刻**：学者「著书传方」、僧侣「诵经祈福」。

#### 4. 蝗灾蔽日
**基础效果**：劫难 +4  
**链式决策**：首人选后，后续可选跟随或旁观；全旁观则劫难额外+3。  
**英雄时刻**：资粮≤3 者选 A 时功德+2。

---

### 人祸共业（2张）- 橙色

#### 5. 苛政如虎
**基础效果**：劫难 +3，所有人资粮 -1。条件：劫难≥10 时加重；任何人资粮=0 时该玩家改为功德-1。  
**抵抗**：慧≥8 者可支付慧-2，取消全体资粮损失，自己功德+2。

#### 6. 兵戈四起
**基础效果**：劫难 +3。依次选择：缴纳军粮 / 藏粮于民（掷骰）/ 挺身护民。  
**英雄时刻**：僧侣「调停止战」：功德-4，劫难-5，他人无需选。
