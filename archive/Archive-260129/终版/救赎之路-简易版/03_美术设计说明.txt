▌设计理念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【主题风格】
• 唐宋时期中国佛教文化
• 古雅、简约、禅意
• 学术性与艺术性结合

【色彩基调】
• 主色调：暖金色、米白色、深棕色
• 辅助色：紫色（影响）、绿色（土地）
• 强调色：红色（重要提示）

【字体风格】
• 中文：使用书法风格字体（如：方正清刻本悦宋）
• 英文：使用古典衬线字体（如：Times New Roman）
• 数字：使用清晰的无衬线字体（如：Arial）

【插画风格】
• 参考：唐宋时期佛教壁画、经书插图
• 技法：工笔画风格，线条细腻
• 元素：莲花、祥云、寺院、经文等

▌正面设计（资源区）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【布局】
┌─────────────────────────────────────┐
│  顶部：道路名称（大号字体）          │
│                                     │
│  中央：资源区（4个圆形槽）          │
│  [💰财富] [☸功德] [⭐影响] [🏠土地]│
│                                     │
│  底部：行动点显示区                 │
│  当前行动点：○ ○                    │
│                                     │
│  右下角：道路完成标记区（✓）        │
└─────────────────────────────────────┘

【设计元素】
• 背景：淡雅的米白色或浅金色
• 边框：深棕色细线，古典纹样
• 资源槽：圆形，带图标和文字
• 道路名称：顶部居中，大号字体，道路专属配色

【配色方案】
• 福田道：深紫色背景，金色文字
• 布施道：暖金色背景，深棕色文字
• 土地道：绿色背景，金色文字
• 文化道：淡蓝色背景，深蓝色文字
• 勤劳道：橙色背景，深棕色文字
• 渡化道：白色背景，金色文字

│  顶部：道路名称 + 图标              │
│                                     │

│                                     │

• 背景：与正面相同配色
• 图标：道路专属图标（大号）
• 文字：清晰易读，层次分明
• 边框：与正面一致的古典纹样

【布局】
┌─────────────────┐
│  [道路图标]     │  ← 左上角
│                 │
│   道路名称      │  ← 中央大号字体
│                 │

│                 │

• 布施道：暖金色渐变
• 土地道：绿色渐变
• 文化道：淡蓝色渐变
• 勤劳道：橙色渐变

▌事件卡设计（30张）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【布局】
┌─────────────────┐
│  [类别图标] 天象 │  ← 右上角
│                 │
│   事件名称       │  ← 顶部大号字体
│                 │
│  [小插画]        │  ← 中央
│                 │
│  选项A：         │
│  +3财富          │  ← 左侧，蓝色背景
│                 │
│  选项B：         │
│  +1功德+1影响    │  ← 右侧，绿色背景
│                 │
│  [装饰边框]      │
└─────────────────┘

【设计元素】
• 背景：米白色或淡金色
• 类别图标：右上角，5种不同图标
• 事件名称：顶部，大号字体
• 小插画：中央，与事件相关
• 选项A/B：清晰标注，不同颜色区分
• 边框：古典纹样

【类别图标设计】
• 天象类：☁️ 云朵图标
• 法会类：🕯️ 蜡烛/香炉图标
• 世俗类：🏮 灯笼/集市图标
• 灾难类：⚡ 闪电/警示图标
• 机缘类：✨ 星星/光芒图标

【配色方案】
• 选项A：蓝色背景，白色文字
• 选项B：绿色背景，白色文字
• 或：选项A/B使用不同边框颜色区分

▌资源标记
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【财富标记（金色圆片）】
• 颜色：暖金色（C0 M20 Y100 K0）
• 表面：光滑，可叠放
• 可选：中心带"💰"符号或"财"字

【功德标记（白色莲花）】
• 形状：莲花形（或圆形带莲花图案）
• 颜色：白色或淡金色
• 表面：光滑，可叠放
• 可选：中心带"☸"符号或"德"字

【影响标记（紫色星星）】
• 形状：星形（五角星）
• 颜色：紫色（C50 M50 Y0 K0）
• 表面：光滑，可叠放
• 可选：中心带"⭐"符号或"影"字

【土地标记（绿色六边形）】
• 形状：六边形
• 颜色：绿色（C60 M0 Y100 K0）
• 表面：光滑，可叠放
• 可选：中心带"🏠"符号或"地"字

【设计原则】
• 颜色鲜明，易于区分
• 形状不同，便于识别
• 尺寸统一，可叠放
• 表面光滑，手感好

▌道路完成标记
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计】
• 形状：圆形
• 颜色：金色
• 图案：中心带"✓"符号
• 或：带"完成"文字

▌回合标记
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计选项1：旋转标记】
• 形状：圆形，可旋转
• 中心：数字1-8，可旋转显示
• 颜色：红色边框，白色背景

【设计选项2：独立标记】
• 8个独立标记，分别标注"第1轮"至"第8轮"
• 当前轮数标记放在图板上
