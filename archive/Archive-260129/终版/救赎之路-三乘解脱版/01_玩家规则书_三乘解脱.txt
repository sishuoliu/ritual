▌1.1 什么是三乘？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

佛教将解脱之道分为三乘：

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│    【小乘】              【大乘】               【金刚乘】                   │
│    声闻乘               菩萨乘                 密宗/藏传                    │
│                                                                             │
│    追求个人解脱          普度众生               即身成佛                    │
│    阿罗汉果              菩萨道                 金刚上师                    │
│    稳健保守              利他奉献               高险高报                    │
│                                                                             │
│         ──升乘──→           ──升乘──→                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

【历史背景】
• 小乘：原始佛教，强调个人修行证果（南传佛教为主）
• 大乘：后期发展，强调菩萨道利益众生（汉传佛教为主）
• 金刚乘：密教传统，强调方便法门快速成就（藏传佛教为主）


▌1.2 乘位规则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【起始乘位】
• 所有玩家开局时处于【小乘】

【乘位限制】
• 必须从小乘开始，逐级提升
• 不能跳过中间乘位
• 升乘后不可降回

【升乘时机】
• 每轮行动阶段结束后，可选择尝试升乘
• 每局每人最多升乘2次（小乘→大乘→金刚乘）

▌2.1 升乘流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

升乘分为三个阶段：

┌─────────────────────────────────────────────────────────────────────────────┐
│  第一步：检查资格                                                           │
│  ↓                                                                          │
│  第二步：支付"发心费"（升乘前成本）                                         │
│  ↓                                                                          │
│  第三步：掷骰判定                                                           │
│  ↓                                                                          │
│  成功 → 支付"舍离费"（升乘后成本）→ 获得新乘位能力                          │
│  失败 → 受到"业障"惩罚                                                      │
└─────────────────────────────────────────────────────────────────────────────┘


▌2.2 小乘→大乘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【资格条件】
• 功德 ≥ 8
• 善行次数 ≥ 2（必须帮助过他人）

【发心费】（升乘前支付，无论成败）
• 3 功德
• 2 影响

【掷骰判定】
• 掷1颗D6
• 1-2：失败
• 3-6：成功
• 成功率：66.7%

【失败惩罚】（业障）
• 额外 -2 功德
• 本轮不可再尝试升乘
• 【贪嗔痴】标记：下轮所有行动效率-1

【成功后】
• 支付"舍离费"：-2 财富 -1 土地（舍弃世俗执着）
• 获得【大乘】乘位卡
• 解锁大乘能力

【大乘能力】
┌─────────────────────────────────────────────────────────────────┐
│  【菩提心】善行时，你和目标各额外 +1 功德                        │
│  【回向力】回向时，目标额外 +2 功德                              │
│  【普度愿】游戏结束时，善行覆盖≥3人则 +8 分                     │
└─────────────────────────────────────────────────────────────────┘

【佛教知识】
发菩提心是进入大乘的关键。菩提心即"为利益一切众生而愿成佛"的心愿。
《华严经》云："忘失菩提心，修诸善法，是名魔业。"


▌2.3 大乘→金刚乘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【资格条件】
• 功德 ≥ 15
• 影响 ≥ 6
• 已建寺（必须有道场）

【发心费】（升乘前支付，无论成败）
• 5 功德
• 3 影响
• 3 财富

【掷骰判定】
• 掷1颗D6
• 1-3：失败
• 4-6：成功
• 成功率：50%

【失败惩罚】（业障）
• 额外 -4 功德
• 寺庙降级为"破庙"（不再提供加成）
• 【魔障】标记：2轮内不可再尝试升乘

【成功后】
• 支付"舍离费"：-1 土地 -5 财富（彻底舍离）
• 获得【金刚乘】乘位卡
• 解锁金刚乘能力

【金刚乘能力】
┌─────────────────────────────────────────────────────────────────┐
│  【即身成佛】所有行动效率 +1（布施、修行等都+1功德）             │
│  【坛城加持】每轮生产阶段额外 +2 功德                            │
│  【金刚心】免疫法难事件惩罚                                      │
│  【业力加速】所有失败惩罚翻倍（高风险）                          │
└─────────────────────────────────────────────────────────────────┘

【佛教知识】
金刚乘认为通过特殊修法可以"即身成佛"，不需累世修行。
但修行者需要具备坚固信心，否则容易"入魔"。
这体现为游戏中的高收益伴随高风险。


▌2.4 升乘决策要点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【小乘不升的理由】
• 稳定收益，无失败风险
• 保留资源用于其他投资
• 道路加成已经很强（如布施道）

【升大乘的理由】
• 善行加成强化（适合文化道）
• 普度愿额外得分
• 成功率较高（66.7%）

【升金刚乘的理由】
• 所有行动效率+1（极强）
• 每轮自动+2功德
• 后期翻盘能力

【不升金刚乘的理由】
• 成功率只有50%
• 失败代价巨大（-4功德+破庙）
• 业力加速使失败更痛苦

▌3.1 小乘专属：【禅定】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【禅定】（仅小乘可用）
• 消耗：1 行动点 + 1 影响
• 效果：+2 功德
• 特点：稳定收益，不受事件影响

【佛教知识】
声闻乘以禅定为核心修行法门，通过四禅八定证得阿罗汉果。


▌3.2 大乘专属：【发愿】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【发愿】（仅大乘可用）
• 消耗：2 行动点
• 效果：抽取1张发愿卡，声明完成条件
• 特点：高风险高回报，见第五章详述

【佛教知识】
菩萨发愿是大乘修行的核心。如观音菩萨"闻声救苦"、
地藏菩萨"地狱不空誓不成佛"等大愿。


▌3.3 金刚乘专属：【灌顶】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【灌顶】（仅金刚乘可用）
• 消耗：3 影响 + 2 财富
• 效果：选择1名玩家，该玩家立即升乘（跳过资格检查和发心费）
• 限制：每局每人只能被灌顶1次

【佛教知识】
灌顶是密宗入门仪式，由具格上师传授，
代表加持力的传递和修行资格的授予。

触发：随机1名玩家
效果：掷D6
  • 1-2：发现假经（-3功德）
  • 3-4：普通经卷（+2功德）
  • 5-6：珍贵孤本（+6功德+3影响）

【贝叶经现世】
触发：影响最高的玩家
效果：选择是否参与
  • 放弃：无效果
  • 参与：掷D6，1-3空手而归（-2功德），4-6获得真经（+8功德）
背景：贝叶经是写在贝多罗树叶上的佛经原本，极为珍贵

【伏藏terma】
触发：金刚乘玩家优先，否则随机
效果：掷D6
  • 1：假伏藏曝光（-6功德-4影响，"骗子"标记）
  • 2-3：无功而返（-2功德）
  • 4-5：发现法器（+4功德+2影响）
  • 6：发现伏藏（+10功德+5影响）
背景：terma是莲花生大士埋藏的圣物，由"掘藏师"发现

【舍利分身】
触发：已建寺的玩家
效果：消耗3影响参与
  • 掷D6，1-2碎裂（-3功德-寺庙），3-6显灵（+7功德+寺庙升级）
背景：佛陀舍利被视为最高圣物，可分身供奉于多处

    - 6：法显重生（+10功德+"法显"标记，游戏结束+10分）

【发愿】是大乘玩家的专属行动，代表菩萨的弘愿。

▌5.1 发愿流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 消耗2行动点
2. 从发愿卡堆抽取1张
3. 向所有玩家公开声明
4. 游戏结束时检验是否达成

【达成】获得愿力加分
【未达成】受到"违愿"惩罚

▌5.3 发愿策略
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【低风险愿力】
• 护法愿：保持高影响即可
• 精进愿：正常游戏即可达成

【中风险愿力】
• 普度愿：需要策略性善行
• 回向愿：需要足够功德支撑

【高风险愿力】
• 舍身愿：主动降低自己功德
• 财施愿：可能影响其他发展
• 清净愿：依赖运气避开惩罚事件

【称号系统】
• 阿罗汉：小乘玩家，功德≥25，从未失败
• 菩萨：大乘玩家，完成发愿，善行≥5次
• 金刚上师：金刚乘玩家，无失败事件，功德≥30


═══════════════════════════════════════════════════════════════════════════════
                          第七章：完整回合流程
═══════════════════════════════════════════════════════════════════════════════

【游戏准备】
1. 按简易版设置
2. 每人获得1张【小乘】乘位卡
3. 洗混历史事件卡堆
4. 发愿卡堆放置一旁

【每轮流程】

┌─────────────────────────────────────────────────────────────────────────────┐
│  1. 事件阶段：抽取1张历史事件卡，按指示执行                                 │
│  2. 生产阶段：所有玩家获得资源（同简易版）                                  │
│  3. 行动阶段：轮流执行行动（包括乘位专属行动）                              │
│  4. 升乘阶段：可选择尝试升乘                                                │
│  5. 结算阶段：检查胜利条件                                                  │
└─────────────────────────────────────────────────────────────────────────────┘

【胜利条件】
• 8轮结束后，总分最高者获胜
• 总分 = 功德×2 + 影响 + 财富/3 + 土地×2 + 道路加分 + 乘位加分 + 发愿加分


═══════════════════════════════════════════════════════════════════════════════
                          第八章：策略建议
═══════════════════════════════════════════════════════════════════════════════

【保守策略：小乘稳健】
• 保持小乘，专注道路加成
• 避开风险事件
• 目标：阿罗汉称号+从未失败加分

【平衡策略：大乘发愿】
• 第3-4轮升大乘
• 选择匹配的发愿卡
• 利用善行加成积累功德

【激进策略：金刚冲击】
• 快速积累资源
• 第5-6轮冲击金刚乘
• 利用"所有行动+1"快速得分
• 高风险高回报

【辅助策略：灌顶大师】
• 达到金刚乘后
• 帮助他人升乘获得影响
• 配合"法布施愿"得高分
