# 《救赎之路：功德经济》骰子版

## 版本信息

- **版本号**: v7.0 骰子版
- **更新日期**: 2026-01-28
- **基于**: v6.3 简易版

## 骰子版特点

### 新增内容
- **福报骰子系统**: 每轮生产阶段掷骰，增加游戏随机性
- **骰子配件**: 1个6面骰

### 平衡调整
- 调整道路启动奖励，改善游戏平衡
- 4人模式福田道条件调整（回向从3人改为2人）

### 骰子规则
在生产阶段，每位玩家掷1个6面骰：
- **1-2 平常**: 无额外收益
- **3-4 小福**: +1财富
- **5-6 大福**: +1财富 +1影响
