▌示例：生产阶段掷骰
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】4人游戏第4轮生产阶段

【流程】
1. 所有玩家同时获得基础生产收入
2. 每位玩家轮流掷福报骰子
3. 根据骰子结果获得额外资源

【骰子结果解读】
• 玩家A掷出2（平常）：无额外收益
• 玩家B掷出4（小福）：+1财富
• 玩家C掷出6（大福）：+1财富 +1影响
• 玩家D掷出3（小福）：+1财富

【重要规则】
• 骰子结果是额外收益，不替代基础生产
• 每轮只掷1次骰子
• 骰子结果不受道路能力影响


▌案例：土地道与骰子收益叠加
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】土地道玩家有3块土地，掷出大福(6)

【计算】
• 基础生产：+2财富 +1影响
• 土地产出：+3财富（基础）+3财富（土地道加成）= +6财富
• 骰子收益：+1财富 +1影响

【总收入】
• 财富：2 + 6 + 1 = 9财富
• 影响：1 + 1 = 2影响

▌案例1：骰子是否必须掷？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】玩家不想要骰子的随机性，能否选择不掷？

【规则】
• 骰子版中，掷骰是强制的
• 所有玩家必须在生产阶段掷骰
• 如果想要纯策略体验，请使用简易版

【处理】
• 掷骰是生产阶段的固定流程
• 骰子结果只可能是正面或无影响，不会有惩罚


▌案例2：忘记掷骰怎么办？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】生产阶段忘记掷骰，已经进入行动阶段

【建议处理】
• 如果刚发现：立即补掷，结果正常生效
• 如果已执行多个行动：视为掷出"平常"（无收益），继续游戏
• 避免争议：建议使用检查表确保流程完整

【预防措施】
• 生产阶段口号："收资源，掷骰子"
• 指定一人提醒所有人掷骰


▌案例3：4人模式福田道条件调整
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】4人骰子版中，福田道的完成条件是什么？

【骰子版调整】
• 简易版：回向3位不同玩家 + 消耗影响≥4
• 骰子版4人模式：回向2位不同玩家 + 消耗影响≥4

【原因】
• 4人游戏只有3个其他玩家
• 平衡调整后，回向2人更合理

【注意】
• 5-6人模式仍需回向3人
• 小成条件不变：回向1人


▌案例4：骰子版启动奖励差异
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】骰子版和简易版的启动奖励有什么不同？

【完整对比】
┌─────────┬──────────────────┬──────────────────┐
│  道路   │    简易版        │    骰子版        │
├─────────┼──────────────────┼──────────────────┤
│  福田道 │  +5影响 +2功德   │  +6影响 +2功德   │
│  布施道 │  +6财富          │  +7财富          │
│  土地道 │  +1土地 +2财富   │  +1土地 +1财富   │
│  文化道 │  +2影响 +2财富   │  +3影响 +2财富   │
│  勤劳道 │  无              │  +2财富          │
│  渡化道 │  +5功德 +2影响   │  +5功德 +3影响   │
└─────────┴──────────────────┴──────────────────┘

【说明】
• 调整是为了配合骰子系统平衡游戏
• 经过10000局AI模拟验证

▌案例：资源不足时
• 不能执行资源不足的行动
• 资源最低为0，不会变成负数

▌案例：回向目标选择
• 必须选择其他玩家（不能选自己）
• 可以重复选择同一人

▌案例：建寺的土地替代
• 可用2土地替代8财富
• 仍需消耗2影响
• 不允许部分替代

▌案例：道路完成条件
• 小成条件满足任一即可
• 完成条件必须全部满足
• 加分不叠加

▌案例：同分判定
• 第一判定：功德数量
• 第二判定：影响数量
• 第三判定：财富数量
