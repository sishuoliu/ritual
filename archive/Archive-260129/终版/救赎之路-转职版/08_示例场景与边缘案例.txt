▌示例：完整转职流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】5人游戏第6轮，玩家A（布施道）准备转职

【玩家A状态】
• 财富12，功德8，影响5，土地1
• 已完成小成（累计捐献≥7财富）
• 当前得分排名：第3名（60%位置）

【转职条件检查】
1. ✓ 已完成道路"小成"
2. ✓ 当前轮数≥第5轮（第6轮）
3. 检查转职成本：
   • 基础成本：5功德 + 3影响 + 3财富
   • 排名调整：第3名/5人 = 60%（后50%-75%之间，无调整）
   • 最终成本：5功德 + 3影响 + 3财富

【执行转职】
• 消耗：5功德 + 3影响 + 3财富
• 转职后状态：财富9，功德3，影响2，土地1
• 获得：大檀越职业卡

【转职后效果】
• 解锁3个专属能力（见职业卡）
• 解锁提前胜利条件：捐献≥20财 + 影响≥8


▌示例：弱者转职成本减免
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】6人游戏第7轮，玩家F（勤劳道）是最后一名

【玩家F状态】
• 财富8，功德4，影响4，土地1
• 已完成小成
• 当前得分排名：第6名（最后，在后50%）

【转职成本计算】
• 基础成本：5功德 + 3影响 + 3财富
• 排名调整：后50%，成本-2功德
• 最终成本：3功德 + 3影响 + 3财富

【执行转职】
• 消耗：3功德 + 3影响 + 3财富
• 转职后状态：财富5，功德1，影响1，土地1
• 获得：行脚僧职业卡

【设计目的】
• 帮助落后玩家获得翻盘机会
• 防止"强者恒强"


▌示例：提前胜利达成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】第9轮结束时，已转职的大檀越检查提前胜利

【玩家状态】
• 累计捐献：22财富 ✓（需≥20）
• 当前影响：9 ✓（需≥8）

【提前胜利判定】
1. 已完成转职 ✓
2. 满足提前胜利条件 ✓
3. 宣布提前胜利！

【重要规则】
• 提前胜利在行动阶段结束后检查
• 如果多人同时满足，比较总得分
• 提前胜利优先于回合结束后的正常胜利

▌案例1：转职是否必须？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】玩家已满足转职条件，能否选择不转职？

【规则】
• 转职是可选的，不是强制的
• 玩家可以选择不转职，继续走"稳健型"路线
• 稳健型道路（福田/土地/勤劳）通常更适合不转职

【处理】
• 满足条件后，玩家在转职阶段决定是否转职
• 选择不转职后，本轮仍可在后续轮次转职
• 一旦转职不可撤销


▌案例2：资源不足无法转职
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】玩家已完成小成，但资源不足以支付转职成本

【规则】
• 必须有足够资源才能转职
• 资源不足时不能转职

【处理】
• 等待积累足够资源后再转职
• 可以先执行行动获取资源
• 转职阶段在生产之后，可利用当轮生产的资源


▌案例3：前25%玩家转职成本增加
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】4人游戏中第1名想转职

【计算】
• 第1名/4人 = 25%（前25%）
• 成本调整：+2功德
• 最终成本：7功德 + 3影响 + 3财富

【设计目的】
• 平衡强者，避免领先者通过转职进一步拉开差距


▌案例4：转职后道路加分如何计算
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】已转职玩家完成道路，加分如何计算？

【规则】
• 小成：+12分（转职时已完成）
• 完成道路：+25分（转职后仍可完成）
• 提前胜利：直接获胜（不需要计算分数）

【处理】
• 转职需要先完成小成，所以至少有+12分
• 转职后可继续完成道路获得+25分
• 如果达成提前胜利，直接获胜无需比分


▌案例5：多人同时提前胜利
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】第9轮结束时，两位玩家同时满足提前胜利条件

【规则】
• 同时满足时，比较当前总得分
• 得分高者获胜

【处理】
• 计算双方当前得分（功德×2 + 影响 + 财富÷3 + 土地×2 + 道路加分）
• 得分高者获胜
• 如仍平局，按标准同分规则判定


▌案例6：第5轮前能否转职？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】玩家在第4轮就完成了小成，能否立即转职？

【规则】
• 转职最早可在第5轮进行
• 第4轮及之前不能转职

【设计目的】
• 确保游戏前期专注于基础发展
• 避免过早转职导致游戏失衡


▌案例7：转职后道路能力是否保留？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情境】布施道转职为大檀越后，还有布施翻倍效果吗？

【规则】
• 原道路能力保留
• 转职后额外获得3个专属能力
• 能力叠加

【处理】
• 大檀越仍享有布施道的捐献翻倍效果
• 同时获得大檀越的3个专属能力
• 这是转职的价值之一
