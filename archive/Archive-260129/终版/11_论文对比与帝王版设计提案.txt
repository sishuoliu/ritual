沃尔什论文提出了佛教经济学的六项核心交易模式：

┌────┬─────────────────────────────────────────────────────────────────────┐
│ ID │                        核心概念                                      │
├────┼─────────────────────────────────────────────────────────────────────┤
│ T1 │ 皇帝御笔捐赠：皇帝向寺院捐赠书法/敕书，寺院因此繁荣                 │
├────┼─────────────────────────────────────────────────────────────────────┤
│ T2 │ 皇室-寺院关系：帝国与寺院的互惠合作                                  │
├────┼─────────────────────────────────────────────────────────────────────┤
│ T3 │ 祈福延寿：僧侣为皇室祈福，换取保护和资源                            │
├────┼─────────────────────────────────────────────────────────────────────┤
│ T4 │ 报答皇恩：寺院需要"报答皇恩"，形成义务关系                          │
├────┼─────────────────────────────────────────────────────────────────────┤
│ T5 │ 功德生产：僧侣必须劳作生产功德，"不耕而食"是可耻的                  │
├────┼─────────────────────────────────────────────────────────────────────┤

└────┴─────────────────────────────────────────────────────────────────────┘

论文核心洞见：

▌1. 功德是"商品"而非"礼物"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"功德从未在没有预期回报的情况下被自由授予"
- 功德是劳动产品
- 功德有使用价值和交换价值
- 功德交换是双向互惠的

▌2. "福田"农业隐喻
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"向僧团捐献就像在大福田中播种——它将生长并提供更多回报"
- 播种（投入）→ 耕耘（劳作）→ 收获（回报）
- 农业文化下的直观理解
- 功德可以"增广"和"积累"

▌3. 双重资本系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
┌──────────────────────────────────────────────────────────────────────────┐
│  经济资本（土地、财富）  ←→  文化资本（声望、地位）                      │
│                                                                          │
│  • 土地捐献 → 功德 → 救赎                                               │
│  • 碑铭竖立 → 认可 → 声望                                               │
│  • 经济资本可转化为文化资本，反之亦然                                    │
└──────────────────────────────────────────────────────────────────────────┘

▌4. 皇室-寺院关系的核心性
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 佛教在中国的成功很大程度上依赖皇权支持
- 皇帝需要寺院：仪式、祈福、社会稳定
- 寺院需要皇帝：保护、土地、合法性
- 这种互惠关系是佛教经济的重要组成部分

▌5. 功德的多样化获取方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
论文引用《大藏经》中记载的七种功德行为：
1. 帮助建造寺庙或宝塔
2. 种树
3. 给病人送药
4. 建造坚固的船只
5. 修桥
6. 修建灌溉渠道
7. 建造公共厕所

▌6. 回向功德的重要性
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"向当地僧团捐献，可以将所获功德回向给已故亲属"
"回向功德的行为增加了施主最初积累的功德"
- 功德可以转移给第三方（尤其是亡者）
- 回向本身也产生功德（增值效应）

▌建议1：增加"皇恩"事件卡（简单改动）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
将论文中的皇室-寺院关系引入事件卡系统

【新事件卡示例】

┌────────────────────────────────────────────────────────────────────┐
│ 【御笔赐寺】                                                        │
│ 皇帝御笔赐予功德最高的玩家。                                       │
│ • 功德最高者：+3影响 +2功德（文化资本提升）                        │
│ • 其他玩家：选择 消耗1功德"朝贡" 或 不参与                        │
└────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────┐
│ 【皇室法会】                                                        │
│ 皇帝召开水陆法会，需要高僧祈福。                                   │
│ • 影响≥5者：可参与，消耗2影响→+4功德                              │
│ • 其他玩家：不参与                                                 │
└────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────┐
│ 【护国祈福】                                                        │
│ 边疆有战事，皇帝请寺院祈福。                                       │
│ • 选择A：消耗3功德为国祈福→+5影响（报答皇恩）                     │
│ • 选择B：不参与（无收益）                                          │
└────────────────────────────────────────────────────────────────────┘


▌建议2：增加"碑铭"得分机制（中等改动）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
体现论文中"文化资本"的积累和展示

【规则】
• 每次"建寺"后，可以选择"立碑"（额外消耗2影响）
• 立碑效果：游戏结束时，每块碑+3分
• 碑上记录：建寺者姓名、回合数
• 主题意义：碑是声望的物质化展示

【配件】
• 碑铭标记：6个小型石碑标记


▌建议3：增加"回向亡者"机制（中等改动）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
体现论文中功德可转移给死者的概念

【规则】
• 修改回向行动：可以选择回向给"亡者牌堆"
• 回向亡者：消耗2功德→+2影响 +1碑铭标记
• 主题意义：为祖先积累功德，自己获得社会认可

【配件】
• 亡者牌堆：放在桌子中央
• 祭祀标记：每次回向亡者放置一个


▌建议4：增加"福田增广"机制（较大改动）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
体现论文中功德"生长"和"增广"的概念

【规则】
• 每位玩家有一个"福田区"
• 每轮结束时，福田区中的功德+10%（向上取整）
• 福田区功德在游戏结束时计入总分
• 玩家可以选择将功德"存入"福田区（行动）

【效果】
• 早期存入的功德会"生长"
• 体现"播种-收获"的农业隐喻
• 增加长期规划的策略深度

▌设计理念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

帝王版的核心创新：引入"皇帝"作为独立角色/机制，体现论文中强调的
皇室-寺院互惠关系。这是佛教经济学最独特的面向之一。

【设计目标】
1. 体现皇帝与寺院的互惠关系
2. 增加政治-宗教的交互维度
3. 创造更复杂的博弈局面
4. 保持原有机制的完整性

▌方案A：皇帝作为NPC（推荐，改动适中）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【概述】
皇帝是一个由规则驱动的NPC，每轮与玩家互动。
玩家可以选择"争取皇恩"或"独立发展"两条路线。

【皇帝系统】

▌方案B：皇帝作为玩家角色（改动较大）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【概述】
增加"皇帝"作为第七个可选角色。
皇帝不走六道路，而是走"护法道"。

【皇帝角色设计】

┌──────────────────────────────────────────────────────────────────────────┐
│                         皇帝·护法道                                       │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│ 【启动资源】                                                             │
│ 财富：12（最多）                                                         │
│ 功德：0（最少）                                                          │
│ 影响：8（特殊）                                                          │
│ 土地：3（帝国土地）                                                      │
│                                                                          │
│ 【特殊能力】                                                             │
│ • 不能直接获得功德（皇帝本人不修行）                                    │
│ • 通过"赐予"他人获得功德（间接获得）                                    │
│ • 每当其他玩家获得功德，皇帝可消耗1影响→+1功德（护法加成）              │
│                                                                          │
│ 【专属行动】                                                             │
│ • 御赐土地：消耗2土地→选一玩家获得1土地，你+3功德                       │
│ • 敕建寺院：消耗8财富→选一玩家"已建寺"状态，你+5功德                   │
│ • 召开法会：消耗5财富→所有玩家+1功德，你+影响等于玩家数                 │
│                                                                          │
│ 【完成条件】                                                             │
│ • 护法道完成：所有其他玩家都完成了道路小成                               │
│ • 护法道小成：至少3位玩家完成了道路小成                                  │
│                                                                          │
│ 【策略】                                                                 │
│ 皇帝通过帮助他人完成道路来获得功德。                                     │
│ 体现"护法"精神：保护佛法，支持寺院发展。                                │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

【游戏动态变化】
• 皇帝与其他玩家形成互惠关系
• 玩家需要决定是否接受皇帝的"恩赐"
• 接受恩赐意味着帮助皇帝完成条件
• 拒绝恩赐意味着独立发展但可能更慢


▌方案C：轮值皇帝制（创新设计）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【概述】
每轮有一位玩家担任"轮值皇帝"，拥有特殊权力。

【规则】
• 第1轮：随机决定首位皇帝
• 之后每轮：皇帝轮换（顺时针）
• 皇帝该轮有特殊能力和责任

【皇帝权力】
• 赐恩：可以选择1位玩家+2功德（包括自己）
• 召令：可以强制1位玩家参与法会事件
• 护法：皇帝该轮行动+1（共3个行动点）

【皇帝责任】
• 如果该轮有灾难事件，皇帝-3财富（赈灾）
• 如果没有玩家建寺，皇帝-2影响（失望）

【设计意图】
• 每位玩家都体验"皇帝"视角
• 理解皇帝与寺院的互惠关系
• 增加每轮的变数和策略考量

▌帝王版策略指南
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【争取皇恩路线】
• 适合：福田道、渡化道（功德充裕）
• 策略：早期祈福积累皇恩，中期御赐土地，后期极恩加分
• 风险：功德消耗大，可能影响道路完成

【独立发展路线】
• 适合：布施道、勤劳道（不依赖皇恩）
• 策略：专注道路完成，忽略皇恩系统
• 风险：遇到法难事件损失大

【平衡路线】
• 适合：文化道、土地道
• 策略：维持皇恩≥3（认可线），专注道路
• 目标：避免惩罚，获得小幅加成

                          第七部分：主题深化建议
═══════════════════════════════════════════════════════════════════════════════

除了帝王版，以下改动可以更好体现佛教经济学：

【建议1】重命名"影响"为"法力"或"威德"
• 原因：论文中没有"影响"概念，有"声望"和"护法"
• 改为：法力（修行所得）或威德（社会认可）

【建议2】增加"福田"视觉化
• 在玩家图板上增加"福田区"图示
• 功德标记放在福田区内
• 体现"播种-收获"的隐喻

【建议3】事件卡增加历史背景
• 每张事件卡加入简短历史说明
• 例如："盂兰盆会"来自《盂兰盆经》
• 增强教育价值和主题沉浸感

【建议4】增加"七善行"变体
• 论文列出7种善行（修桥、种树、济病等）
• 可设计为可选模块
• 每轮抽取1种"本轮善行"，执行者获得额外奖励

                              结论
═══════════════════════════════════════════════════════════════════════════════

【现状评估】
当前游戏已较好地体现了论文约70%的核心概念。
最大的缺失是"皇室-寺院关系"——这是论文中反复强调的核心交易模式。

【帝王版价值】
帝王版可以填补这一空白，让玩家体验：
• 宗教与政治的互惠关系
• "报答皇恩"的义务与回报
• 皇权保护下的寺院发展
• 历史上"法难"的风险

【推荐实施路径】
1. 先在简易版中加入3-4张皇室事件卡（试水）
2. 如果反馈好，开发完整帝王版
3. 帝王版作为独立SKU或扩展包发行

【最终目标】
让玩家通过游戏体验和理解：
"要获得必先给予；要接受必先施予"
——佛教救赎经济学的核心法则
