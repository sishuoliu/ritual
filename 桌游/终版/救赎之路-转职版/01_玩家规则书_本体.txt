▌游戏背景
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在唐宋时期的中国，佛教寺院不仅是宗教中心，更是重要的经济实体。
信众通过布施积累功德，追求来世的解脱或现世的福报。

在这款游戏中，你扮演一位追求解脱的修行者。
通过布施、修行、善行，积累功德，完成你的"道路"。

核心机制：功德交换
"你必须给予，才能获得；你必须播种，才能收获。"

▌游戏目标
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在8轮游戏结束后，得分最高的玩家获胜。
（转职版为10轮）

得分 = 功德×2 + 影响 + 财富÷3 + 土地×2 + 道路加分

完成你的专属"道路"可以获得大量加分（+25分）！

▌设置步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 【选择道路】
   每位玩家从6条道路中选择1条（不能重复）。
   将对应的道路卡放在自己面前。

2. 【领取初始资源】
   所有玩家初始资源相同：
   • 财富：6
   • 功德：2
   • 影响：3
   • 土地：1
   
   然后根据选择的道路，领取"启动奖励"：
   
   ┌─────────┬──────────────────────────────────────┐
   │  道路   │  启动奖励                            │
   ├─────────┼──────────────────────────────────────┤
   │  福田道 │  +5影响 +2功德                      │
   │  布施道 │  +6财富                              │
   │  土地道 │  +1土地 +2财富                      │
   │  文化道 │  +2影响 +2财富                      │
   │  勤劳道 │  无（但有额外行动点）                │
   │  渡化道 │  +5功德 +2影响                      │
   └─────────┴──────────────────────────────────────┘

3. 【准备事件卡】
   将30张事件卡洗混，放在桌子中央。

4. 【放置回合标记】
   将回合标记放在"第1轮"位置。

5. 【决定起始玩家】
   随机决定一位玩家作为起始玩家。

▌资源说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【财富】
• 用途：用于布施、善行、建寺
• 获得：每轮生产+2，耕作+2，土地产出
• 得分：最终财富÷3计入总分

【功德】
• 用途：决定胜负的核心资源（得分×2）
• 获得：布施、修行、善行、回向、事件卡
• 得分：最终功德×2计入总分

【影响】
• 用途：用于修行、回向、建寺
• 获得：每轮生产+1，回向+1，事件卡
• 得分：最终影响×1计入总分

【土地】
• 用途：产出财富，可捐地获功德
• 获得：初始1块，土地道启动+1
• 得分：最终土地×2计入总分

▌道路一：福田道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】+5影响 +2功德
【专属加成】修行效率+100%（修行获得双倍功德）

【完成条件】
• 回向3位不同玩家
• 消耗影响≥4

【小成条件】回向1位玩家（+12分）

【策略提示】
你起始影响很高，专注于修行积累功德。
然后通过回向分享给他人，完成道路条件。

▌道路二：布施道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】+6财富
【专属加成】捐献翻倍（布施获得2功德而非1功德）

【完成条件】
• 累计捐献≥12财富
• 当前影响≥3

【小成条件】累计捐献≥7财富（+12分）

【策略提示】
大量积累财富然后捐献。
注意：完成道路需要保持影响≥3，不要把所有影响都花光。

▌道路三：土地道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】+1土地 +2财富
【专属加成】土地产出+1（每块土地每轮多产1财富）

【完成条件】
• 已建寺
• 捐地≥3块

【小成条件】已建寺 或 捐地≥2块（+12分）

【策略提示】
前期积累土地和财富，中后期建寺和捐地。
注意：完成道路需要同时建寺和捐3块地，规划好资源。

▌道路四：文化道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】+2影响 +2财富
【专属加成】善行翻倍（善行双方各获得2功德而非1功德）

【完成条件】
• 影响≥7
• 善行3位不同玩家

【小成条件】影响≥5 或 善行2位玩家（+12分）

【策略提示】
通过善行与多位玩家建立联系。
同时保持高影响力。平衡积累和互动。

▌道路五：勤劳道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】无（但有额外行动点能力）
【专属加成】耕作收益+1 + 每轮额外行动点

【完成条件】
• 善行≥5次
• 耕作≥5次

【小成条件】善行≥3次 或 耕作≥3次（+12分）

【策略提示】
你有更多行动点，可以同时做更多事。
平衡善行和耕作，两者都要达标。

▌道路六：渡化道
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【启动奖励】+5功德 +2影响
【专属加成】回向效率+100%（回向给予双倍功德）

【完成条件】
• 回向≥3次
• 渡化2位不同玩家（通过回向或善行）

【小成条件】回向≥1次 或 渡化1位玩家（+12分）

【策略提示】
你起始功德最高，善用回向帮助他人。
回向会消耗自己的功德但增加影响力。

▌基础行动（消耗1行动点）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【1. 布施】
• 消耗：3财富
• 获得：1功德
• 布施道加成：获得2功德（翻倍）

【2. 修行】
• 消耗：2影响
• 获得：1功德
• 福田道加成：获得2功德

【3. 善行】
• 消耗：1财富
• 效果：你+1功德，选择另一人也+1功德
• 文化道加成：双方各获得2功德（翻倍）

【4. 回向】
• 消耗：2功德
• 效果：选择另一人+3功德，你+1影响
• 渡化道加成：目标+6功德

【5. 耕作】
• 消耗：无
• 获得：2财富
• 土地道加成：额外+（土地数）财富
• 勤劳道加成：额外+1财富

▌特殊行动
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【6. 建寺】
• 消耗：8财富 + 2影响
• 获得：5功德
• 标记：在玩家图板上标记"已建寺"
• 替代：可用2土地替代8财富

【7. 捐地】
• 消耗：1土地
• 获得：3功德 + 1影响

▌事件卡规则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

每轮开始时，翻开1张事件卡。
所有玩家同时选择该事件卡的选项A或选项B。
选择后立即执行效果。

【天象类】6张

• 灾异（2张）：选择 -3财富 或 -2功德

【法会类】6张
• 盂兰盆会（2张）：选择 消耗2财富+3功德 或 不参与

• 瘟疫（2张）：选择 -2功德 或 -2影响

【机缘类】6张
• 高僧（2张）：选择 +3功德 或 +2功德+1影响

▌每轮四个阶段
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【阶段1：天机】
1. 翻开1张事件卡
2. 所有玩家同时选择选项A或B
3. 执行事件效果

【阶段2：生产】
所有玩家同时获得：
• +2财富
• +1影响
• +（土地数）财富（每块土地产1财富）
• 土地道额外+（土地数）财富

【阶段3：行动】
从起始玩家开始，顺时针依次执行行动。
每位玩家有2个行动点（勤劳道有3个）。
用完行动点后轮到下一位玩家。

【阶段4：结算】
1. 检查道路完成情况
2. 移动回合标记到下一轮
3. 传递起始玩家标记（顺时针）

▌游戏结束
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

游戏在第8轮（转职版：第10轮）结束后结算。

计算每位玩家的最终得分：
得分 = 功德×2 + 影响 + 财富÷3 + 土地×2 + 道路加分

道路加分：
• 完成道路：+25分
• 完成小成：+12分

得分最高者获胜！
如果平局，功德最高者获胜。
