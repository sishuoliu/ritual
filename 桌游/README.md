# 救赎之路：功德经济 - 桌游项目

**Path to Salvation: The Merit Economy**

## 项目概述

基于学术论文《救赎的经济学：中国佛教交换理论》设计的策略桌游。

## 目录结构

```
桌游/
├── 终版/                        # 三个终版游戏
│   ├── 救赎之路-简易版/         # v6.3 纯净基础版
│   ├── 救赎之路-骰子版/         # v7.0 带生产骰子
│   ├── 救赎之路-转职版/         # v8.3 本体+转职
│   └── 版本清单.txt             # 版本对比说明
├── 01_代码_模拟器/              # 模拟器代码
│   ├── 转职版模拟器/            # v7.x-v8.x 转职版模拟器
│   └── (其他版本模拟器)
├── 04_测试数据/                 # 测试结果数据
│   ├── 转职版测试/              # v7.x-v8.x 测试数据
│   └── (其他版本测试数据)
└── README.md                    # 本文件
```

## 三个终版

| 版本 | 版本号 | 轮数 | 特点 | 推荐人群 |
|------|--------|------|------|---------|
| 简易版 | v6.3 | 8轮 | 纯净基础版 | 新手入门 |
| 骰子版 | v7.0 | 8轮 | 带生产骰子 | 喜欢随机性 |
| 转职版 | v8.3 | 10轮 | 本体+转职 | 策略玩家 |

## 快速开始

1. **新手玩家**：从 `终版/救赎之路-简易版/` 开始
2. **策略玩家**：查看 `终版/救赎之路-转职版/`
3. **开发者**：模拟器代码在 `01_代码_模拟器/`

## 版本历史

- **v6.3** - 简易版最终平衡
- **v7.0** - 骰子版
- **v8.3** - 转职版最终平衡

## 平衡性验证

所有版本都经过大规模模拟测试：
- 简易版：经过v1-v6迭代，最终v6.3达成平衡
- 骰子版：专门的骰子系统测试
- 转职版：10000轮模拟，v8.3达成平衡目标
