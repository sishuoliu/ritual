# 佛教经济学桌游项目

**Buddhist Economics Board Game Project**

基于学术论文《救赎的经济学：中国佛教交换理论》(The Economics of Salvation: Toward a Theory of Exchange in Chinese Buddhism) 设计的策略桌游。

---

## 项目包含两个游戏系列

### 1. 救赎之路 (Path to Salvation)
竞争型策略桌游，玩家扮演唐宋时期不同身份角色，通过资源管理和行动选择积累功德。

### 2. 功德轮回：众生百态 (Merit Cycle: Lives of All Beings) ★新★
**半合作型**策略桌游，玩家合作拯救众生、化解劫难，同时竞争个人修行果位。

## 功德轮回：众生百态 v0.6 ★最新设计★

### 核心特色

| 特色 | 说明 |
|------|------|
| **半合作模式** | 团队必须合作拯救众生，个人可以竞争修行果位 |
| **修行阶段系统** | 4阶段修行：凡夫→初果→菩萨→圆满 |
| **交易系统** | 布施、借贷、承诺交易，体现佛教经济学 |
| **三毒系统** | 贪嗔痴阻碍修行，需要克服 |
| **发愿系统** | 12张发愿卡，利他/平衡/自利三类 |
| **众生渡化** | 20张众生卡，需要合作救度 |
| **骰子判定** | 2d6判定升级成功，高风险高回报 |

### 设计历程

```
v0.1 纯竞争模式 → v0.3 加入发愿系统 → v0.4 合作模式重构 
→ v0.5 加入修行阶段 → v0.6 半合作模式+交易系统 ★最终定稿★
```

### 胜利条件

**团队必须达成（合作）：**
1. 劫难≤12
2. 渡化众生≥5
3. 建寺≥2
4. 所有玩家功德≥12
5. 至少1人达到修行阶段3

**个人竞争：**
- 最高分者获得"佛果"称号（唯一胜者）
- 其他人获得菩萨/阿罗汉/初果等果位

### 四种身份

| 身份 | 专长 | 升级难度 | 策略定位 |
|------|------|----------|----------|
| 僧侣 | 修行、讲经 | 容易（+2骰子） | 修行领袖 |
| 贵族 | 赈灾、建寺 | 中等（需财富） | 财力支持 |
| 商人 | 贸易、布施 | 困难（-1骰子） | 经济引擎 |
| 农夫 | 虔诚布施 | 容易（成本低） | 虔诚根基 |

| 布施波罗蜜 | 财布施、法布施、无畏施 |
| 正命 | 正当交易 vs 高利盘剥 |
