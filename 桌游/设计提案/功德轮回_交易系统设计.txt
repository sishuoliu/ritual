▌Catan为什么交易系统成功？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心要素】
1. 非零和博弈：双方都能受益
2. 资源稀缺性：每人缺不同资源
3. 自由谈判：无限制，纯社交博弈
4. 信息不对称：不知道对方真实意图
5. 时机重要：领先者难获得交易

【Catan交易规则】
• 只有当前回合玩家可发起交易
• 可与任意玩家交易
• 可交易任意数量资源
• 完全自由谈判（甚至可以"贿赂"）
• 与银行交易：4:1或3:1（港口）


▌Bohnanza为什么交易系统成功？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心要素】
1. 交易是核心机制（非可选）
2. 必须在特定阶段交易
3. 交易物必须立即使用
4. 可以无偿赠送
5. 完全自由谈判

【Bohnanza交易规则】
• 仅在"翻牌与交易"阶段
• 可交易手牌或刚翻开的牌
• 交易后必须立即种植
• 可以拒绝交易
• 甚至可以免费赠送


▌合作游戏的交易问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【研究发现】（来自搜索）
• 交易系统可以增强合作游戏的互动
• 但需要小心设计，避免导致"贪婪和竞争"
• Eco游戏实验：加入资本主义激励后，出现"争吵、贪婪、金融危机"
• 资源稀缺是必要的（无限资源让游戏无意义）

【关键教训】
• 交易系统要服务于共同目标
• 不能让交易成为纯粹的个人积累
• 需要设计限制，避免"囤积居奇"

▌2.1 什么可以交易？什么不能交易？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【可以交易的资源】

┌──────────────────────────────────────────────────────────────────────┐
│  ✓ 财富（Wealth）                                                    │
│    • 最基本的交易物                                                  │
│    • 可自由交易任意数量                                              │
│    • 佛教意义：财布施，舍财求法                                      │
│                                                                      │
│  ✓ 影响（Influence）                                                │
│    • 可交易，但有限制                                                │
│    • 每轮最多交易5点影响                                             │
│    • 佛教意义：世俗权力，可以让渡                                    │
│                                                                      │
│  ✓ 土地（Land）                                                     │
│    • 可以出售或赠送                                                  │
│    • 佛教意义：产业，可舍可得                                        │
│                                                                      │
│  ✓ 商路（Trade Routes）                                             │
│    • 可以共享收益（不能完全转让）                                    │
│    • 佛教意义：合作共赢                                              │
│                                                                      │
│  ~ 特殊物品（药物、著作等）                                          │
│    • 可以交易，但需要对方同意                                        │
│    • 有特定使用限制                                                  │
└──────────────────────────────────────────────────────────────────────┘

【不能交易的资源】

┌──────────────────────────────────────────────────────────────────────┐
│  ✗ 功德（Merit）                                                     │
│    • 绝对不可交易                                                    │
│    • 佛教意义：功德是个人修行所得，不可转让                          │
│    • 例外：僧侣"回向"行动可帮他人获得功德，但自己功德不减少          │
│                                                                      │
│  ✗ 修行阶段（Cultivation Stage）                                    │
│    • 不可交易或转让                                                  │
│    • 佛教意义：证果是个人悟道，无法交易                              │
│                                                                      │
│  ✗ 三毒（Three Poisons）                                            │
│    • 不可交易                                                        │
│    • 佛教意义：业力是自作自受                                        │
│                                                                      │
│  ✗ 戒律/虔诚/声望（Special Tracks）                                 │
│    • 不可直接交易                                                    │
│    • 佛教意义：修行积累，无法转让                                    │
│                                                                      │
│  ✗ 发愿卡（Vow Cards）                                              │
│    • 不可交换或交易                                                  │
│    • 佛教意义：发愿是个人誓言，不可转让                              │
└──────────────────────────────────────────────────────────────────────┘


▌2.2 交易规则设计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【交易时机】

┌──────────────────────────────────────────────────────────────────────┐
│  方案A：自由交易（Catan式）                                          │
│    • 任何时候都可以发起交易                                          │
│    • 在协商阶段特别鼓励                                              │
│    • 行动阶段也可以临时交易                                          │
│    • 优点：灵活，社交互动多                                          │
│    • 缺点：可能拖慢游戏节奏                                          │
│                                                                      │
│  方案B：限定阶段交易（Bohnanza式）★推荐★                            │
│    • 仅在"协商阶段"可以交易                                          │
│    • 3分钟协商时间                                                   │
│    • 所有交易必须在此阶段完成                                        │
│    • 优点：节奏清晰，不影响其他阶段                                  │
│    • 缺点：灵活性稍低                                                │
└──────────────────────────────────────────────────────────────────────┘

【推荐：方案B（限定阶段）】
理由：控制游戏节奏，避免无休止谈判


▌2.3 交易类型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【1. 即时交易】（当轮结算）
┌──────────────────────────────────────────────────────────────────────┐
│  玩家A："我给你3财富，换你2影响"                                     │
│  玩家B："好"                                                         │
│  → 立即交换资源                                                      │
│                                                                      │
│  规则：                                                              │
│  • 双方自由协商比例                                                  │
│  • 可以不等价交易                                                    │
│  • 立即结算                                                          │
└──────────────────────────────────────────────────────────────────────┘

【2. 承诺交易】（未来兑现）
┌──────────────────────────────────────────────────────────────────────┐
│  玩家A："我这轮给你3财富，你下轮帮我建寺好吗？"                      │
│  玩家B："好的"                                                       │
│  → 当轮给予3财富，下轮B需要履行                                      │
│                                                                      │
│  规则：                                                              │
│  • 口头承诺，无强制执行                                              │
│  • 若违背承诺：+2嗔，-2影响（业力惩罚）                              │
│  • 若履行承诺：-1嗔（积累信誉）                                      │
│                                                                      │
│  ★ 关键设计：                                                       │
│  • 允许"背叛"承诺（创造半合作张力）                                  │
│  • 但有业力代价（三毒系统）                                          │
│  • 被背叛者可拒绝未来合作                                            │
└──────────────────────────────────────────────────────────────────────┘

【3. 无偿赠送】（布施）
┌──────────────────────────────────────────────────────────────────────┐
│  玩家A："我无偿给你5财富"                                            │
│  玩家B："谢谢"                                                       │
│  → A获得布施功德                                                     │
│                                                                      │
│  规则：                                                              │
│  • 无偿给予任意资源                                                  │
│  • 给予者：+功德（根据数量）                                         │
│  • 接受者：无需回报                                                  │
│  • 佛教意义：布施波罗蜜，不求回报                                    │
│                                                                      │
│  布施功德表：                                                        │
│  • 1-2财富：+1功德                                                   │
│  • 3-5财富：+2功德                                                   │
│  • 6-10财富：+4功德                                                  │
│  • 10+财富：+6功德                                                   │
└──────────────────────────────────────────────────────────────────────┘

【4. 借贷】（有偿借贷）
┌──────────────────────────────────────────────────────────────────────┐
│  玩家A："借我5财富，下轮还你6财富"                                   │
│  玩家B："好"                                                         │
│  → 当轮给予5财富，下轮需归还6                                        │
│                                                                      │
│  规则：                                                              │
│  • 可约定利息（最多20%）                                             │
│  • 若不归还：+3贪，-3影响，被追债                                    │
│  • 若归还：履行承诺，-1贪                                            │
│                                                                      │
│  高利贷惩罚：                                                        │
│  • 若利息>20%：放贷者+2贪                                            │
│  • 借贷者可拒绝高利贷                                                │
│                                                                      │
│  ★ 佛教意义：                                                       │
│  • 借贷可以，但不可高利盘剥                                          │
│  • 不归还债务是"不诚信"，有业力惩罚                                  │
└──────────────────────────────────────────────────────────────────────┘

▌3.1 交易 vs 布施
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【佛教经济学的核心矛盾】

┌──────────────────────────────────────────────────────────────────────┐
│  交易（等价交换）                                                    │
│  • 你给我3财富，我给你2影响                                          │
│  • 双方计较得失                                                      │
│  • 世俗经济行为                                                      │
│  • 不增加功德                                                        │
│                                                                      │
│  布施（无偿给予）                                                    │
│  • 我给你5财富，不求回报                                             │
│  • 放下执着                                                          │
│  • 佛教核心实践                                                      │
│  • 增加功德                                                          │
│                                                                      │
│  ★ 游戏设计意义：                                                   │
│  • 玩家可以选择"交易"（不得功德）或"布施"（得功德）                  │
│  • 创造道德抉择：要功德还是要等价？                                  │
└──────────────────────────────────────────────────────────────────────┘


▌3.2 交易的三毒风险
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【贪】
• 在交易中索要过高价格：+1贪
• 拒绝合理交易（自己囤积）：+1贪
• 高利贷（利息>20%）：+2贪

【嗔】
• 违背交易承诺：+2嗔
• 拒绝归还借贷：+3嗔
• 恶意抬价：+1嗔

【痴】
• 接受明显不利的交易：+1痴
• 被高利贷盘剥：+1痴

【设计意义】
• 交易不是"无代价"的
• 贪婪的交易会增加三毒
• 慈悲的布施会减少三毒

▌4.1 交易阶段（在协商阶段进行）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【时间】
• 每轮协商阶段（3分钟）
• 所有交易必须在此阶段完成
• 行动阶段不可交易

【参与者】
• 任意两名玩家可发起交易
• 第三方可以参与（三方交易）
• 所有交易都是公开的（其他人可见）


▌4.2 交易类型详细规则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【类型1：即时交易】
┌──────────────────────────────────────────────────────────────────────┐
│  示例："我给你3财富，换你2影响"                                      │
│                                                                      │
│  规则：                                                              │
│  1. 双方协商比例                                                     │
│  2. 双方同意后，立即交换                                             │
│  3. 交易完成，不可反悔                                               │
│  4. 无功德奖励（等价交换）                                           │
│                                                                      │
│  三毒影响：                                                          │
│  • 交易比例合理（±20%等价）：无三毒影响                              │
│  • 明显不利交易（>50%不等价）：接受者+1痴                            │
│  • 明显暴利交易（索要>2倍价值）：索要者+1贪                          │
└──────────────────────────────────────────────────────────────────────┘

【类型2：无偿布施】
┌──────────────────────────────────────────────────────────────────────┐
│  示例："我无偿给你5财富"                                             │
│                                                                      │
│  规则：                                                              │
│  1. 给予者声明"无偿布施"                                             │
│  2. 接受者可接受或拒绝                                               │
│  3. 给予者获得功德（根据布施功德表）                                 │
│  4. 给予者-1贪（放下执着）                                           │
│  5. 接受者无需回报                                                   │
│                                                                      │
│  布施功德表：                                                        │
│  • 1-2财富：+1功德                                                   │
│  • 3-5财富：+2功德                                                   │
│  • 6-10财富：+4功德                                                  │
│  • 10+财富：+6功德                                                   │
│  • 布施影响：每2影响=1功德                                           │
│                                                                      │
│  特殊：                                                              │
│  • 若给予≥10财富，额外-1贪                                           │
│  • 僧侣接受布施：布施者额外+1功德（供养三宝）                        │
└──────────────────────────────────────────────────────────────────────┘

【类型3：承诺交易】
┌──────────────────────────────────────────────────────────────────────┐
│  示例："我这轮给你3财富，你下轮帮我建寺"                             │
│                                                                      │
│  规则：                                                              │
│  1. 双方口头约定未来行动                                             │
│  2. 当轮资源立即转移                                                 │
│  3. 未来轮次，受约者应履行承诺                                       │
│  4. 若履行：-1嗔（守信）                                             │
│  5. 若违背：+2嗔，-2影响，被记录                                     │
│                                                                      │
│  承诺记录：                                                          │
│  • 使用"承诺标记"记录                                                │
│  • 放在受约者面前                                                    │
│  • 履行后移除，违背后翻面（黑色标记）                                │
│  • 黑色标记≥2：其他玩家可拒绝与你交易                                │
│                                                                      │
│  佛教意义：                                                          │
│  • 妄语戒：违背承诺是破戒                                            │
│  • 因果业报：不守信用会失去信誉                                      │
└──────────────────────────────────────────────────────────────────────┘

【类型4：借贷】
┌──────────────────────────────────────────────────────────────────────┐
│  示例："借我5财富，2轮后还你6财富"                                   │
│                                                                      │
│  规则：                                                              │
│  1. 约定本金和利息（利息≤20%）                                       │
│  2. 约定归还时间（1-3轮）                                            │
│  3. 当轮转移本金                                                     │
│  4. 到期归还本金+利息                                                │
│  5. 若归还：履行承诺，-1贪                                           │
│  6. 若不归还：+3贪，-3影响，进入"债务状态"                           │
│                                                                      │
│  债务状态：                                                          │
│  • 每轮自动-1功德（良心不安）                                        │
│  • 其他玩家可拒绝与你交易                                            │
│  • 清偿债务后解除                                                    │
│                                                                      │
│  高利贷惩罚：                                                        │
│  • 利息>20%：放贷者+2贪                                              │
│  • 借贷者可公开谴责，放贷者-2影响                                    │
│                                                                      │
│  佛教意义：                                                          │
│  • 借贷可行，但不可高利盘剥                                          │
│  • 欠债不还是不诚信，有业力惩罚                                      │
└──────────────────────────────────────────────────────────────────────┘


▌4.3 交易限制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【禁止交易】
1. 功德不可交易（佛教原则）
2. 修行阶段不可交易
3. 三毒不可交易
4. 发愿卡不可交换
5. 专属能力不可转让

【限制交易】
1. 影响每轮最多交易5点
2. 土地最多交易1块/轮
3. 承诺交易最多3个未完成承诺/人
4. 借贷最多2笔未清偿债务/人

【鼓励交易】
1. 无偿布施获得功德
2. 履行承诺减少嗔
3. 合理交易无三毒惩罚
4. 协助他人升级获得奖励

▌各职业的交易特色
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【僧侣】
• 交易弱势：缺少财富
• 特殊能力：托钵（请求布施）
• 交易价值：接受布施者获得额外功德
• 策略：多请求布施，少交易

【贵族】
• 交易强势：财富和影响都充足
• 特殊能力：大布施（10财富→+4功德+2影响）
• 交易价值：可以资助他人
• 策略：用财富换取功德和声望

【商人】
• 交易专家：天生的交易者
• 特殊能力：讨价还价（所有交易可重新谈判1次）
• 交易价值：经济引擎，产生财富
• 策略：通过贸易积累财富，再布施获得功德

【农夫】
• 交易弱势：资源最少
• 特殊能力：虔诚布施（1财富→1功德+1虔诚，效率高）
• 交易价值：虔诚感动他人
• 策略：小额持续布施，不参与大额交易

▌布施波罗蜜的游戏化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【佛教三种布施】

1. 财布施（Dana）
   游戏中：无偿给予财富，获得功德

2. 法布施（Dharma Dana）
   游戏中：僧侣讲经，传播佛法，所有人+功德

3. 无畏施（Abhaya Dana）
   游戏中：医者救治，消除恐惧，获得阴德

【布施的"三轮清净"】
• 施者清净：不求回报（无偿布施）
• 受者清净：感恩接受
• 施物清净：真实给予（不虚假）

→ 游戏中体现：
• 无偿布施获得功德
• 索要回报则无功德
• 虚假承诺增加嗔


▌交易的佛教辩证
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【佛教并不反对交易】
• 商人是正当职业
• 正当贸易是"正命"（八正道之一）
• 但反对：高利贷、欺诈、剥削

【游戏体现】
• 允许合理交易（±20%等价）
• 惩罚高利贷（利息>20%，+贪）
• 惩罚违背承诺（+嗔）
• 鼓励无偿布施（+功德，-贪）

【交易系统设计总结】

✓ 推荐加入交易系统，理由：
1. 增加社交互动（Catan的成功要素）
2. 缓解资源不均（游戏平衡需要）
3. 创造道德抉择（布施vs交易）
4. 增加商人职业特色（贸易专家）
5. 符合佛教经济学主题（布施、正命）

✓ 推荐规则：
• 限定在协商阶段（控制节奏）
• 4种交易类型：即时、布施、承诺、借贷
• 功德/修行阶段/三毒不可交易
• 三毒系统惩罚贪婪交易
• 基础版简化（仅即时+布施）

✓ 佛教意义：
• 布施获得功德，交易不获得
• 承诺与信用的业力
• 正当交易vs高利盘剥
• 体现"正命"与"布施波罗蜜"
