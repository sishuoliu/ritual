═══════════════════════════════════════════════════════════════════════════════
                          第三章：游戏流程
═══════════════════════════════════════════════════════════════════════════════

游戏进行【6轮】，每轮包含【5个阶段】

▌阶段1：收入阶段（1分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 商人获得+2财富（贸易能力）
2. 已建寺庙：每座寺庙使所有人+1功德

▌阶段2：协商阶段（2分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 讨论本轮策略
2. 协商资源分配
3. 可进行【布施】

【布施规则】
┌──────────────────────────────────────────────────────────────────────┐
│  布施 = 无偿将财富给予他人                                          │
│                                                                      │
│  布施者获得功德：                                                    │
│  • 普通玩家：每3财富 = +1功德                                       │
│  • 农夫：每2财富 = +1功德（虔诚能力）                               │
│                                                                      │
│  接受者无需回报，直接获得财富                                        │
└──────────────────────────────────────────────────────────────────────┘

▌阶段3：行动阶段（5分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【每人选择2个行动】（可重复选择同一行动）

【基础行动】所有人可用
┌──────────────────────────────────────────────────────────────────────┐
│  1. 修行：+2功德                                                    │
│                                                                      │
│  2. 劳作：+3财富                                                    │
│                                                                      │
│  3. 护法：消耗3财富 → 劫难-2                                        │
│                                                                      │
│  4. 渡化：尝试渡化1个众生（见众生卡要求）                           │
│     • 多人可共同出资渡化同一众生                                    │
│     • 成功后每位参与者+2功德                                        │
│                                                                      │
│  5. 建寺：消耗8财富+4影响 → 建造1座寺庙                             │
│     • 多人可共同出资                                                │
│     • 建成后每轮所有人+1功德                                        │
└──────────────────────────────────────────────────────────────────────┘

【专属行动】根据身份（每轮最多使用1次）
┌──────────────────────────────────────────────────────────────────────┐
│  僧侣 - 讲经：消耗1功德 → 选1人+2功德，劫难-1，自己+1功德           │
│                                                                      │
│  贵族 - 赈灾：消耗5财富 → 劫难-3，自己+1功德                        │
│                                                                      │
│  商人 - 贸易：被动能力，不占用行动                                  │
│                                                                      │
│  农夫 - 虔诚布施：每2财富布施=+1功德                                │
│         （替代普通布施规则）                                        │
└──────────────────────────────────────────────────────────────────────┘

▌阶段4：事件阶段（3分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1：翻开众生卡】
• 从众生牌堆翻开2张，放入众生区
• 众生需在2轮内渡化

【步骤2：翻开事件卡】
• 从事件牌堆翻开1张，立即执行效果

▌阶段5：劫难阶段（1分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1：劫难增长】
• 劫难+3

【步骤2：众生超时检查】
• 众生区停留满2轮的众生：劫难+4，移除该众生（警告：超时惩罚严重！）

【步骤3：失败检查】
• 若劫难≥20：【立即失败】，全员输

【步骤4：轮次结束】
• 第6轮结束后进入游戏结算

▌6.1 结算流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1：检查团队胜利】
□ 劫难≤15
□ 渡化≥5个
□ 建寺≥2座
□ 所有人功德≥10

若任一未达成 → 全员失败，游戏结束

【步骤2：公开发愿】
每人公开自己的发愿卡

【步骤3：计算个人得分】

个人得分 = 功德×2 + 发愿分 + 成就分

【成就分】
• 渡化最多：+3分
• 建寺贡献最多：+3分
• 布施最多：+3分
• 护法最多：+2分
（平局时各得）

【步骤4：评定果位】
• 最高分者：★佛果★（游戏胜者）
• 40+分：菩萨果
• 30-39分：阿罗汉果
• 20-29分：初果
• <20分：凡夫

▌6.2 计分示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【示例局】
• 团队渡化6个众生，建寺2座，劫难12 → 团队胜利

僧侣：功德20，发愿"菩萨道"达成（+11），渡化最多（+3）
→ 20×2 + 11 + 3 = 54分 → ★佛果★

贵族：功德14，发愿"护法檀越"达成（+13），建寺最多（+3）
→ 14×2 + 13 + 3 = 44分 → 菩萨果

商人：功德12，发愿"福慧商人"达成（+10）
→ 12×2 + 10 = 34分 → 阿罗汉果

农夫：功德18，发愿"布施长者"达成（+12），布施最多（+3）
→ 18×2 + 12 + 3 = 51分 → 菩萨果

▌8.1 团队合作要点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优先级】
劫难控制 > 渡化众生 > 建寺 > 个人修行

【分工建议】
• 贵族：主力赈灾（-3劫难效率最高）
• 僧侣：讲经帮助他人，渡化需影响的众生
• 商人：提供财富支持，必要时直接参与渡化
• 农夫：通过虔诚布施高效积累功德

【资源共享】
• 积极布施帮助资源不足的玩家
• 共同出资渡化众生和建寺
• 布施不是牺牲——布施者也获得功德

▌8.2 个人竞争要点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【发愿选择】
• 根据身份选择匹配的发愿
• 抽到的2张中选择更可行的那张
• 考虑其他人可能的发愿避免冲突

【平衡合作与竞争】
• 不能只顾自己——团队失败你也输
• 不能完全无私——最终要比较个人分数
• 找到平衡点：刚好帮助团队成功，同时最大化个人得分

【功德积累途径】
1. 修行行动：稳定+2/次
2. 布施：转化财富为功德
3. 渡化众生：参与者各+2
4. 寺庙产出：长期收益

▌为什么是半合作？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

这个设计体现了佛教的核心精神：

【菩萨道】普度众生 → 团队必须合作拯救世界
【阿罗汉道】自证涅槃 → 个人追求最高修行

在佛教中，修行者既要利他（帮助众生），也要自利（提升自己）。
游戏完美体现了这种"自利利他"的平衡。
