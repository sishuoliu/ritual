         《功德轮回：众生百态》v2.1 皈依抉择版规则书
         
         设计原则：
         • 渡化需要智慧门槛
         • 皈依与不皈依都有活路
         • 金钱只是资源，不计入得分
         • 金钱制胜不可行，但金钱可以利滚利
═══════════════════════════════════════════════════════════════════════════════

更新日期：2026年1月28日
版本：v2.1.3（多策略测试平衡版）


═══════════════════════════════════════════════════════════════════════════════
                   核心机制
═══════════════════════════════════════════════════════════════════════════════

【渡化门槛】
• 慧≥5 才能执行渡化行动
• 体现：智慧不足者无法引导他人

【皈依选择】
游戏开始时，每位玩家选择皈依或不皈依（僧侣必须皈依）

┌──────────────────────────────────────────────────────────────────────┐
│  选择        │  初始加成    │  持续效果                              │
│  ─────────────────────────────────────────────────────────────────  │
│  皈依者      │  +1福+1慧   │  修行+1慧，事件福慧+50%                 │
│  不皈依者    │  +3财富     │  劳作+4财富（世俗成功）                 │
└──────────────────────────────────────────────────────────────────────┘

【金钱机制】
• 金钱不计入最终得分
• 投资行动：消耗5财富，每2回合获得30%回报
• 劫难越高布施越有价值（劫难≥10时布施+1福）

【关键验证】
✓ 渡化门槛：慧不足者无法渡化
✓ 皈依/不皈依都有活路
✓ 金钱制胜不可行（贪婪策略0%胜率）
✓ 财富优先策略有风险但非完全失败（1.1%胜率）
✓ 角色平衡良好（差距<15%）
