▌A.1 交易时机
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

交易仅在【协商阶段】进行，时间延长至3分钟。

▌A.2 可交易资源
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 可交易：财富、影响（每轮最多5点）
✗ 不可交易：功德、发愿卡

【设计原理】
功德代表个人修行所得，无法转让（佛教原则）
但财富和影响是世俗资源，可以自由流通

▌A.3 四种交易类型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【类型1：即时交易】
┌──────────────────────────────────────────────────────────────────────┐
│  示例："我给你3财富，换你2影响"                                      │
│                                                                      │
│  规则：                                                              │
│  • 双方协商比例，同意后立即交换                                      │

│  • 完全自由，无强制比例                                              │

┌──────────────────────────────────────────────────────────────────────┐

│  • 1-2财富：+1功德                                                   │
│  • 3-5财富：+2功德                                                   │
│  • 6-10财富：+4功德                                                  │
│  • 10+财富：+6功德                                                   │
│                                                                      │
│  特殊：布施给僧侣额外+1功德（供养三宝）                              │
│                                                                      │

│  布施是六度波罗蜜之首，三轮清净的布施获得最大功德                    │

▌B.1 三毒概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

三毒是佛教核心概念，代表导致轮回的三种根本烦恼。

【贪】执着于财富和利益
【嗔】愤怒、违背承诺
【痴】愚昧、被骗、错误决策

每个玩家有三毒轨道（0-10），起始为0。

▌B.2 三毒触发
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【贪的触发】
• 拒绝合理求助（囤积资源）：+1贪
• 索要过高交易价格：+1贪
• 高利贷（>20%利息）：+2贪
• 不归还借贷：+3贪
• 财富≥15且从不布施：+1贪/轮

【嗔的触发】
• 违背承诺：+2嗔
• 恶意抬价：+1嗔
• 与他人争吵（GM判定）：+1嗔

【痴的触发】
• 接受明显不利交易（>2倍差价）：+1痴
• 被高利贷盘剥：+1痴
• 连续2轮无有效行动：+1痴

▌B.3 三毒惩罚
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│  三毒惩罚表                                                          │
│                                                                      │
│  贪≥5：每轮-1功德（贪心侵蚀修行）                                    │
│  嗔≥5：无法获得他人帮助（孤立）                                      │
│  痴≥5：所有行动效率-1（愚痴阻碍）                                    │
│                                                                      │
│  任意≥8：该毒对应惩罚加倍                                            │
│  任意≥10：最终果位最多为"凡夫"（无论分数多高）                       │
│                                                                      │
│  三毒总和≥15：立即触发"堕落事件"                                     │
│  • 随机失去3功德或5财富或3影响                                       │
│  • 三毒各-2（业报已受）                                              │
└──────────────────────────────────────────────────────────────────────┘

▌B.4 三毒净化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【降低贪】
• 布施：-1贪
• 大布施（≥10财富）：-2贪

【降低嗔】
• 履行承诺：-1嗔
• 帮助他人渡化众生：-1嗔
• 主动道歉（宣布放弃1个行动点）：-2嗔

【降低痴】
• 听僧侣讲经：-1痴
• 成功渡化众生：-1痴
• 修行行动：-1痴

▌B.5 配件需求
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▌E.3 业力循环机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│  【业力循环规则】                                                    │
│                                                                      │
│  当你使用事件卡的特殊效果时（对他人造成负面影响），                  │
│  被影响的玩家可以将该卡放入自己的"来世牌库"。                        │
│                                                                      │
│  在游戏后半段（第4轮起），玩家可以使用"来世牌库"中的卡。             │
│                                                                      │
│  这意味着：                                                          │
│  • 你对他人使用的负面效果，可能被他人用回你身上                      │
│  • 因果报应，业力循环                                                │
│                                                                      │
│  【策略考量】                                                        │
│  • 使用负面卡时要考虑后果                                            │
│  • 纯粹害人的行为会回来伤害自己                                      │
└──────────────────────────────────────────────────────────────────────┘

灵感来源：沃尔什《救赎的经济学》论文

"福田"是佛教核心概念：向僧团布施就像在肥沃的田地里播种，
种下善因，必将收获功德之果。

▌F.1 核心概念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│  【福田投资】                                                        │
│                                                                      │
│  玩家可以在游戏早期"投资"功德田：                                    │
│  • 投入：X财富（最低3财富）                                          │
│  • 锁定轮数：2-4轮                                                   │
│  • 到期收获：投入×系数 的功德                                        │
│                                                                      │
│  收获系数：                                                          │
│  • 投资2轮：×1.0（保本）                                             │
│  • 投资3轮：×1.5（1.5倍）                                            │
│  • 投资4轮：×2.0（2倍）                                              │
│                                                                      │
│  示例：                                                              │
│  第2轮投入6财富，锁定3轮                                             │
│  第5轮收获：6×1.5=9功德                                              │
└──────────────────────────────────────────────────────────────────────┘

▌F.2 风险与回报
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【风险】
• 投资期间财富被锁定，无法使用
• 若游戏提前结束（团队失败），投资失效
• 若劫难过高（≥15），福田收成减半

【策略意义】
• 早期投资：长期收益高，但短期资源紧张
• 晚期投资：风险低，但收益少
• 体现"业力投资"的佛教概念

┌──────────────────────────────────────────────────────────────────────┐

│                                                                      │
│  佛教意义：                                                          │

└──────────────────────────────────────────────────────────────────────┘

▌推荐加入顺序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 增加道德考量
• 与交易系统形成良好配合

• 增加长期规划

【第4阶段】按需选择其他模块
• 模块D/E/F：深化特定体验
• 模块G/H：扩展玩家人数和多样性
