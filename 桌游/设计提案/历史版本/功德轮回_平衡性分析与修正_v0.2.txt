【分析原则】
1. 难度越高的事件/目标 → 奖励越高
2. 风险越大 → 期望回报越高
3. 稀缺资源的获取 → 成本更高
4. 负面事件的严重程度 → 应与触发概率成反比

【评估维度】
• 达成难度：★☆☆☆☆(很容易) 到 ★★★★★(极难)
• 稀缺度：该身份获取该资源的难易程度
• 期望值：(成功概率×奖励) + (失败概率×惩罚)

▌皇帝能力分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

原设计：
• 【颁诏】消耗5影响：改变本轮事件效果 → OK
• 【赐寺】消耗10财富：任意玩家获得1座寺庙 → 过强，寺庙价值约15
• 【灭佛】消耗10影响：所有僧侣-5功德，皇帝-8皇恩 → 惩罚过轻
• 【护法】消耗5功德：所有玩家修行+2功德 → 成本过高

修正后：
┌──────────────┬────────────────────┬───────────┬──────────────────────────┐
│  能力        │  成本              │  收益     │  分析                    │
├──────────────┼────────────────────┼───────────┼──────────────────────────┤
│  颁诏        │  4影响             │  改变事件 │  影响对皇帝不稀缺，-1成本│
│  赐寺        │  12财富+3影响      │  赐寺+皇恩│  增加成本，平衡收益      │
│  灭佛        │  8影响             │  详见下   │  增加代价，减少收益      │
│  护法        │  3功德             │  全体+2   │  减少成本，功德对皇帝稀缺│
└──────────────┴────────────────────┴───────────┴──────────────────────────┘

灭佛修正：
• 成本：8影响
• 效果：所有僧侣-3功德（原-5过强）
• 代价：皇帝-10皇恩（原-8过轻），所有贵族对皇帝-2声望
