═══════════════════════════════════════════════════════════════════════════════
                   游戏概述
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】45-60分钟
【游戏轮数】6轮

【半合作机制】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：福慧得分+发愿加成最高者获胜

【核心理念】
• 福慧双修，缺一不可
• 渡化需要智慧门槛（慧≥5）
• 金钱只是资源，不计入得分
• 资源多者收益递减，防止滚雪球

═══════════════════════════════════════════════════════════════════════════════
                   资源递减机制
═══════════════════════════════════════════════════════════════════════════════

【防止滚雪球】
当某项资源≥15时，该资源相关行动收益-1
当某项资源≥20时，该资源相关行动收益-2

示例：
• 财富20时，劳作收益从3降为1
• 福25时，渡化福收益-2
• 慧18时，修行慧收益-1

┌──────────────────────────────────────────────────────────────────────┐

│                                                                      │
│  【布施】                                                            │
│  消耗：3财富                                                         │
│  效果：+2福+1慧（商人递减后仍较高）                                  │
│  劫难≥10时：额外+1福                                                 │
│                                                                      │
│  【渡化】★需要慧≥5                                                  │
│  消耗：众生卡成本+角色修正                                           │
│  效果：+众生卡福慧奖励+递减奖励                                      │
│  递减：每渡化1次，后续奖励-1                                         │
│                                                                      │
│  【护法】                                                            │
│  消耗：2财富                                                         │
│  效果：+2福+1慧，劫难-2                                              │

└──────────────────────────────────────────────────────────────────────┘

【学者专属】
• 讲学：自己+2福+1慧，目标+1福+1慧

【僧侣专属】
• 托钵（骰子）：成功+3财+1福+1慧，失败+1财+1慧
• 法事：+4财-2福-1慧（应急用）
