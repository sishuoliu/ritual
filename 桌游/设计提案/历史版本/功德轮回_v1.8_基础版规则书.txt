═══════════════════════════════════════════════════════════════════════════════
                   游戏概述
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】30-45分钟
【游戏轮数】6轮

【半合作机制】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：福慧得分最高者获得最高果位


═══════════════════════════════════════════════════════════════════════════════
                   资源系统
═══════════════════════════════════════════════════════════════════════════════

【三种资源】

┌──────────────────────────────────────────────────────────────────────┐
│  资源    │  含义          │  主要获取方式                           │
│  ─────────────────────────────────────────────────────────────────  │
│  财富    │  世俗资源      │  劳作+3                                 │
│  福      │  利他的成果    │  布施、渡化、护法、讲法、供养           │
│  慧      │  自度的成果    │  修行、托钵、渡化（部分）               │
└──────────────────────────────────────────────────────────────────────┘

【生存消耗】
每2回合结束时，所有玩家-1财富
财富<0时：财富归0，-1福，-1慧（饥饿惩罚）


═══════════════════════════════════════════════════════════════════════════════
                   角色系统
═══════════════════════════════════════════════════════════════════════════════

【4个角色】

┌──────────────────────────────────────────────────────────────────────┐
│  角色  │  初始      │  特殊能力                                     │
│  ─────────────────────────────────────────────────────────────────  │
│  农夫  │  财4福1慧1 │ 持续帮助时+2福（勤劳持久）                    │
│  商人  │  财6福0慧1 │ 布施+2福（善于结缘）                          │
│  官员  │  财5福2慧2 │ 渡化+2福（号召力）                            │
│  僧侣  │  财1福2慧3 │ 托钵/法事/讲法（不能正常劳作）                │
└──────────────────────────────────────────────────────────────────────┘

【僧侣特殊行动】
• 托钵（清修）：+2财富+1福+1慧
• 法事（应酬）：+4财富-2福-2慧
• 讲法：给他人+1福+1慧，自己+1福

【僧侣限制】
• 僧侣选择"劳作"时：只获得+1财富，且-1福


═══════════════════════════════════════════════════════════════════════════════
                   回合结构
═══════════════════════════════════════════════════════════════════════════════

【每轮流程】

┌──────────────────────────────────────────────────────────────────────┐
│  阶段1：事件阶段                                                     │
│  • 翻开1张事件卡，全体执行效果                                       │
│                                                                      │
│  阶段2：众生阶段                                                     │
│  • 翻开1张众生卡，放入众生区                                         │
│  • 检查超时：众生区满2轮未渡化 → 劫难+3，移除                        │
│                                                                      │
│  阶段3：行动阶段                                                     │
│  • 每人按顺序执行2个行动                                             │
│                                                                      │
│  阶段4：持续帮助奖励                                                 │
│  • 本轮帮助过他人的玩家，连续帮助轮数+1                              │
│  • 连续帮助3-4轮的玩家：+1福+1慧                                     │
│  • 农夫特殊：连续帮助3-4轮时+2福+1慧                                 │
│  • 僧侣特殊：不获得持续帮助的福加成（讲法本身已获福）                │
│                                                                      │
│  阶段5：生存结算（偶数轮）                                           │
│  • 第2、4、6轮结束时，所有玩家-1财富                                 │
│                                                                      │
│  阶段6：结算阶段                                                     │
│  • 检查团队胜利/失败条件                                             │
└──────────────────────────────────────────────────────────────────────┘


═══════════════════════════════════════════════════════════════════════════════
                   行动系统
═══════════════════════════════════════════════════════════════════════════════

【通用行动】（每回合选2个，可重复）

┌──────────────────────────────────────────────────────────────────────┐
│  【劳作】                                                            │
│  效果：+3财富                                                        │
│  僧侣限制：只+1财富且-1福                                            │
│                                                                      │
│  【修行】                                                            │
│  效果：+2慧（僧侣+3慧）                                              │
│                                                                      │
│  【布施】                                                            │
│  消耗：3财富                                                         │
│  效果：+2福+1慧，劫难-1                                              │
│  商人特殊：+4福+1慧                                                  │
│                                                                      │
│  【渡化】                                                            │
│  消耗：众生卡标注的财富                                              │
│  效果：+众生卡福慧奖励+渡化加成                                      │
│  渡化递减奖励：第1次+3福，第2次+2福，第3次+1福，第4次以上+0福        │
│  官员特殊：渡化额外+2福                                              │
│                                                                      │
│  【护法】                                                            │
│  消耗：2财富                                                         │
│  效果：+2福+1慧，劫难-2                                              │
│                                                                      │
│  【供养僧侣】（其他角色对僧侣使用）                                  │
│  消耗：2财富                                                         │
│  效果：僧侣+2财富，自己+2福，僧侣+1福                                │
└──────────────────────────────────────────────────────────────────────┘

【僧侣专属行动】

┌──────────────────────────────────────────────────────────────────────┐
│  【托钵】（清修）                                                    │
│  效果：+2财富+1福+1慧                                                │
│  说明：化缘修行，身心清净                                            │
│                                                                      │
│  【法事】（应酬）                                                    │
│  效果：+4财富-2福-2慧                                                │
│  说明：贪财做法事，损耗福慧                                          │
│                                                                      │
│  【讲法】                                                            │
│  效果：目标玩家+1福+1慧，自己+1福                                    │
│  说明：法布施，自利利他                                              │
└──────────────────────────────────────────────────────────────────────┘

【帮助行为】（影响持续帮助奖励）
• 物质帮助：布施、渡化、护法、供养（计入持续帮助）
• 法布施：讲法（不计入持续帮助，精神帮助与物质帮助区分）


═══════════════════════════════════════════════════════════════════════════════
                   计分系统
═══════════════════════════════════════════════════════════════════════════════

【个人得分计算】

┌──────────────────────────────────────────────────────────────────────┐
│  基础得分 = √(福 × 慧) × 3                                          │
│                                                                      │
│  示例：                                                              │
│  • 福=20，慧=5 → √(100)×3 = 30                                      │
│  • 福=12，慧=12 → √(144)×3 = 36 ← 平衡更优                         │
│  • 福=20，慧=20 → √(400)×3 = 60 ← 高度平衡最优                     │
│                                                                      │
│  道的加分（根据行为模式评定）：                                      │
│  • 菩萨道（福≥15，慧≥15）：+18                                      │
│  • 布施道（福≥18，慧≥8）：+15                                       │
│  • 禅修道（慧≥18，福≥8）：+15                                       │
│  • 居士道（福≥12，慧≥12，差≤5）：+14                                │
│  • 觉悟道（福≥20 或 慧≥20）：+16                                    │
│  • 世俗道（未达成以上）：+8                                         │
│                                                                      │
│  最终得分 = 基础得分 + 道的加分                                      │
└──────────────────────────────────────────────────────────────────────┘

【设计验证】
✓ 渡化递减：农夫渡化最多但奖励递减，避免一人独大
✓ 角色特色：农夫渡化，商人布施，官员护法，僧侣讲法
✓ 僧侣托钵：贪财（法事）会损福慧，清修（托钵）才能积累
✓ 持续帮助：鼓励每轮物质帮助他人
✓ 法布施区分：讲法是精神帮助，不计入持续帮助轨道
