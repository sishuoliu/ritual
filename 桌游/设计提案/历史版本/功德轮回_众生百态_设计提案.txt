▌1.1 与《救赎之路》的关系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

《救赎之路》系列 ─ 纯策略桌游
• 玩家选择"道路"（布施道、福田道等）
• 核心是资源转化和效率优化
• 玩家身份相同，仅道路不同
• 目标群体：策略玩家

《功德轮回：众生百态》─ 角色驱动桌游（新系列）
• 玩家扮演古代社会的不同身份
• 核心是角色体验和叙事沉浸
• 每个身份有完全不同的玩法
• 目标群体：喜欢角色扮演的玩家

共享元素：
• 佛教经济学主题
• 功德/财富/影响资源体系
• 三乘修行概念


▌1.2 游戏类型参考
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【主要参考】
• Root（非对称派系）
• Dead of Winter（秘密目标+角色事件）
• Spirit Island（独特角色能力+共享核心机制）
• Gloomhaven（角色发展+战役）

【设计理念】
• 每个身份是"一个独立的小游戏"
• 但在共享的佛教世界中互动
• 强调角色之间的社会关系和经济依赖

▌2.2 皇帝（The Emperor）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
天子受命于天，护持或毁灭佛法皆在一念之间。
历史上既有护法明君如梁武帝，也有灭佛暴君如唐武宗。

【起始资源】
• 财富：20（国库）
• 影响：15（皇权）
• 功德：0（帝王功德需自己积累）
• 特殊：皇恩轨道（起始5）

【专属能力】
• 【颁诏】消耗5影响：改变本轮事件效果（放大或取消）
• 【赐寺】消耗10财富：任意玩家获得1座寺庙
• 【灭佛】消耗10影响：所有僧侣玩家-5功德，皇帝-8皇恩
• 【护法】消耗5功德：所有玩家本轮修行+2功德

【秘密目标（4选1）】
1. 明君：游戏结束时皇恩≥8且无灭佛 → +30分
2. 灭佛之主：成功执行灭佛3次 → +25分
3. 平衡者：所有玩家功德差距≤5 → +35分
4. 万世之基：帮助≥3名玩家升乘 → +28分

【专属事件（8张）】
• 宫廷政变：皇帝掷骰，1-2失去皇位（游戏结束），3-6+5皇恩
• 外族入侵：皇帝消耗10财富或-8皇恩
• 皇子诞生：+3功德+5影响
• 太后信佛：皇帝必须对僧侣执行善行
• 方士献丹：掷骰1-2中毒（-5功德），3-6长生（+8功德）
• 泰山封禅：消耗20财富→+15功德+10影响
• 赈灾救民：消耗10财富→+8功德，不执行则-10皇恩
• 高僧觐见：可选择护法（+5功德+3皇恩）或驱逐（-3皇恩）


▌2.3 僧侣（The Monk）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
出家修行，以修证菩提为目标。
既可潜心修行成阿罗汉，也可弘法利生走菩萨道。

【起始资源】
• 财富：2（托钵所得）
• 影响：5（法师威望）
• 功德：8（出家功德）
• 特殊：戒律轨道（起始10）

【专属能力】
• 【托钵】不消耗行动点：向任意玩家请求布施（对方选择给或不给）
• 【讲经】消耗2影响：所有玩家+1功德（包括自己）
• 【闭关】消耗1轮行动：下轮所有修行效率翻倍
• 【化缘】财富为0时：自动获得2财富

【戒律系统】
• 杀生：-3戒律
• 偷盗：-5戒律
• 妄语：-2戒律
• 饮酒：-1戒律
• 戒律降到0：还俗（变为农夫身份）
• 戒律保持10：每轮+1功德

【秘密目标（4选1）】
1. 阿罗汉：游戏结束时功德≥35且戒律10 → +25分
2. 高僧大德：建寺2座且影响≥12 → +30分
3. 苦行僧：全程财富未超过5 → +20分
4. 普度众生：善行覆盖所有玩家各2次 → +28分

【专属事件（8张）】
• 魔障考验：掷骰1-3破戒（-3戒律-2功德），4-6破魔（+5功德）
• 贵人供养：获得5财富
• 法难将至：需要隐藏（本轮不可行动）或抗争（掷骰）
• 传法东瀛：可选择东渡（风险+高回报）
• 开悟时刻：随机触发顿悟事件
• 俗家弟子：一名其他玩家可选择皈依（给僧侣+2功德）
• 经卷残损：需要消耗3影响修复或-2功德
• 方丈圆寂：若有寺庙，掷骰决定继承（成为方丈+5影响）或争端（-3影响）


▌2.4 贵族（The Noble）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
世家大族，以财富和权势求取功德。
可成为大檀越供养僧团，也可建寺立碑扬名后世。

【起始资源】
• 财富：15（家族产业）
• 影响：8（门第威望）
• 功德：3（祖上积德）
• 土地：2
• 特殊：声望轨道（起始3）

【专属能力】
• 【大布施】消耗10财富：+5功德+3影响+2声望
• 【联姻】与另一玩家"结盟"，双方行动可互相加成
• 【修族谱】消耗5影响：游戏结束时声望转化为分数（1声望=2分）
• 【家族产业】每块土地每轮额外+1财富

【声望系统】
• 布施+1功德时：+0.5声望
• 建寺：+3声望
• 被皇帝赐寺：+5声望
• 帮助农夫/长工：+1声望
• 声望≥10：所有交易对自己更有利（买东西-1成本）

【秘密目标（4选1）】
1. 名门望族：声望≥15 → +30分
2. 护法檀越：供养僧侣≥5次 → +25分
3. 千秋功业：建寺3座 → +35分
4. 权倾朝野：影响≥15且与皇帝结盟 → +32分

【专属事件（8张）】
• 科举高中：+5影响+2声望
• 家族丑闻：-3声望-2影响
• 权力斗争：与另一贵族玩家竞争，影响高者+3声望
• 赈济灾民：选择捐助（-8财富+5功德+3声望）或不捐（-2声望）
• 祖坟风水：消耗5财富→每轮+1功德
• 皇帝召见：掷骰决定嘉奖（+5声望）或问责（-3声望）
• 佛事法会：可选择主办（消耗10财富→+8功德+4声望）
• 家族分裂：掷骰1-3失去1土地，4-6获得1土地


▌2.5 商人（The Merchant）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
往来各地，积累财富。
商人可通过财布施求取功德，也可资助寺院换取庇护。

【起始资源】
• 财富：12
• 影响：3
• 功德：1
• 特殊：贸易路线（起始1条）

【专属能力】
• 【贸易】消耗1行动：财富+（贸易路线数×2）
• 【开辟商路】消耗8财富：+1贸易路线（永久）
• 【资助寺院】消耗5财富给僧侣：你+2功德，僧侣+3财富
• 【讨价还价】所有交易可重新谈判1次

【贸易系统】
• 贸易路线1条：每轮+2财富
• 贸易路线2条：每轮+5财富
• 贸易路线3条：每轮+9财富
• 贸易路线可被盗匪抢劫

【秘密目标（4选1）】
1. 富甲天下：财富≥30 → +25分
2. 功德商人：通过财布施获得≥20功德 → +30分
3. 丝路之主：贸易路线≥4条 → +28分
4. 寺院护法：资助僧侣≥6次 → +24分

【专属事件（8张）】
• 商路畅通：本轮贸易收入翻倍
• 强盗劫道：失去1条贸易路线（若有盗匪玩家，由盗匪获得财富）
• 外邦商机：消耗10财富→下轮+20财富
• 官府勒索：-5财富或-3影响
• 佛像订单：消耗5财富→+4功德+2影响
• 善心施舍：选择施舍乞丐（-2财富+2功德）或拒绝（无效果）
• 合作契约：与另一商人玩家共享1条贸易路线
• 诚信危机：掷骰1-3失去2影响，4-6获得3影响


▌2.6 士人（The Scholar）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
读书人以文化传承为己任。
可著书立说传播佛理，也可批判佛教为"蛮夷之学"。

【起始资源】
• 财富：5
• 影响：6
• 功德：4
• 特殊：著作（起始0本）

【专属能力】
• 【著书】消耗3影响：+1著作，每本著作游戏结束+4分
• 【弘法著作】消耗2功德+2影响：+1佛学著作，每本+6分且所有玩家+1功德
• 【批判佛教】消耗3影响：皇帝皇恩对佛教-2，你+4影响
• 【讲学】消耗1行动：另一玩家+2影响，你+1影响

【著作系统】
• 普通著作：游戏结束+4分
• 佛学著作：游戏结束+6分+所有玩家读过+1功德
• 反佛著作：游戏结束+5分，但僧侣对你敌对

【秘密目标（4选1）】
1. 一代文宗：著作≥5本 → +30分
2. 佛学大师：佛学著作≥3本 → +28分
3. 帝师：被皇帝"采纳建议"≥3次 → +32分
4. 百家争鸣：同时有佛学和反佛著作 → +35分（最难）

【专属事件（8张）】
• 文字狱：掷骰1-2入狱（跳过1轮），3-6无事
• 科举状元：+8影响+3功德
• 著作流传：随机1本著作效果翻倍
• 翻译佛经：与僧侣合作→双方各+4功德
• 名士之交：与另一士人交换1本著作
• 皇帝垂询：影响最高的士人+5影响+3功德
• 私塾弟子：每本著作+1影响
• 禁书查抄：失去1本著作（若有反佛著作优先）


▌2.7 农夫（The Farmer）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
田间耕作，以勤劳为本。
农夫虽贫，但虔诚小额布施同样积累功德。

【起始资源】
• 财富：3
• 影响：1
• 功德：2
• 土地：1（租种，不可出售）
• 特殊：虔诚度（起始5）

【专属能力】
• 【勤劳】每轮自动+3财富（土地产出）
• 【虔诚布施】消耗1财富：+1功德+1虔诚（小额布施效率更高）
• 【拜佛许愿】消耗1影响：抽取愿望卡（小愿望容易实现）
• 【互助】帮助另一农夫：双方各+1功德+1财富

【虔诚系统】
• 虔诚≥10：每轮自动+1功德
• 虔诚≥15：布施效率翻倍
• 虔诚≥20：游戏结束+15分

【秘密目标（4选1）】
1. 虔诚信徒：虔诚≥20 → +25分
2. 勤劳致富：财富≥15 → +20分
3. 积德行善：功德≥20 → +22分
4. 祖传田地：保住土地且虔诚≥10 → +18分

【专属事件（8张）】
• 风调雨顺：本轮财富产出翻倍
• 天灾歉收：本轮无财富产出
• 地主索租：-3财富或失去土地
• 和尚化缘：可选择布施（-1财富+2虔诚）
• 病痛缠身：需要医者帮助或-2功德
• 儿孙满堂：+2虔诚+1影响
• 义举扬名：帮助他人后+3影响
• 菩萨显灵：虔诚最高的农夫+5功德


▌2.8 医者（The Healer）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
悬壶济世，以医术救人。
佛教认为"救人一命胜造七级浮屠"。

【起始资源】
• 财富：6
• 影响：4
• 功德：5
• 特殊：阴德（起始3）

【专属能力】
• 【治病救人】消耗1财富：移除另一玩家的负面状态，你+2阴德
• 【义诊】消耗2行动：所有玩家移除负面状态，你+5阴德
• 【制药】消耗3财富：获得3药物（可存储）
• 【药到病除】消耗1药物：抵消任意负面事件效果

【阴德系统】
• 阴德游戏结束转化为功德（1阴德=1.5功德）
• 阴德≥15：免疫所有负面事件
• 若拒绝救治：-3阴德

【秘密目标（4选1）】
1. 神医：阴德≥20 → +30分
2. 药王：制药≥10次 → +25分
3. 妙手回春：治疗≥8人次 → +28分
4. 悬壶济世：治疗所有玩家各1次 → +24分

【专属事件（8张）】
• 瘟疫爆发：医者必须义诊（-5财富+8阴德）或拒绝（-5阴德-3影响）
• 贵人求医：治疗贵族→+5财富+2影响
• 采药深山：掷骰1-3迷路（跳过1轮），4-6获得5药物
• 医术精进：所有治疗效果翻倍持续2轮
• 江湖郎中：需要证明医术（消耗2影响）或-3阴德
• 皇帝召医：治疗皇帝→+8影响+5财富+3功德
• 医者自医：移除自己1个负面状态
• 传授医术：另一玩家学会"简单治疗"（可消耗2财富移除负面状态）


▌2.9 盗匪（The Bandit）★隐藏角色★
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【角色背景】
落草为寇，可劫富济贫成为义贼，也可作恶多端堕入恶道。
业力系统决定最终结局。

【起始资源】
• 财富：5
• 影响：2（江湖名声）
• 功德：0
• 特殊：业力（起始0，可正可负）

【专属能力】
• 【劫道】抢劫商人1条贸易路线收入，业力-2
• 【劫富】抢劫贵族5财富，业力-3
• 【济贫】给农夫5财富，业力+4
• 【洗心革面】消耗10财富+5影响：转化为僧侣身份（保留业力）

【业力系统】（核心机制）
• 业力>10：成为"义贼"，所有济贫+2功德
• 业力<-10：成为"恶匪"，劫掠效率+50%但被通缉
• 业力=0：游戏结束时无加分
• 游戏结束：业力×2=分数（可正可负）

【通缉系统】
• 被皇帝通缉后：每轮掷骰1-2被捕（游戏结束-20分）
• 僧侣可为盗匪"超度"解除通缉（消耗5功德）

【秘密目标（4选1）】
1. 义贼：业力≥20 → +35分
2. 恶霸：财富≥25且业力<-15 → +25分（高风险）
3. 浪子回头：洗心革面成功 → +40分
4. 江湖传说：影响≥10（无论善恶） → +28分

【专属事件（8张）】
• 官兵围剿：掷骰1-3被捕，4-6逃脱
• 义气相投：与另一盗匪结盟（若有）
• 良心发现：+5业力+3功德
• 受害者诅咒：业力<0时，-3功德-2影响
• 劫持人质：获得10财富，业力-5
• 山寨壮大：+3影响+劫掠效率+20%
• 高僧点化：可选择皈依（+10业力）或拒绝
• 朝廷招安：消耗10财富→解除通缉+5影响，业力归0

▌3.1 社会关系网络
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【关系类型】
• 供养：商人/贵族 → 僧侣（给钱换功德）
• 保护：皇帝 → 僧侣（护法）
• 雇佣：贵族/商人 → 农夫（给钱）
• 医患：医者 → 任何人
• 敌对：盗匪 ↔ 商人/贵族
• 教化：士人/僧侣 → 任何人

【关系加成】
• 建立关系后，双方特定行动获得加成
• 关系可随事件变化（结盟/敌对/中立）

▌3.3 三乘系统整合
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【升乘规则】
• 沿用三乘解脱版规则
• 但非僧侣升乘难度+1（需掷更高点数）
