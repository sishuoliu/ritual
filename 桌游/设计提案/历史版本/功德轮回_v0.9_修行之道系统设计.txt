▌1.1 佛学核心精髓
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【不执着于形式】
• 不纠结于是否皈依
• 不纠结于哪个流派
• 不纠结于特定仪式
• 任何人都可能成佛

【真正的修行】
• 管理好自己（自度）
• 帮助他人（度人）
• 两者缺一不可

【中道智慧】
• 不是极端自我牺牲（一贫如洗但善心）
• 不是纯粹自利（只顾自己）
• 是：顾好自己 + 增加余力 + 有策略地帮助他人

【可持续性】
• 能力越大，越能帮助他人
• 临死前的捐赠可能来不及
• 持续的善行比一次性爆发更有价值

▌1.2 游戏设计目标
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

让玩家在游戏结束时感悟到：

1. 不管走什么路线，都可能"成道"
2. 光有善心不够，需要有能力
3. 光有能力不够，需要愿意帮助
4. 持续帮助比临终捐赠更有效
5. 先管理好自己，才能更好帮助他人

▌2.1 取消"皈依"机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【设计决策】
不再有"皈依"这个独立行动。
你的"道"不是通过一个仪式决定的，
而是通过整个游戏过程中的行为模式自然形成的。

【哲学依据】
正如六祖慧能所说：
"菩提本无树，明镜亦非台"
修行不在外在形式，在于内心和行动

▌2.2 双资源系统：福与慧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【福】代表你帮助他人的能力和意愿
• 通过帮助他人获得
• 体现"利他"

【慧】代表你管理自己的能力
• 通过修行和积累获得
• 体现"自利"

┌──────────────────────────────────────────────────────────────────────┐
│                     福 与 慧                                         │
│                                                                      │
│  【福】= 帮助他人所得                                                │
│  • 布施获得福                                                        │
│  • 渡化众生获得福                                                    │
│  • 帮助队友获得福                                                    │
│                                                                      │
│  【慧】= 管理自己所得                                                │
│  • 修行获得慧                                                        │
│  • 劳作积累资源是为了增加"余力"                                     │
│  • 有余力才能更好地帮助他人                                          │
│                                                                      │
│  【福慧双修】                                                        │
│  最终得分 = 福×慧 的某种函数                                        │
│  而非简单的加法                                                      │
│  这意味着：只有福或只有慧都不够                                      │
└──────────────────────────────────────────────────────────────────────┘

▌2.3 福慧乘积计分法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心公式】

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│     修行得分 = √(福 × 慧) × 3                                       │
│                                                                      │
│  这个公式的意义：                                                    │
│                                                                      │
│  • 福=20，慧=0 → 得分=0（只帮助他人，不管理自己）                   │
│  • 福=0，慧=20 → 得分=0（只管理自己，不帮助他人）                   │
│  • 福=10，慧=10 → 得分=30（平衡发展）                               │
│  • 福=15，慧=15 → 得分=45（高度平衡）                               │
│  • 福=20，慧=5 → 得分=30（偏福）                                    │
│  • 福=5，慧=20 → 得分=30（偏慧）                                    │
│                                                                      │
│  【设计意义】                                                        │
│  乘积开根号，鼓励平衡发展                                            │
│  单项极高但另一项为0，得分为0                                        │
│  这体现了"福慧双修"的核心理念                                       │
└──────────────────────────────────────────────────────────────────────┘

                   第三章：余力机制——能力越大责任越大
═══════════════════════════════════════════════════════════════════════════════

▌3.1 余力概念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【什么是余力】
余力 = 当前财富 - 基本生存需求

只有余力>0时，你才能有效地帮助他人
一贫如洗时布施，虽有善心，但效率很低

【基本生存需求】
每轮需要保留至少2财富作为"生存需求"
低于此值时：
• 进入"贫困状态"
• 帮助效率降低
• 自己也需要被帮助

▌3.2 帮助效率曲线
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【帮助效率公式】

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  帮助效率 = 基于余力的系数                                          │
│                                                                      │
│  余力（财富-2）  │  帮助效率  │  说明                                │
│  ───────────────────────────────────────────────────────────────    │
│  ≤0（贫困）      │  ×0.5     │  自身难保，帮助效率减半              │
│  1-3（紧张）     │  ×0.8     │  勉强度日，帮助有心无力              │
│  4-6（小康）     │  ×1.0     │  正常帮助                            │
│  7-10（富裕）    │  ×1.2     │  有余力，帮助更有效                  │
│  11+（富足）     │  ×1.5     │  能力强大，帮助效果显著              │
│                                                                      │
│  【示例】                                                            │
│  布施3财富给他人：                                                   │
│  • 贫困时布施：他人获得3财富，你获得0.5福                           │
│  • 富足时布施：他人获得3财富，你获得1.5福                           │
│                                                                      │
│  【设计意义】                                                        │
│  • 先顾好自己，才能更好帮助他人                                      │
│  • 能力越大，帮助越有效                                              │
│  • 不鼓励"倾家荡产式"的帮助                                        │
└──────────────────────────────────────────────────────────────────────┘

▌3.3 贫困状态
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【触发】财富<2

【效果】
• 帮助效率×0.5
• 无法参与建寺
• 无法主动渡化众生
• 需要他人帮助才能脱困

【脱困】
• 他人布施给你
• 劳作恢复财富

【设计意义】
不是说贫困者的善心没有价值
而是说：先稳定自己，才能持续帮助他人

▌4.1 问题：临终大额捐赠
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【现实问题】
有人一生不帮助他人，临死前捐出全部财产
这是否等同于一生持续帮助的人？

【佛学观点】
临终善念有价值，但不如一生修行
"平时不烧香，临时抱佛脚"效果有限

【游戏设计】
需要机制区分"持续帮助"和"临终捐赠"

▌4.2 持续帮助奖励机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【连续帮助轨道】

每轮如果你帮助了他人（布施、渡化、讲经等），
在你的"善行轨道"上前进一格

┌──────────────────────────────────────────────────────────────────────┐
│  【善行轨道】                                                        │
│                                                                      │
│  轮次  │  帮助  │  轨道位置  │  累积加成                            │
│  ─────────────────────────────────────────────────────────────────  │
│  第1轮 │  ✓    │  1         │  +0                                   │
│  第2轮 │  ✓    │  2         │  +1福（连续2轮）                      │
│  第3轮 │  ✓    │  3         │  +2福（连续3轮）                      │
│  第4轮 │  ✗    │  0（归零） │  连续中断                             │
│  第5轮 │  ✓    │  1         │  +0（重新开始）                       │
│  第6轮 │  ✓    │  2         │  +1福                                 │
│                                                                      │
│  【设计意义】                                                        │
│  • 持续帮助获得累积奖励                                              │
│  • 中断一轮，奖励归零                                                │
│  • 鼓励"每轮都帮助一点"而非"最后一轮帮助很多"                    │
└──────────────────────────────────────────────────────────────────────┘

▌4.3 临终帮助的贬值
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【最后一轮帮助效率】

┌──────────────────────────────────────────────────────────────────────┐
│  第6轮（最后一轮）的帮助效率系数：                                   │
│                                                                      │
│  • 如果前5轮都有帮助：×1.5（始终如一）                               │
│  • 如果前5轮帮助≥3轮：×1.0（正常）                                  │
│  • 如果前5轮帮助1-2轮：×0.7（偶尔善行）                             │
│  • 如果前5轮从未帮助：×0.3（临终悔悟）                              │
│                                                                      │
│  【示例】                                                            │
│  最后一轮布施10财富：                                                │
│  • 始终帮助者：获得5福（10×0.5×1.5=7.5→取5福）                      │
│  • 临终才帮助者：获得1.5福（10×0.5×0.3=1.5福）                      │
│                                                                      │
│  【设计意义】                                                        │
│  • 临终大额捐赠不如持续小额帮助                                      │
│  • "死前捐赠可能来不及"                                             │
│  • 鼓励有策略地持续帮助                                              │
└──────────────────────────────────────────────────────────────────────┘

▌5.1 取消固定发愿，引入"道的评定"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【原设计问题】
固定发愿限制了玩家的自由
"我选了这个发愿，就必须走这条路"

【新设计】
不再抽取固定发愿
游戏结束时，根据你的行为模式评定你的"道"

▌5.2 八种道的评定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

游戏结束时，根据你的行为自动评定：

┌──────────────────────────────────────────────────────────────────────┐
│  【八种道】                                                          │
│                                                                      │
│  1. 菩萨道（高福+高慧+持续帮助）                                     │
│     "上求佛道，下化众生"                                            │
│     条件：福≥15，慧≥12，连续帮助≥4轮                                │
│     加分：+20                                                        │
│                                                                      │
│  2. 布施道（高福+中慧）                                              │
│     "财法无畏，三施并行"                                            │
│     条件：福≥18，慧≥8，布施≥15财富                                  │
│     加分：+16                                                        │
│                                                                      │
│  3. 护法道（高福+护法行动多）                                        │
│     "护持三宝，功德无量"                                            │
│     条件：福≥12，护法≥4次，建寺≥1                                   │
│     加分：+15                                                        │
│                                                                      │
│  4. 禅修道（高慧+修行多）                                            │
│     "明心见性，自证涅槃"                                            │
│     条件：慧≥18，修行≥8次                                           │
│     加分：+14                                                        │
│                                                                      │
│  5. 商道（财富高+有布施）                                            │
│     "正命生财，广结善缘"                                            │
│     条件：财富≥12，福≥8                                             │
│     加分：+12                                                        │
│                                                                      │
│  6. 居士道（福慧均衡）                                               │
│     "在家修行，福慧双修"                                            │
│     条件：福≥10，慧≥10，差值≤3                                      │
│     加分：+14                                                        │
│                                                                      │
│  7. 济世道（渡化众生多）                                             │
│     "慈悲济世，普度众生"                                            │
│     条件：渡化≥4个，福≥10                                           │
│     加分：+16                                                        │
│                                                                      │
│  8. 觉悟道（任意条件达成极致）                                       │
│     "万法归一，道道皆通"                                            │
│     条件：任一资源≥20 且 其他无负面                                  │
│     加分：+18                                                        │
│                                                                      │
│  【无特定道】                                                        │
│  若不满足任何条件：                                                  │
│  "世间行走，尚在迷途"                                               │
│  加分：+0                                                            │
└──────────────────────────────────────────────────────────────────────┘

▌5.3 道的设计意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【不预设路线】
你不需要在游戏开始时选择走哪条道
你的行为自然会形成你的道

【多元成就】
有多种不同的"道"可以达成
商人可以走"商道"
农夫可以走"布施道"
没有唯一正确的答案

【佛学体现】
"条条大路通罗马"
不在于你选择了什么形式
在于你如何行动

▌6.1 团队胜利条件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【必须全部达成】
┌──────────────────────────────────────────────────────────────────────┐
│  □ 劫难≤15                                                          │
│  □ 渡化≥5个众生                                                     │
│  □ 建寺≥2座                                                         │
│  □ 团队总福≥30                                                      │
│  □ 团队总慧≥30                                                      │
│  □ 无人处于贫困状态                                                  │
│                                                                      │
│  任一未达成 = 全员失败                                               │
└──────────────────────────────────────────────────────────────────────┘

【设计意义】
• 不仅要帮助他人（福），也要管理自己（慧）
• 不能有人被抛下（无人贫困）
• 这是真正的"共同解脱"

▌6.2 个人得分计算
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【得分公式】

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  个人得分 = 修行得分 + 道的加分 + 成就加分                          │
│                                                                      │
│  其中：                                                              │
│  修行得分 = √(福 × 慧) × 3                                          │
│  道的加分 = 根据评定的道（0-20分）                                   │
│  成就加分 = 渡化最多+3，建寺最多+3，等                               │
│                                                                      │
│  【示例1：平衡型玩家】                                               │
│  福=12，慧=12，达成"居士道"                                         │
│  修行得分 = √(12×12)×3 = 36                                         │
│  道的加分 = 14                                                       │
│  总分 = 50                                                           │
│                                                                      │
│  【示例2：偏福型玩家】                                               │
│  福=18，慧=6，达成"布施道"                                          │
│  修行得分 = √(18×6)×3 = 31                                          │
│  道的加分 = 16                                                       │
│  总分 = 47                                                           │
│                                                                      │
│  【示例3：偏慧型玩家】                                               │
│  福=6，慧=18，达成"禅修道"                                          │
│  修行得分 = √(6×18)×3 = 31                                          │
│  道的加分 = 14                                                       │
│  总分 = 45                                                           │
│                                                                      │
│  【观察】                                                            │
│  平衡发展的玩家得分更高                                              │
│  偏向一边也可以，但需要达成对应的道                                  │
└──────────────────────────────────────────────────────────────────────┘

▌6.3 果位评定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【果位表】
┌──────────────────────────────────────────────────────────────────────┐
│  得分     │  果位          │  含义                                  │
│  ─────────────────────────────────────────────────────────────────  │
│  最高分   │  ★圆满觉悟★  │  福慧双修，道果圆成                    │

│  30-39    │  初果          │  初入圣道                              │

【关键】
• 不是"佛果"给最高分者
• 而是"圆满觉悟"——谁福慧双修最好
• 体现：不在于你选择什么形式，在于你如何实践

▌7.1 福慧获取方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【获得慧的方式】
┌──────────────────────────────────────────────────────────────────────┐
│  修行：+2慧                                                          │
│  劳作：+3财富（积累余力）                                            │
│  学习（新行动）：消耗2财富，+1慧，下轮行动效率+1                     │
│  持戒：若本轮无负面行为，+1慧                                        │
└──────────────────────────────────────────────────────────────────────┘

【获得福的方式】
┌──────────────────────────────────────────────────────────────────────┐
│  布施：3财富→+1福（×余力系数）                                      │
│  渡化众生：参与者各+2福                                              │
│  护法：消耗3财富→劫难-2，+1福                                       │
│  帮助队友：任何帮助他人的行为+0.5福                                  │
│  讲经（僧侣）：他人+2慧，自己+1福                                    │
└──────────────────────────────────────────────────────────────────────┘

【同时获得福和慧】
┌──────────────────────────────────────────────────────────────────────┐
│  建寺：消耗8财富+4影响                                               │
│        建成后每轮：建造者+1福+1慧                                    │
│        体现：护持三宝是福慧双修                                      │
│                                                                      │
│  连续帮助加成：                                                      │
│        连续帮助3轮以上，每轮额外+0.5慧                               │
│        体现：持续的善行本身就是修行                                  │
└──────────────────────────────────────────────────────────────────────┘

▌7.2 负面行为的后果
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【慧的流失】

• 拒绝合理帮助请求：-1慧
• 高利贷：-1慧

【福的流失】
• 恶意抬价：-1福
• 囤积不用：本轮福获取减半

▌8.1 玩家的心路历程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第1-2轮：积累期】
玩家自然会想先积累资源
"我先赚点钱，之后再帮助别人"
→ 这是正确的！游戏鼓励这样做

【第3-4轮：平衡期】
玩家开始意识到需要帮助他人
"我有余力了，该布施一些"
→ 余力系数让帮助更有效率

【第5-6轮：收获期】
玩家看到持续帮助的回报
"连续帮助有加成，我应该坚持"
→ 持续性奖励起作用

【游戏结束：感悟】
玩家发现：
• 平衡发展的人得分最高
• 临终大额捐赠效果不好
• 先顾好自己再帮助他人是对的
• 不同的道都可以成就

▌8.2 设计感悟点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【感悟1：能力与善心】
"光有善心不够，需要有能力"
→ 余力系数机制

【感悟2：持续与爆发】
"持续小额帮助比临终大额捐赠有效"
→ 连续帮助轨道 + 临终贬值

【感悟3：自度与度人】
"先管理好自己，才能更好帮助他人"
→ 福慧乘积计分

【感悟4：形式与实践】
"不在于走什么路线，在于如何行动"
→ 道的自然评定，无预设发愿

【感悟5：共同解脱】
"一个人得救不算真得救"
→ 团队胜利需要无人贫困

═══════════════════════════════════════════════════════════════════════════════

【设计总结】

本系统通过三个核心机制，让玩家自然感悟佛学精髓：

1. 【福慧乘积】
   → 只有福或只有慧都不够
   → 管理好自己+帮助他人

2. 【余力系数】
   → 能力越大帮助越有效
   → 先顾好自己才能更好帮助他人

3. 【持续性奖励】
   → 细水长流比临终捐赠有效
   → 有策略地持续帮助

【与原设计的核心区别】

原设计：皈依与否是关键抉择
新设计：行为模式决定你的道

原设计：预设发愿决定路线
新设计：行为自然形成道

原设计：功德是唯一资源
新设计：福+慧双资源，乘积计分

【玩家游戏后的感悟】

"原来不需要纠结于形式"
"原来要先顾好自己才能帮助他人"
"原来持续帮助比一次性帮助更有价值"
"原来每条路都可以成道"
