═══════════════════════════════════════════════════════════════════════════════
                   游戏概述
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】45-60分钟
【游戏轮数】6轮

【半合作机制】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：福慧得分+发愿加成最高者获得最高果位


═══════════════════════════════════════════════════════════════════════════════
                   骰子系统（2d6）
═══════════════════════════════════════════════════════════════════════════════

【判定规则】
┌──────────────────────────────────────────────────────────────────────┐
│  骰点    │  结果      │  概率      │  效果                          │
│  ─────────────────────────────────────────────────────────────────  │
│  2-4    │  大失败    │  16.7%    │  负面效果或减半                │
│  5-7    │  失败      │  41.7%    │  基础效果或无                  │
│  8-9    │  成功      │  27.8%    │  正常效果                      │
│  10-12  │  大成功    │  13.9%    │  额外奖励                      │
└──────────────────────────────────────────────────────────────────────┘

【需要骰子判定的行动】
• 修行：影响慧收益（大成功+1，大失败-1）
• 托钵：影响财富收益（成功+3财+1福，失败+1财）
• 部分专属事件

                   角色系统
═══════════════════════════════════════════════════════════════════════════════

【4个角色】

┌──────────────────────────────────────────────────────────────────────┐
│  角色    │  初始         │  特色                                    │
│  ─────────────────────────────────────────────────────────────────  │
│  农夫    │  财4福1慧1    │ 勤劳积累，持续帮助+2福                   │
│  商人    │  财6福1慧1    │ 布施+4福+2慧，善于结缘                   │
│  学者    │  财2福1慧4    │ 修行+3慧，讲学+2福+1慧                   │
│  僧侣    │  财0福2慧2    │ 需供养，托钵/法事/修行                   │
└──────────────────────────────────────────────────────────────────────┘

【僧侣特殊机制】
• 初始财富为0，需要他人供养或托钵获取财富
• 托钵（骰子判定）：成功+3财+1福+1慧，失败+1财+1慧
• 法事：+4财-2福-1慧（应急用，损福慧）
• 不能正常劳作（劳作只+1财且-1福）

▌专属事件（每人每轮抽1张）
每个角色有8张专属事件卡，抽到后立即执行。
类型包括：纯效果、骰子判定、选择事件。

【农夫专属事件示例】
• 丰收年（骰子）：成功+4财，失败+1财
• 村中长者求助（选择）：帮助+2福-2财，忽略无效果
• 遇僧求施（选择）：帮助+3福-1财，忽略无效果

【商人专属事件示例】
• 商路畅通（骰子）：成功+6财，失败+2财
• 贫民求借（选择）：帮助+3福-3财，忽略无效果
• 发现商机（骰子）：成功+5财，失败-1财

【学者专属事件示例】
• 著书立说（骰子）：成功+4慧，失败+1慧
• 受邀讲学（骰子）：成功+2慧+2财，失败+1慧
• 贫困求助（选择）：帮助+2福-2财，忽略无效果

【僧侣专属事件示例】
• 托钵化缘（骰子）：成功+3财+1福，失败+1财
• 魔障考验（骰子）：成功+3慧，失败-2慧-1福
• 法事邀请（选择）：做法事+4财-2福-1慧，拒绝+1福


═══════════════════════════════════════════════════════════════════════════════
                   回合结构
═══════════════════════════════════════════════════════════════════════════════

【每轮流程】

┌──────────────────────────────────────────────────────────────────────┐
│  阶段1：共享事件阶段                                                 │
│  • 翻开1张共享事件卡，全体执行效果                                   │
│                                                                      │
│  阶段2：众生阶段                                                     │
│  • 翻开1张众生卡，放入众生区                                         │
│  • 检查超时：众生区满2轮未渡化 → 劫难+3，移除                        │
│                                                                      │
│  阶段3：玩家回合（按顺序）                                           │
│  每位玩家依次执行：                                                  │
│  1. 抽取专属事件卡，执行效果                                         │
│  2. 执行2个行动                                                      │
│                                                                      │
│  阶段4：持续帮助奖励                                                 │
│  • 本轮帮助过的玩家，连续帮助轮数+1                                  │
│  • 连续帮助3-4轮的玩家：+1福+1慧                                     │
│  • 农夫特殊：连续帮助3-4轮时+2福+1慧                                 │
│                                                                      │
│  阶段5：生存结算（偶数轮）                                           │
│  • 第2、4、6轮结束时，所有玩家-1财富                                 │
│  • 财富<0时：财富归0，-1福，-1慧                                     │
│                                                                      │
│  阶段6：结算阶段                                                     │
│  • 检查团队胜利/失败条件                                             │
└──────────────────────────────────────────────────────────────────────┘

【个人得分计算】

┌──────────────────────────────────────────────────────────────────────┐
│  基础得分 = √(福 × 慧) × 3                                          │
│                                                                      │
│  道的加分（根据传统）：                                              │
│  • 居士道（福≥12，慧≥12）：+14                                      │
│  • 声闻道（慧≥18，福≥8）：+15                                       │
│  • 菩萨道（福≥18，慧≥8）：+16                                       │
│                                                                      │
│  发愿加分：                                                          │
│  • 小愿：+10~14分                                                    │
│  • 中愿：+18~22分（失败-5分）                                        │
│  • 大愿：+26~32分（失败-10~12分）                                    │
│                                                                      │
│  最终得分 = 基础得分 + 道的加分 + 发愿加分                           │
└──────────────────────────────────────────────────────────────────────┘
