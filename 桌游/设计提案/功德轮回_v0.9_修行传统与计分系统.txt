▌1.1 "只管自己也能赢"的漏洞
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【问题】
如果计分公式是 √(福×慧)×3，那么：
• 福=5，慧=20 → 得分30
• 福=12，慧=12 → 得分36

看起来平衡更好，但如果其他人更差呢？
一个"帮助一点点，主要管自己"的人可能混进去赢了。

【更深层问题】
这违背了游戏的核心理念：
"管理好自己+帮助他人"不是"主要管自己+象征性帮助"

【解决方案】
需要设计防自私机制

▌1.2 皈依/不皈依的设计困境
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【原设计思路】
v0.9皈依抉择系统：皈依者和不皈依者走不同路线

【问题】
1. 如果完全不同的路线，可能难以比较
2. 如果路线太相似，皈依选择就没意义
3. 事件卡如何处理？

【新思考】
也许"皈依"不应该是一个二元选择
而是"你在多大程度上投入于正式修行"的光谱

▌2.1 核心设计决策
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【决策1：统一计分，但有最低门槛】
所有人使用同一个计分公式
但设置福和慧的最低门槛，防止极端策略

【决策2：取消皈依二元选择，改为"修行传统"选择】
玩家选择一个"修行传统"（类似职业+路线的结合）
每个传统有不同的福慧比例要求和特殊能力

【决策3：事件卡采用"双效果"设计】
同一张卡，不同传统玩家获得不同效果
这样减少组件，也让玩家看到"另一条路会怎样"

▌2.3 传统选择的时机
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【选项A：游戏开始时选择】
• 优点：明确方向，简单
• 缺点：限制自由度

【选项B：游戏中途选择（推荐）】
• 在第2轮结束时选择传统
• 第1轮大家都是"初心者"，行动效果相同
• 第1轮后你看到了局势，再决定走什么路

【选项C：游戏结束时评定（最自由）】
• 不选择，行为自动匹配最合适的传统
• 优点：完全自由
• 缺点：可能最后不满足任何传统

【推荐：选项B】
第1轮摸索，第2轮开始选择传统
这既保留了选择的意义，又不会太早限制

▌3.1 问题重述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果有人选择"声闻传统"，然后只满足最低福要求（5福）
拼命修慧（20慧），是否可以赢？

计算：√(5×20)×3 = 30 + 阿罗汉道14 = 44分

对比：居士传统，福=12，慧=12
√(12×12)×3 = 36 + 居士道12 = 48分

看起来平衡更好，但问题是：
1. 达成5福太容易
2. 这种玩法破坏游戏体验（不互动）

▌3.3 新的平衡验证
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【情景1：自私的声闻玩家】
• 选择声闻传统
• 只满足最低要求：福=8（最低7.5向上取整）
• 拼命修慧：慧=20
• 参与渡化1次（最低要求）

得分计算：
• 修行得分：√(8×20)×3 = 38
• 道的加分：阿罗汉道+14
• 福排名惩罚：假设是最低-5
• 总分：38+14-5 = 47

【情景2：平衡的居士玩家】
• 选择居士传统
• 福=12，慧=12
• 参与渡化2次

得分计算：
• 修行得分：√(12×12)×3 = 36
• 道的加分：居士道+12
• 福排名惩罚：假设第二，-0
• 总分：36+12 = 48

【情景3：利他的菩萨玩家】
• 选择菩萨传统
• 福=16，慧=10
• 参与渡化3次

得分计算：
• 修行得分：√(16×10)×3 = 38
• 道的加分：菩萨道+16
• 福排名惩罚：最高，-0
• 总分：38+16 = 54

【结论】
通过福排名惩罚和渡化参与奖励
菩萨玩家 > 居士玩家 > 自私声闻玩家
但差距不是压倒性的，每种玩法都有可能赢

▌4.1 设计决策：同一张卡，不同传统不同效果
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【优点】
• 减少卡牌数量
• 每次抽卡都能看到"另一条路的结果"
• 增加选择传统时的考量

【缺点】
• 卡面信息量大
• 需要更大的卡牌

【解决方案】
采用三栏设计：居士/声闻/菩萨

▌5.1 各传统的优劣势分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│  传统    │  优势                  │  劣势                          │
│  ─────────────────────────────────────────────────────────────────  │
│  居士    │  灵活，资源积累快      │  道的加分较低（+12）           │
│          │  无强制约束            │  没有突出的能力加成            │
│          │                        │                                │
│  声闻    │  慧容易积累            │  福积累较慢                    │
│          │  受劫难影响小          │  必须慧≥福，限制发展           │
│          │  修行效率+1            │  可能成为"最低福者"被惩罚     │
│          │                        │                                │
│  菩萨    │  布施效率+50%          │  事件卡常有强制要求            │
│          │  道的加分最高（+16）   │  受劫难影响大                  │
│          │  团队贡献大，不会被惩罚│  必须福≥慧，需要更多资源      │
└──────────────────────────────────────────────────────────────────────┘

▌5.2 不同游戏阶段的策略
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【早期（第1-2轮）】
• 居士：劳作积累财富
• 声闻：修行积累慧
• 菩萨：劳作+布施开始积累福

【中期（第3-4轮）】
• 居士：根据局势调整
• 声闻：开始参与渡化（满足最低要求）
• 菩萨：大量渡化，积累福

【后期（第5-6轮）】
• 居士：确保达成道的条件
• 声闻：确保慧≥福，但福不能太低
• 菩萨：确保团队胜利，福排名第一

▌5.3 为什么每种传统都不能"只管自己"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【居士】
• 必须福≥6，慧≥6
• 福≥6需要至少2次有效布施或渡化
• 不能完全不帮助

【声闻】
• 必须福≥8（团队胜利要求最低×1.5）
• 必须参与至少1次渡化
• 如果福最低，-5分惩罚
• 不帮助会被惩罚

【菩萨】
• 必须福≥10
• 必须福≥慧
• 事件卡有强制帮助要求
• 不帮助直接"破戒"，失去传统加成

▌6.1 统一计分公式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  【步骤1：检查团队胜利】                                             │
│  所有条件达成 → 继续个人计分                                         │
│  任一条件未达成 → 全员失败，无排名                                   │
│                                                                      │
│  【步骤2：检查传统达成】                                             │
│  满足传统要求 → 获得传统加分                                         │
│  不满足 → 传统加分为0                                                │
│                                                                      │
│  【步骤3：计算修行得分】                                             │
│  修行得分 = √(福 × 慧) × 3                                          │
│                                                                      │
│  【步骤4：计算福排名惩罚】                                           │
│  福最低者：-5分                                                      │
│  福次低者：-2分                                                      │
│  其他：0分                                                           │
│                                                                      │
│  【步骤5：计算成就加分】                                             │
│  渡化最多：+3分                                                      │
│  建寺贡献最多：+3分                                                  │
│  持续帮助（连续5轮）：+3分                                           │
│                                                                      │
│  【最终得分】                                                        │
│  = 修行得分 + 传统加分 + 成就加分 - 惩罚                             │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘

▌6.3 为什么是统一计分而非分开计分
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【分开计分的问题】
如果皈依者和不皈依者分开计分：
• 两个独立的排名系统
• 难以比较谁"更厉害"
• 可能导致策略博弈（选择竞争少的组）

【统一计分的好处】
• 公平比较
• 不同传统的加分不同，体现差异
• 福慧乘积确保不能极端偏向

【佛学意义】
"法门八万四千，归元无二"
无论走哪条路，最终都在同一个标准下评判
这体现了"条条大路通罗马"的精神

▌7.1 回答最初的问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【Q1：事件卡如何处理？】
→ 同一张卡，三栏设计（居士/声闻/菩萨）
→ 减少组件，增加比较意义

【Q2：小乘/大乘/密宗如何体现？】
→ 通过"修行传统"系统体现
→ 基础版：居士/声闻/菩萨
→ 扩展版：增加密行/净土

【Q3：如何防止只管自己也能赢？】
→ 团队胜利需要每人福≥传统最低×1.5
→ 每人必须参与至少1次渡化
→ 福最低者-5分惩罚
→ 渡化有累积奖励

【Q4：统一计分还是分开计分？】
→ 统一计分
→ 不同传统的加分不同（+12/+14/+16）体现差异
→ 福慧乘积确保不能极端偏向

▌7.2 基础版推荐配置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【组件】
• 3种传统卡（居士/声闻/菩萨）
• 双效果事件卡×20张
• 角色卡×4（农夫/商人/官员/僧侣）
• 众生卡×15张

【第1轮】
所有人都是"初心者"，行动效果相同

【第2轮开始】
选择传统，获得对应能力和约束

▌7.3 设计核心理念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
