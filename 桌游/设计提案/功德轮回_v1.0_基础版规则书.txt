═══════════════════════════════════════════════════════════════════════════════
                   核心理念
═══════════════════════════════════════════════════════════════════════════════

【一句话】
"管理好自己，帮助他人，共同渡过劫难。"

【半合作】
• 团队目标：劫难≤12，渡化≥6个众生
• 个人目标：功德最高者获得最高果位

【核心矛盾】
资源有限，帮助他人会消耗自己的资源
但不帮助，团队会失败，所有人都输


═══════════════════════════════════════════════════════════════════════════════
                   第一章：游戏组件
═══════════════════════════════════════════════════════════════════════════════

【玩家数量】3-4人
【游戏时长】30-45分钟
【游戏轮数】6轮

【组件】
• 角色卡 ×4（农夫、商人、官员、僧侣）
• 众生卡 ×12
• 事件卡 ×18
• 劫难轨道 ×1（0-20）
• 资源标记（财富、功德）


═══════════════════════════════════════════════════════════════════════════════
                   第二章：资源系统（极简）
═══════════════════════════════════════════════════════════════════════════════

【只有2种资源】

┌──────────────────────────────────────────────────────────────────────┐
│  资源    │  含义              │  用途                               │
│  ─────────────────────────────────────────────────────────────────  │
│  财富    │  世俗资源          │  渡化众生、建寺、布施               │
│  功德    │  修行成果+善行     │  个人得分、减劫难、特殊能力         │
└──────────────────────────────────────────────────────────────────────┘

【设计说明】
• 取消了"慧"、"福"、"善缘"等多资源
• 功德=修行+善行的综合体现
• 财富=行动能力，功德=最终目标


═══════════════════════════════════════════════════════════════════════════════
                   第三章：角色系统
═══════════════════════════════════════════════════════════════════════════════

【4个角色】

┌──────────────────────────────────────────────────────────────────────┐
│  角色  │  初始财富  │  初始功德  │  特殊能力                        │
│  ─────────────────────────────────────────────────────────────────  │
│  农夫  │  4        │  2         │ 【勤劳】劳作时+1财富              │
│  商人  │  6        │  1         │ 【经营】可与他人交换资源          │
│  官员  │  5        │  1         │ 【号召】渡化时可邀请他人参与      │
│  僧侣  │  2        │  4         │ 【讲法】消耗1功德，他人+2功德     │
└──────────────────────────────────────────────────────────────────────┘

【设计说明】
• 农夫：稳定资源获取
• 商人：资源流动性
• 官员：团队协作
• 僧侣：功德输出（自己少赚，帮别人多赚）

                   第四章：回合结构
═══════════════════════════════════════════════════════════════════════════════

【每轮流程】

┌──────────────────────────────────────────────────────────────────────┐
│  阶段1：事件阶段                                                     │
│  • 翻开1张事件卡，全体执行                                           │
│                                                                      │
│  阶段2：众生阶段                                                     │
│  • 翻开1张众生卡，放入众生区                                         │
│  • 检查超时：众生区满2轮未渡化的众生 → 劫难+3，移除                  │
│                                                                      │
│  阶段3：行动阶段                                                     │
│  • 每人按顺序执行2个行动                                             │
│                                                                      │
│  阶段4：结算阶段                                                     │
│  • 检查团队胜利/失败条件                                             │
└──────────────────────────────────────────────────────────────────────┘


═══════════════════════════════════════════════════════════════════════════════
                   第五章：行动系统（核心）
═══════════════════════════════════════════════════════════════════════════════

【5种基本行动】（每回合选2个，可重复）

┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  【劳作】                                                            │
│  效果：+3财富                                                        │
│  说明：积累世俗资源                                                  │
│                                                                      │
│  【修行】                                                            │
│  效果：+2功德                                                        │
│  说明：自我提升                                                      │
│                                                                      │
│  【布施】                                                            │
│  消耗：3财富                                                         │
│  效果：+2功德，劫难-1                                                │
│  说明：散财积德，减少世间苦难                                        │
│                                                                      │
│  【渡化】                                                            │
│  消耗：众生卡标注的财富                                              │
│  效果：功德+众生卡标注值，移除该众生                                 │
│  说明：帮助众生脱苦                                                  │
│  特殊：可多人合作，财富分摊，功德平分（向上取整）                    │
│                                                                      │
│  【护法】                                                            │
│  消耗：2财富                                                         │
│  效果：劫难-2                                                        │
│  说明：减少灾难                                                      │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘

【行动设计说明】

资源流向：
• 劳作：无→财富
• 修行：无→功德
• 布施：财富→功德+公共利益
• 渡化：财富→功德+团队进度
• 护法：财富→公共利益

核心选择：
• 劳作vs修行：积累财富还是直接获得功德？
• 布施vs渡化：个人布施还是合作渡化？
• 何时护法：劫难高时必须有人牺牲

【众生卡设计】

┌──────────────────────────────────────────────────────────────────────┐
│  众生      │  渡化费用  │  功德奖励  │  特殊效果                     │
│  ─────────────────────────────────────────────────────────────────  │
│  饥民      │  3财富     │  2功德     │  无                           │
│  病人      │  4财富     │  3功德     │  无                           │
│  孤儿      │  3财富     │  2功德     │  渡化者下轮+1财富             │
│  老者      │  2财富     │  2功德     │  无                           │
│  流浪者    │  5财富     │  4功德     │  无                           │
│  冤魂      │  6财富     │  5功德     │  渡化后劫难-1                 │
│  恶人      │  8财富     │  6功德     │  未渡化则每轮劫难+1           │
│  富商      │  4财富     │  2功德     │  渡化后所有人+1财富           │
│  官吏      │  5财富     │  3功德     │  渡化者下轮行动+1             │
│  将军      │  7财富     │  5功德     │  渡化后劫难-2                 │
│  皇族      │  10财富    │  7功德     │  必须3人合作才能渡化          │
│  高僧      │  6财富     │  4功德     │  渡化者获得+2功德（共6）      │
└──────────────────────────────────────────────────────────────────────┘

【超时惩罚】
众生在众生区停留满2轮未被渡化：
• 劫难+3
• 移除该众生
• 不获得任何功德

【事件卡类型】

┌──────────────────────────────────────────────────────────────────────┐
│  类型        │  数量  │  示例                                       │
│  ─────────────────────────────────────────────────────────────────  │

【示例事件卡】

劫难事件：
• 旱灾：劫难+2
• 洪水：劫难+3
• 瘟疫：劫难+2，所有人-1财富
• 战乱：劫难+4
• 饥荒：劫难+2，本轮渡化费用+2
• 妖邪：劫难+3，本轮不能修行

机遇事件：
• 丰收：所有人+2财富
• 法会：所有人+1功德
• 施主到来：财富最少者+3财富
• 高僧开示：功德最少者+2功德
• 国泰民安：劫难-2
• 佛诞节：所有人可免费渡化1个众生

选择事件：
• 乞丐求施：可选择布施1财富获得1功德
• 迷途者：可消耗2财富帮助，获得2功德
• 富商供养：可接受3财富，但功德-1
• 恶人忏悔：可消耗2功德帮助，劫难-2
• 寺院修缮：可集体捐献，每人捐1财富则劫难-1
• 渡河：可消耗1财富帮助他人过河，双方各+1功德

                   第八章：胜负条件
═══════════════════════════════════════════════════════════════════════════════

【团队失败条件】（满足任一）
┌──────────────────────────────────────────────────────────────────────┐
│  • 劫难≥20（末法降临）                                               │
│  • 6轮结束时，渡化众生<6个                                           │
│                                                                      │
│  团队失败 = 全员失败，无人获胜                                       │
└──────────────────────────────────────────────────────────────────────┘

【团队胜利条件】（必须全部达成）
┌──────────────────────────────────────────────────────────────────────┐
│  • 6轮结束时，劫难≤12                                                │
│  • 6轮结束时，渡化众生≥6个                                           │
│                                                                      │
│  团队胜利后，按个人功德排名：                                        │
│  • 第1名：圆满果位（大赢家）                                         │
│  • 第2名：菩萨果位                                                   │
│  • 第3名：罗汉果位                                                   │
│  • 第4名：善信果位                                                   │
└──────────────────────────────────────────────────────────────────────┘

【设计说明】
• 团队胜利是前提，没有人能独赢
• 个人排名是动力，激励竞争
• 这体现了"共同解脱"的理念


═══════════════════════════════════════════════════════════════════════════════
                   第九章：策略深度
═══════════════════════════════════════════════════════════════════════════════

【核心博弈】

1. 【资源分配】
   劳作多→财富多→能帮助多→但功德主要给别人
   修行多→功德多→但缺财富帮助他人→团队可能失败
   
2. 【渡化合作】
   单独渡化：全部功德归自己，但花费大
   合作渡化：功德分享，但花费少
   
3. 【角色配合】
   僧侣：讲法给别人功德（利他）
   商人：资源流动（互通有无）
   官员：组织合作渡化
   农夫：稳定资源供应
   
4. 【时机选择】
   劫难高时：必须有人护法
   众生即将超时：必须有人渡化
   但这些行为消耗资源，功德不高

【僧侣困境（核心设计）】
僧侣的"讲法"能力：消耗1功德，他人+2功德
• 如果僧侣持续讲法，团队功德总量增加
• 但僧侣自己的功德会减少
• 这是"牺牲自己，成就他人"的核心选择
• 其他角色应该帮助僧侣吗？

                   第十二章：平衡性测试结论
═══════════════════════════════════════════════════════════════════════════════

【Monte Carlo 模拟测试结果】（3000局/配置）

┌──────────────────────────────────────────────────────────────────────┐
│                      核心发现                                        │
│  ─────────────────────────────────────────────────────────────────  │
│  1. 自私策略必败（0%胜率）                                           │
│     → 合作是必须的                                                   │
│                                                                      │
│  2. 平衡发展最优（99%胜率）                                          │
│     → 先顾好自己再帮助他人                                           │
│                                                                      │
│  3. 纯利他反而失败（1.7%胜率）                                       │
│     → 过度牺牲导致资源不足                                           │
│                                                                      │
│  4. 混合策略可行（75%胜率）                                          │
│     → 团队有不同类型玩家也能成功                                     │
└──────────────────────────────────────────────────────────────────────┘
