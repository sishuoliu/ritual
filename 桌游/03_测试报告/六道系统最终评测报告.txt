                          第一部分：六道系统设计理念
═══════════════════════════════════════════════════════════════════════════════

【核心理念】
每个角色拥有独特的"道"——专属修行路径与胜利机制，
而非仅依靠数值调整来平衡。

【六道体系】

┌─────────┬──────────────┬─────────────────────────────────┐
│  角色   │   专属道路   │   核心机制                      │
├─────────┼──────────────┼─────────────────────────────────┤
│  寡妇   │   菩萨道     │   发大愿，渡众生                │
│  僧侣   │   涅槃道     │   授法度人，法力归零            │
│  富商   │   善财道     │   散尽家财，功德圆满            │
│  地主   │   舍宅道     │   捐献祖产，兴建寺院            │
│  官员   │   清官道     │   致仕归隐，为民请命            │
│  农民   │   勤劳道     │   质朴修行，日行一善            │
└─────────┴──────────────┴─────────────────────────────────┘

▌v2.2 基础版（解脱加分机制）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
问题：寡妇55%胜率，僧侣0.75%，农民1.08%

▌v2.3 菩萨道机制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
新增寡妇"发大愿·渡众生"机制
结果：寡妇49%，僧侣9.4%，标准胜利从38%提升到63%

▌v2.4 六道系统初版
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
为每个角色添加专属机制
问题：官员76%胜率（声望自动增长过强）

▌v2.5-v2.7 迭代调整
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
持续调整各角色专属道路加分
v2.5: 官员87%（仍过高）
v2.6: 僧侣51%（过高），官员0%（过低）
v2.7: 地主54%（过高），僧侣0%（过低）

▌v2.8 最终平衡版
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
精确调整所有道路加分
结果：五个角色胜率合理分布

▌问题：地主胜率过低（0.02%）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【原因分析】
1. 地主虽然100%完成舍宅道，但加分（42）低于其他角色
2. 地主的基础分（148.6）是所有角色中最低的
3. 舍宅道机制设计可能需要重新审视

【建议修改】
• 方案A：舍宅道加分从42提高到58
• 方案B：增加地主的传承点加分系数
• 方案C：为地主增加新机制"耕读传家"

▌"发大愿·渡众生"机制最终评估
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【触发数据】
• 菩萨状态触发率：~27%（需功德≥8，声望=0，资源≤10）
• 平均渡化众生数：~2人
• 菩萨道胜利贡献：20.89%

【设计成功之处】
1. 主题完美契合大乘佛教"自觉觉他"精神
2. 创造了独特的"高风险高回报"玩法
3. 增加了玩家互动和合作元素
4. 使寡妇从"受苦者"转化为"救度者"

【设计风险】
1. 寡妇得分波动极大（标准差62.4，远高于其他角色）
2. 未能触发菩萨状态的寡妇几乎无法获胜
3. 这种设计增加了游戏的不确定性

                          结论
═══════════════════════════════════════════════════════════════════════════════

【六道系统评估】

设计理念：★★★★★（5/5）
• 每个角色有独特的修行路径和叙事弧线
• 完美契合佛教"条条大路通涅槃"的思想
• 机制设计融入了佛学概念（菩萨道、涅槃、布施等）

游戏多样性：★★★★☆（4/5）
• 六种不同的胜利路径
• 玩家可以根据角色选择不同策略
• 每局游戏的胜利方式各不相同

平衡性（v2.8）：★★★☆☆（3/5）
• 五个角色胜率在8-34%之间（良好）
• 地主需要进一步调整
• 僧侣略强，可微调

主题契合度：★★★★★（5/5）
• 菩萨道"渡众生"机制极具创意
• 清官道"为民请命"符合儒家理想
• 勤劳道体现"日行一善"的朴素美德

【最终建议】
v2.8已达到可玩状态，建议：
1. 提高地主舍宅道加分到55-60
2. 略微降低僧侣涅槃道加分到48
3. 进行真人测试以验证AI模拟结论
